Manipulating Styles with JavaScript

Why Manipulate Styles?

There will be times when you need to manipulate the styles of an HTML element inside your JavaScript file.

Common Use Cases

  • Show/Hide Menus: Display navigation when a user clicks a button
  • Toggle Features: Enable or disable UI components
  • Visual Feedback: Highlight elements on interaction
  • Dynamic Theming: Change colors and styles on the fly
  • Responsive Behavior: Adjust styles based on user actions

Two Primary Methods

You'll learn how to manipulate styles using Element.style and Element.classList properties.

Element.style Property

What is Element.style?

The Element.style property is a read-only property that represents the inline style of an element.

What Can You Do?

  • Get Styles: Read the current inline style values
  • Set Styles: Change CSS properties directly
  • Modify Appearance: Update visual properties dynamically

Important Note

While the style property itself is read-only, you can set individual CSS properties through it.

Using Element.style

HTML Structure

<p id="para">This is a paragraph</p>

JavaScript - Changing Color

const para = document.getElementById("para");
para.style.color = "red";

Result

The paragraph text color changes to red instantly.

Direct Style Manipulation

This adds an inline style to the element: style="color: red;"

CSS Properties with Element.style

Many Properties Available

The style property can be used to set many CSS properties.

Common CSS Properties

color
backgroundColor
fontSize
fontWeight
margin
padding
border
display
width
height

Naming Convention

CSS properties in JavaScript use camelCase instead of kebab-case:

  • CSS: background-color → JavaScript: backgroundColor
  • CSS: font-size → JavaScript: fontSize

Element.classList Property

A Better Way to Manage Styles

The Element.classList property is a read-only property that can be used to add, remove, or toggle classes on an element.

Why Use classList?

  • Cleaner Code: Keep styles in CSS, logic in JavaScript
  • Multiple Properties: Change many styles at once with a class
  • Reusability: Use the same classes across multiple elements
  • Maintainability: Update styles in CSS without touching JavaScript

classList provides methods to manipulate CSS classes dynamically.

classList.add() Method

Adding Classes

Use classList.add() to add one or more classes to an element.

CSS

.highlight {
  background-color: yellow;
}

JavaScript - Add Single Class

const para = document.getElementById("para");
para.classList.add("highlight");

Result

The paragraph element gets the highlight class, and its background turns yellow.

Adding Multiple Classes

Multiple Classes at Once

You can add multiple classes to an element by passing them as separate arguments.

Syntax

element.classList.add("class1", "class2", "class3");

Example

const para = document.getElementById("para");
para.classList.add("highlight", "bold", "large");

Result

The element receives all three classes: highlight, bold, and large.

classList.remove() Method

Removing Classes

If you need to remove a class from an element, use classList.remove().

Syntax

element.classList.remove("className");

Example

const para = document.getElementById("para");
para.classList.remove("highlight");

Result

The highlight class is removed from the paragraph, and the yellow background disappears.

You can also remove multiple classes by passing them as separate arguments, just like with add().

classList.toggle() Method

Toggle Classes On/Off

Use classList.toggle() to add a class if it's not present, or remove it if it is.

How It Works

Class Not Present

If the class doesn't exist on the element, toggle() adds it.

Class Present

If the class already exists on the element, toggle() removes it.

Syntax

element.classList.toggle("className");

Perfect for Interactive Elements

Great for buttons, menus, and other UI elements that switch between states.

Toggle Example: Menu Visibility

Real-World Scenario

Create a button that toggles a navigation menu's visibility.

HTML

<button id="toggle-btn">Toggle Menu</button>
<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>

Toggle Example: CSS Styles

CSS

.menu {
  display: none; /* Hidden by default */
}

.menu.show {
  display: block; /* Visible when 'show' class is added */
}

How It Works

  • The menu is hidden by default (display: none)
  • When the show class is added, the menu becomes visible (display: block)

Toggle Example: JavaScript

const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => {
  menu.classList.toggle("show");
});

What Happens?

  • We get references to the menu and button elements
  • We add a click event listener to the button
  • When clicked, we toggle the show class on the menu

Toggle Example: Behavior

Interactive Behavior

When a user clicks the button, here's what happens:

First Click

✓ The show class is added to the menu element

✓ The menu becomes visible (display: block)

Second Click

✓ The show class is removed from the menu element

✓ The menu becomes hidden (display: none)

Subsequent Clicks

Each click continues to toggle between visible and hidden, providing an interactive user experience.

Element.style vs Element.classList

Element.style

  • Direct manipulation of inline styles
  • Changes individual CSS properties
  • Good for single property changes
  • Adds inline styles to HTML
element.style.color = "red";

Element.classList

  • Manages CSS classes on elements
  • Changes multiple properties at once
  • Better separation of concerns
  • More maintainable
element.classList.add("highlight");

Summary

Key Methods and Properties

Element.style

Direct manipulation of inline styles. Change individual CSS properties like color, backgroundColor, fontSize, etc.

Element.classList Methods

  • add(): Add one or more classes to an element
  • remove(): Remove one or more classes from an element
  • toggle(): Add class if absent, remove if present

Creating Dynamic Web Pages

By using these methods and properties, you can easily manipulate an element's styles with JavaScript, creating dynamic and interactive web pages.
