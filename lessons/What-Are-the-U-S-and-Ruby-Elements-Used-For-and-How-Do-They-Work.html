The U Element

The unarticulated annotation element, or <u> element for short, is used to represent inline text that has non-textual annotation applied.

Example: Highlighting Spelling Errors

<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
Output:
You can use the unarticulated annotation element to highlight inccccort spling issses.
In this example, the words incorrect, spelling, and issues are misspelled.
Default Styling: The <u> element displays a black underline underneath the text.

U Element Usage Guidelines

HTML4

The <u> element was used for styling purposes.

HTML5

The <u> element should only be used to indicate that text has non-textual annotation applied.

In HTML5, the <u> element has semantic meaning - it represents text with non-textual annotation, such as:
Spelling errors (as shown in the previous example)
Proper name marks in Chinese text
Other non-textual annotations
Important: If you want to style a piece of text with an underline for visual purposes, you should use CSS instead of HTML.
/* CSS for styling underlines */
.underlined {
  text-decoration: underline;
}

The S Element

The strikethrough element, or <s> element for short, should be used to represent when text is no longer accurate or relevant.

Example: Canceled Activity

<p><s>Tomorrow's hike will be meeting at noon.</s></p>
<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
Output:

Tomorrow's hike will be meeting at noon.

Due to unforeseen weather conditions, the hike has been canceled.

In this example, the first sentence is crossed out because the hike was canceled due to weather reasons. The text is no longer accurate.

When NOT to Use the S Element

Important: The <s> element should never be used just to show changes to a document.
For showing document changes, use more appropriate elements:

<del> Element

The deleted text element indicates content that has been removed from a document.

<ins> Element

The inserted text element indicates content that has been added to a document.

Use <s> when: Information is no longer accurate or relevant (like expired offers, canceled events, outdated prices)

Use <del> and <ins> when: Tracking editorial changes or version history in a document

The Ruby Element

The ruby element (<ruby>) represents small text shown above or below the main text. It is typically used to show the pronunciation of East Asian characters.

Component Elements:
<rp> (Ruby Fallback Parenthesis): Used as a fallback for browsers lacking support for displaying ruby annotations
<rt> (Ruby Text): Used to indicate text for the ruby annotation, usually for pronunciation or translation details
While the ruby element can be used for other types of annotations, the most common use case is for East Asian typography.
