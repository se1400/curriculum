The querySelectorAll() Method

What is querySelectorAll()?

A method that gets a list of all DOM elements matching a specific CSS selector.

Basic Syntax

document.querySelectorAll(selectors);

• Call it on the document object

• Pass a valid CSS selector string as an argument

• Invalid selectors throw a SyntaxError

Return Value

// Returns a NodeList object const matches = document.querySelectorAll(".class");

• Returns a NodeList object

• Contains one Element object per match

• Empty list if no matches found

• Elements appear in HTML document order

Selector Examples

Common CSS Selectors

By Element Type

document.querySelectorAll("div");

Selects all <div> elements

By Class Name

document.querySelectorAll(".rounded");

Selects all elements with class "rounded"

By ID

document.querySelectorAll("#logo");

Selects all elements with id "logo"

By Attribute

document.querySelectorAll( "a[href='https://www.freecodecamp.org/']" );

Selects links with specific href attribute

Complex CSS Selectors

Combining Selectors for Precise Targeting

You can use complex CSS selectors to target specific elements within a hierarchy.

Descendant Selector Example

document.querySelectorAll("ul.ingredients li");

Matches all <li> elements within a <ul> that has class "ingredients"

HTML Structure

<ul class="ingredients"> <li>Flour</li> <li>Cheese</li> <li>Water</li> </ul>

Interactive Demo

  • Flour
  • Cheese
  • Water

Working with the NodeList

Storing and Using Results

Assign the returned NodeList to a variable to work with the collection.

Assigning to a Variable

const matches = document.querySelectorAll("ul.ingredients li");

NodeList Structure

// NodeList(3) { 0: `<li>Flour</li>`, 1: `<li>Cheese</li>`, 2: `<li>Water</li>`, length: 3 }

Key Points

• NodeList is an array-like collection of nodes

• Each node represents an element that matched the selector

• Work with it like a JavaScript array

• Nodes are indexed starting from 0

Accessing NodeList Elements

Display the Collection

console.log(matches);

Shows the entire NodeList in the console

Check the Length

console.log(matches.length);
3

Returns the number of elements in the collection

Access by Index

console.log(matches[0]); console.log(matches[1]); console.log(matches[2]);
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>

Access individual elements using bracket notation

• First element is at index 0

Iterating Over NodeList Elements

Using a For Loop

Loop through all elements in the NodeList to perform operations on each one.

For Loop Example

for (let i = 0; i < matches.length; i++) { console.log(matches[i]); }
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>

Why This is Powerful

• Process all matching elements at once

• Modify content, styles, or attributes

• Add event listeners to multiple elements

• Extract data from multiple elements

Key Takeaways

querySelectorAll() Summary

A powerful tool for selecting multiple elements based on CSS selectors.

Feature Description
Purpose Select multiple DOM elements using CSS selectors
Return Value NodeList object containing matched elements
Empty Result Returns empty NodeList (not null or error)
Element Order Matches appear in HTML document order
Usage Access by index, check length, iterate with loops

Remember

• Must be a valid CSS selector string

• Works with any CSS selector (simple or complex)

• Returned NodeList is array-like but not a true array

• Perfect for manipulating multiple elements at once
