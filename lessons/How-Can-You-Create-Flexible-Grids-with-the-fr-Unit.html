Flexible Grids with the fr Unit

Creating Flexible Layouts

In previous lessons, you were introduced to CSS Grid which can be used to create complex and fluid layouts in your web pages.

In this lesson, we will explore how to create flexible grid layouts using the fr unit.

What You'll Learn: How to use the fr (fractional) unit to create responsive, proportional grid columns that adapt automatically to screen sizes.

Why fr Units Matter: They provide a more flexible and intuitive way to create grid layouts compared to percentages or fixed pixel values.

Basic CSS Grid Setup

Activating Grid Layout

Inside the CSS, we set the display property to grid for the container.

.grid-container { display: grid; }

Key CSS Properties:

  • display: grid; – Activates grid layout
  • grid-template-columns – Sets column sizes
  • gap – Creates space between columns

Note: Setting display: grid alone doesn't create columns. We need to define the column structure with grid-template-columns.

Grid with Percentages

The Traditional Approach

grid-template-columns is used to set the size for each column. In this case, each column size will be 25% of the container.

.grid-container { display: grid; grid-template-columns: 25% 25% 25% 25%; gap: 20px; }

The gap property creates a 20-pixel space around each column, making the layout more readable.

Introducing the fr Unit

A Better Way

So far we have been using percentages for the column size, but we can also use the fr unit.

The fr unit is a fractional unit which represents a fraction of the space for the grid container.

Definition: "fr" stands for "fraction" – it represents a portion of the available space in the grid container.

Understanding Fractions:
1fr
1 fraction of available space
Flexible and proportional

Converting to fr Units

Refactored Code

Here is what the code will look like when it is refactored to use fr units instead of percentages.

.grid-container { display: grid; grid-template-columns: 1fr 1fr 1fr 1fr; gap: 20px; }

How fr Units Work

Equal Distribution

Each column will take up one fraction of the available space. Since there are four columns, each column will have an equal share of the space available in the grid container.

The Math: 4 columns × 1fr each = 4fr total. Each column gets 1fr ÷ 4fr = 25% of available space, automatically calculated!

Percentages vs fr Units

Key Differences

Percentages

grid-template-columns: 25% 25% 25% 25%;
  • Fixed percentages
  • Must add up to 100%
  • Less flexible with gaps
  • Manual calculations

fr Units ✓

grid-template-columns: 1fr 1fr 1fr 1fr;
  • Flexible fractions
  • Automatic distribution
  • Works with gaps naturally
  • Browser calculates

Creating Unequal Columns

Proportional Sizing

The real power of fr units becomes apparent when you want unequal column sizes. Different fr values create proportional columns.

Advantages of fr Units

Why You'll Use Them More Often

As you start to build your grid layouts, you will find yourself wanting to use fr units more often because they provide several key benefits:

  • Flexible – Adapts to container size automatically
  • Proportional – Easy to create balanced layouts
  • Responsive – Works perfectly with varying screen sizes
  • No manual calculations – Browser does the math
  • Gap-friendly – Works naturally with gap property
  • Intuitive – Easier to understand than percentages

Bottom Line: fr units allow you to create responsive layouts that adapt to varying screen sizes without needing to manually adjust pixel values.
