CSS Flexbox

CSS Flexbox is a one-dimensional layout model that allows you to arrange elements in rows and columns within a container.

What Makes It "One-Dimensional"?

Flexbox focuses on arranging elements along a single axis at a time. The axis can be either horizontal or vertical.

✓ Primary Uses:

  • Responsive websites - Adapt to different screen sizes
  • Navigation menus - Horizontal or vertical layouts
  • Card layouts - Evenly distributed items
  • Centering content - Easy vertical and horizontal centering
  • Form layouts - Aligned input fields

Two Key Concepts

Before working with Flexbox, you need to understand these two fundamental concepts:

Flex Container

The parent element with flex layout

  • ✓ Created with display: flex
  • ✓ Controls layout of children
  • ✓ Defines axis direction
  • ✓ Manages space distribution
Flex Items

Direct children of flex container

  • ✓ Can shrink or expand
  • ✓ Arranged along axis
  • ✓ Can be reordered
  • ✓ Individual alignment control

Creating a Flex Container

To make an HTML element a flex container, add display: flex to its CSS styles:

HTML
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
CSS
main {
  display: flex; /* Creates flex container */
}

Important: By default, a flex container will be a block-level element, so the container itself will be on its own row relative to other elements.

The Flex Model: Main Axis and Cross Axis

Every flex container has two axes that determine how items are arranged:

1 Main Axis: The primary axis along which flex items are arranged
2 Cross Axis: The perpendicular axis to the main axis

Default Orientation:

  • Main axis: Horizontal (left to right)
  • Cross axis: Vertical (top to bottom)
  • Items are arranged in the direction of the main axis
  • The cross axis is perpendicular to the main axis

The flex-direction Property

The flex-direction property sets the direction of the main axis:

CSS
/* Four possible values: */

flex-direction: row; /* Default - horizontal → */
flex-direction: row-reverse; /* Horizontal reversed ← */
flex-direction: column; /* Vertical ↓ */
flex-direction: column-reverse; /* Vertical reversed ↑ */

What It Does:

  • Changes main axis direction - Determines layout flow
  • Rotates cross axis - Always perpendicular to main
  • Affects item order - Can reverse visual order

flex-direction: row (Default)

Places all flex items on the same row, in the direction of your browser's default language (left to right):

CSS
main {
  display: flex;
  flex-direction: row; /* Default */
}

Common Flex Properties

Beyond flex-direction, there are several other important flex properties:

1 justify-content: Aligns items along the main axis
(horizontal alignment in row, vertical in column)
2 align-items: Aligns items along the cross axis
(vertical alignment in row, horizontal in column)
3 flex-wrap: Controls whether items wrap to new lines
(allows multiple rows or columns)
4 gap: Adds space between flex items
(modern property, very convenient!)

When to Use Flexbox

✓ Perfect For:

  • Navigation bars - Horizontal or vertical menus
  • Button groups - Evenly spaced buttons
  • Card layouts - Equal-height cards
  • Form layouts - Aligned labels and inputs
  • Centering elements - Easy vertical/horizontal centering
  • Footer layouts - Distributed footer content
  • One-dimensional layouts - Single row or column

When to Use Something Else:

  • CSS Grid - For two-dimensional layouts (rows AND columns)
  • Float - Generally avoid, Flexbox is better
  • Position - For overlapping elements or specific positioning

Practical Example: Navigation Bar

HTML
<nav>
  <a href="#">Home</a>
  <a href="#">About</a>
  <a href="#">Services</a>
  <a href="#">Contact</a>
</nav>
CSS
nav {
  display: flex;
  gap: 2rem;
  background: #333;
  padding: 1rem;
}

nav a {
  color: white;
  text-decoration: none;
}

Practical Example: Centering

Flexbox makes centering content incredibly easy!

CSS
.container {
  display: flex;
  justify-content: center; /* Horizontal centering */
  align-items: center; /* Vertical centering */
  height: 300px;
}
Perfectly Centered!

This is one of Flexbox's most popular uses! Before Flexbox, centering was notoriously difficult.

Best Practices

✓ Do These:

  • Use Flexbox for one-dimensional layouts
  • Combine with CSS Grid for complex layouts
  • Use gap instead of margins for spacing
  • Test responsive behavior at different screen sizes
  • Consider accessibility - visual order vs DOM order
  • Use semantic HTML with Flexbox

❌ Avoid These:

  • Using Flexbox for entire page layouts (use Grid instead)
  • Relying on visual order instead of DOM order
  • Over-nesting flex containers
  • Using only for centering (though it's great at it!)

Quick Reference

Basic Flexbox Setup
/* Create flex container */
.container {
  display: flex;
}

/* Control direction */
flex-direction: row; /* Default: horizontal → */
flex-direction: row-reverse; /* Horizontal ← */
flex-direction: column; /* Vertical ↓ */
flex-direction: column-reverse; /* Vertical ↑ */

/* Common properties */
justify-content: center; /* Align on main axis */
align-items: center; /* Align on cross axis */
gap: 1rem; /* Space between items */
flex-wrap: wrap; /* Allow wrapping */

Key Takeaways

What Flexbox Is:

  • One-dimensional layout model - Row OR column at a time
  • Created with display: flex
  • Items can shrink or expand automatically
  • Perfect for responsive layouts

Two Key Concepts:

  • Flex Container: Parent element with display: flex
  • Flex Items: Direct children of flex container

The Flex Model:

  • Main Axis: Primary direction (controlled by flex-direction)
  • Cross Axis: Perpendicular to main axis
  • Axes determine how properties like justify-content and align-items work

Remember:

Flexbox is a powerful and flexible layout model that provides an efficient way to arrange elements within a container. By understanding flex containers, flex items, and the flex model, you can create dynamic and responsive websites that adapt to different screen sizes!
