What is an iframe?

The iframe element stands for inline frame. It's an inline element used to embed other HTML content directly within an HTML page.

You can embed various types of content:

  • Videos from services like YouTube and Vimeo
  • Maps
  • Other web pages
  • Direct HTML content
Basic iframe Structure
<iframe
  src="URL"
  width="400"
  height="400"
  title="Description"
></iframe>

Embedding Videos

You can embed videos from platforms like YouTube directly into your page using an iframe.

The video source can come from any service—it doesn't have to be limited to YouTube or Vimeo.

YouTube Video Example
<iframe
  width="400"
  height="400"
  src="https://www.youtube.com/embed/
       PkZNo7MFNFg"
  title="Learn JavaScript - 
         Full Course for Beginners"
  allow="accelerometer; autoplay; 
          clipboard-write; encrypted-media; 
          gyroscope"
  allowfullscreen
></iframe>

Key iframe Attributes

  • src - Specifies the URL of the page you want to embed
  • width - Specifies the width of the iframe
  • height - Specifies the height of the iframe
  • allowfullscreen - Allows the user to display the iframe in full screen mode
  • title - Provides a description for accessibility (good practice to include)
Attributes in Action
<iframe
  <!-- Location of content -->
  src="https://example.com"
  
  <!-- Dimensions -->
  width="600"
  height="400"
  
  <!-- Accessibility -->
  title="Description of content"
  
  <!-- Enable fullscreen -->
  allowfullscreen
></iframe>

Embedding Maps

You can also embed maps, other web pages, or direct HTML within the iframe element.

Maps from services like OpenStreetMap can be embedded and users can interact with them by zooming in and out.

Map Embedding Example
<h1>A Map from Openstreetmap.org</h1>

<iframe
  width="425"
  height="350"
  src="https://www.openstreetmap.org/
       export/embed.html?bbox=..."
  title="Map of Lagos area, Nigeria"
  style="border: 1px solid black"
></iframe>

<br>
<small>
  <a href="...">View Larger Map</a>
</small>

Embedding Direct HTML

If you want to embed direct HTML content within the iframe element, you must use the srcdoc attribute instead of src.

This allows you to write HTML directly in the attribute rather than referencing an external URL.

Using srcdoc for HTML
<iframe
  srcdoc="<h1>Hello World</h1>
           <p>Direct HTML content</p>"
  width="400"
  height="200"
  title="Direct HTML example"
></iframe>
Key Difference
<!-- Use src for URLs -->
src="https://example.com"

<!-- Use srcdoc for HTML -->
srcdoc="<h1>HTML Content</h1>"
