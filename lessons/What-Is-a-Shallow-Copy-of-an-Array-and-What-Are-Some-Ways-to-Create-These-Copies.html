The includes() Method

What Is includes()?

The includes() method is a simple and efficient way to check if an array contains a specific value.

This method returns a boolean value:

  • true if the array contains the specified element
  • false if the element is not found

Basic Syntax

array.includes(searchElement, fromIndex)
Key Benefit:

The includes() method is particularly useful when you need to quickly verify the presence of an element without needing to know its exact position.

Basic Usage Examples

Checking for Elements

let fruits = ["apple", "banana", "orange", "mango"];

console.log(fruits.includes("banana")); // true
console.log(fruits.includes("grape"));  // false
true
- "banana" is in the array

false
- "grape" is not in the array

Case Sensitivity

String Comparisons Are Case-Sensitive

The includes() method is case-sensitive when dealing with strings.

"Banana" (capital B) and "banana" (lowercase) are considered different values. 

let fruits = ["apple", "banana", "orange"];

console.log(fruits.includes("banana")); // true
console.log(fruits.includes("Banana")); // false
true
- "banana" matches exactly

false
- "Banana" does not match (different case)

✓ Found

Array: ["apple", "banana", "orange"]
Search: "banana"
Result: true

✗ Not Found

Array: ["apple", "banana", "orange"]
Search: "Banana"
Result: false
Remember:

Always match the exact case of the string when using includes(). Even a single letter's capitalization difference will result in false.

Starting Position: fromIndex

Optional Second Parameter

The includes() method accepts an optional second parameter that specifies the position to start the search.

This is useful if you want to check for an element's presence in a specific part of the array. 

let numbers = [10, 20, 30, 40, 50, 30, 60];

console.log(numbers.includes(30, 3)); // true
console.log(numbers.includes(30, 4)); // true
Both return true because 30 appears after the specified starting positions

Strict Equality (===)

Type Matters!

The includes() method uses strict equality comparison (===), which means it can distinguish between different types. 

let mixedArray = [1, "2", 3, "4", 5];

console.log(mixedArray.includes(2));    // false (number)
console.log(mixedArray.includes("2"));  // true (string)
false
- Searching for number 2, but array has string "2"

true
- Searching for string "2", which exists in array

Type Comparison

Array: [1, "2", 3, "4", 5]

Search for: 2 (number)

2 !== "2"
Number vs String
false

Search for: "2" (string)

"2" === "2"
String vs String
true
Key Concept:

The number 2 and the string "2" are considered different data types, so they are not equal under strict comparison.

Summary & Benefits

Why Use includes()?

  • Simple: Easy to use and understand
  • Efficient: Returns boolean for quick checks
  • Readable: Makes code more maintainable
  • Versatile: Works with strings, numbers, and mixed types
  • No loops needed: Saves you from writing complex conditions

Key Behaviors to Remember

Important Points:
✓ Returns boolean (true/false), not an index
✓ Case-sensitive for strings
✓ Uses strict equality (===) for comparison
✓ Can start search from specific position with fromIndex
✓ Distinguishes between different data types

Quick Reference

Syntax:
array.includes(searchElement, fromIndex)
searchElement: The value to search for
fromIndex: (Optional) Position to start searching

Common Use Cases

  • Validating user input against a whitelist
  • Checking if an item exists before processing
  • Filtering or conditional logic based on array contents
  • Quick membership tests without needing the position
