Common Design Terms

As a developer, you may need to work closely with designers. If you understand basic design terminology, you'll have a shared vocabulary.

Benefits of Shared Design Language

  • Participate more actively in the design process
  • Make your workflow more efficient
  • Result in a better user experience

Layout

Layout is how the visual elements are arranged on a page or screen to communicate a message.

Elements Include

  • Text
  • Images
  • White space

The layout is like the blueprint of a design. Designers must consider the placement, size, and hierarchy of each element.

Alignment

Alignment is how the elements are placed in relation to one another.

Purpose

Using alignment correctly is helpful for making the design look clean and organized.

Designers create visual order by aligning elements along imaginary lines, edges, or a central point.

Alignment Examples

Left-Aligned

This content is aligned to the left. Most common for body text.

Center-Aligned

This content is centered. Useful for titles and headings.

Right-Aligned

This content is aligned to the right. Used for special emphasis.

Composition

Composition is the art of arranging elements to create a harmonious design.

It determines how elements like images, text, and shapes relate to each other and contribute to the design in an artistic way.

Layout vs Composition

  • Layout: Focuses on the placement of elements
  • Composition: Also considers the artistic impact that placement will have in the overall design

Balance

Balance is how the visual weight is distributed within a composition.

Designers aim to create an equilibrium through symmetrical or asymmetrical arrangements.

Hierarchy

Hierarchy establishes the order of importance of the elements in a design.

It's about making sure the most important information is noticed first.

Implement Hierarchy With

  • Size
  • Color
  • Contrast
  • Alignment
  • White space
  • Typography

Contrast

Contrast creates clear distinctions between elements.

Contrast is helpful for guiding user attention to what you want to emphasize.

Create Contrast Through

  • Variations in color
  • Differences in size
  • Changes in shape
  • Variations in texture
  • Any other visual characteristic

Strong contrast is also helpful for improving readability.

Contrast Examples

White Space (Negative Space)

White Space, also known as "negative space", is the empty space in a design.

It's the area surrounding the elements.

Important to Know

White space is not necessarily white. It can be space in any color or texture.

Purpose of White Space

White Space Helps To:

  • Improve readability
  • Enhance visual hierarchy
  • Create focus on important elements
  • Make designs feel more professional and clean

UI (User Interface)

The user interface is how humans interact with computers.

UI Includes

The visual and interactive elements that users can see on their screens:

  • Icons
  • Images
  • Text
  • Menus
  • Links
  • Buttons

UX (User Experience)

The user experience is about how users feel when using a product or service.

Well-Designed UX Is:

  • Intuitive - Easy to understand
  • Easy to use - Minimal learning curve
  • Efficient - Tasks completed quickly
  • Accessible - Usable by everyone
  • Enjoyable - Pleasant to interact with

UI vs UX

The Relationship

The user interface plays a key role in making the user experience as easy and enjoyable as possible.

UI and UX are very closely related.

Simple Analogy

  • UI: What you see and interact with (the buttons, colors, layout)
  • UX: How you feel about the interaction (frustrated, delighted, confused)

Design Terms Summary

Key Terms to Remember

  • Layout - Arrangement of visual elements
  • Alignment - Elements in relation to each other
  • Composition - Artistic arrangement of elements
  • Balance - Visual weight distribution
  • Hierarchy - Order of importance
  • Contrast - Distinctions between elements
  • White Space - Empty space around elements
  • UI - How humans interact with computers
  • UX - How users feel when using a product

Take an Active Role

With this knowledge of common design terms, you can now take a more active role in the design process.

Benefits

  • Better communication with designers
  • More effective collaboration
  • Improved end products
  • Stronger team synergy

Understanding these terms bridges the gap between development and design!
