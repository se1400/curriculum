What is JavaScript?

JavaScript is a powerful programming language that brings interactivity and dynamic behavior to websites.

HTML & CSS

Markup languages used to:

  • Structure content (HTML)
  • Style elements (CSS)

JavaScript

Programming language that goes beyond markup to enable:

  • Complex functionality
  • Interactive behavior
  • Dynamic responses

What JavaScript Can Do

JavaScript enables complex functionality that makes websites come alive:

Interactive Features

  • Handle user input: Respond to keyboard typing, mouse clicks, and touch gestures
  • Animate elements: Create smooth transitions and visual effects
  • Build web applications: Create complete, interactive programs in the browser

How HTML, CSS, and JavaScript Work Together

Each technology has a specific role in creating a complete web experience:

HTML - Structure

Defines the content and elements on the page

<h1>Hello, World!</h1>
<button>Click me</button>

CSS - Style

Applies visual styling to elements

h1 {
  color: green;
}

JavaScript - Behavior

Adds interactivity and dynamic functionality

onclick="alert('Button clicked!')"

Complete Working Example

Here's a complete example showing all three technologies working together:

<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>

How This Code Works:

  • HTML defines the content: a heading (h1 element) and a button
  • CSS applies styles: makes the heading text green
  • JavaScript enables behavior: displays an alert when button is clicked
