Text Shadows in CSS

CSS doesn't apply any shadows to text by default.

Hello, World!

Example paragraph without any shadows

To add shadows to text, use the text-shadow property

text-shadow Syntax

In CSS, you can describe a shadow through a combination of values.

Basic Syntax:

text-shadow: /* Values */;

Shadow Components:

  • X offset (horizontal distance)
  • Y offset (vertical distance)
  • Blur radius (optional)
  • Color

X and Y Offset

The X offset and Y offset represent the horizontal and vertical distance of the shadow from the text.

REQUIRED

X Offset

Horizontal distance of shadow from text

Y Offset

Vertical distance of shadow from text

Example:

p {
  text-shadow: 3px 2px;
}

X offset: 3px, Y offset: 2px

Basic Offset Example

Applying an X offset of 3px and Y offset of 2px:

p {
  text-shadow: 3px 2px;
}
Hello, World!

The text and shadow look similar. We need to add color to distinguish them.

Adding Color to Shadows

Customize the shadow color by specifying the value before or after the offset.

Color After Offset:

p {
  text-shadow: 3px 2px #4a90e2;
}

Color Before Offset (Equivalent):

p {
  text-shadow: #4a90e2 3px 2px;
}

You can use any valid color format: hexadecimal, RGB, RGBA, named colors, etc.

Colored Shadow Example

With a blue shadow color, we can easily distinguish the shadow from the text:

p {
  text-shadow: 3px 2px #4a90e2;
}
Hello, World!

Now the shadow has a specific color, making it much easier to see the shadow effect.

  • Positive X: Shadow moves right
  • Negative X: Shadow moves left
  • Positive Y: Shadow moves down
  • Negative Y: Shadow moves up

Negative Values Example

Using negative values moves the shadow left and up:

p {
  text-shadow: -3px -2px #4a90e2;
}
Hello, World!

The shadow now appears in the upper-left direction relative to the text.

Blur Radius

The blur radius is the third value that makes the shadow look smoother and more subtle.

OPTIONAL

How Blur Radius Works:

  • Default value: 0 (no blur)
  • Higher value: Bigger blur
  • Effect: Shadow becomes lighter and more diffused

Syntax:

text-shadow: x-offset y-offset blur-radius color;

Blur Radius Example

Adding a blur radius of 3px makes the shadow look much better:

p {
  text-shadow: 3px 2px 3px #4a90e2;
}
Hello, World!

Now it's starting to look like a real shadow—more blurry and subtle, allowing us to focus on the main text while the shadow adds style in the background.

Multiple Shadows

Text can have more than one shadow. Write them in the same text-shadow property separated by commas.

Layering:

Shadows are applied in layers, from front to back, with the first shadow at the top.

Syntax:

text-shadow:
  shadow1,
  shadow2,
  shadow3;

Multiple Shadows Example

Three shadows applied in layers:

p {
  text-shadow:
    2px 2px 3px rgba(0, 0, 0, 0.3),
    4px 4px 5px rgba(0, 0, 0, 0.2),
    6px 6px 7px rgba(0, 0, 0, 0.1);
}
Hello, World!

Notice how all shadows are defined in the same property, separated by commas.

The Power of text-shadow

text-shadow is a powerful CSS property for:

  • Making text stand out
  • Enhancing overall design of web applications
  • Creating depth and dimension
  • Adding visual interest
  • Improving readability in certain contexts
By choosing the right offset, blur, and color, you can create a wide range of shadow effects

text-shadow Summary

Key Takeaways:

  • No default shadows: CSS doesn't apply shadows by default
  • X and Y offset: Required values for shadow position
  • Positive values: Move shadow right and down
  • Negative values: Move shadow left and up
  • Color: Can be before or after offset values
  • Blur radius: Optional third value for smoother shadows
  • Multiple shadows: Separate with commas, layered front to back

Complete Syntax:

text-shadow: x-offset y-offset blur-radius color;
