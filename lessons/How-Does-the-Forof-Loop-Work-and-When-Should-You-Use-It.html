The For...of Loop

What Is a For...of Loop?

A for...of loop is used when you need to loop over values from an iterable.

What Are Iterables?

Iterables are data structures that can be looped over. Examples include:

  • Arrays - Collections of values
  • Strings - Sequences of characters

Basic Syntax

for (variable of iterable) { // code block to be executed }
Key Concept:

The variable represents the current value of the iterable being looped over.

• For an array of numbers → variable is the current number
• For a string → variable is the current character

Looping Over Arrays

Array Example

Let's loop over an array of numbers and log each one to the console: 

const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
    console.log(num);
}
Output:
1
2
3
4
5

How It Works

We create a variable called num that represents the current number in the array:

Iteration 1: num = 1 → console.log(1)
Iteration 2: num = 2 → console.log(2)
Iteration 3: num = 3 → console.log(3)
Iteration 4: num = 4 → console.log(4)
Iteration 5: num = 5 → console.log(5)
Notice:

The for...of loop automatically goes through each element in the array. You don't need to manage an index counter like in a traditional for loop!

Looping Over Strings

String Example

We can also loop over each character in a string: 

const str = 'freeCodeCamp';

for (let char of str) {
    console.log(char);
}
Output:
f
r
e
e
C
o
d
e
C
a
m
p

Character by Character

The variable char represents each character:

Iteration 1: char = 'f'
Iteration 2: char = 'r'
Iteration 3: char = 'e'
... and so on for each character
Use Case:

Looping over strings is useful for character-by-character processing, such as counting vowels, validating input, or transforming text.

Using let vs const

Both Work in For...of Loops

You can use either let or const when declaring the variable in a for...of loop.

✓ Using const (Read-Only)

const colors = ['red', 'blue'];

for (const color of colors) {
    console.log(color);
    // No reassignment
}

Works fine because we're not changing the value of color

✓ Using let (Can Modify)

const numbers = [1, 2, 3];

for (let num of numbers) {
    num = num * 2;
    console.log(num);
}

Works because let allows reassignment

⚠️ Important Rule:

If you use const, make sure the value of the variable does not change inside the loop. If it does, you will get an error!

⚠️ const Reassignment Error

What Happens When You Try to Change const?

❌ This Will Cause an Error: 
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
    console.log(num);
    num = num + 1; // ❌ This will cause an error!
}
TypeError: Assignment to constant variable

The Solution: Use let

✓ Correct Way:

const numbers = [1, 2, 3, 4, 5];

for (let num of numbers) {
    console.log(num);
    num = num + 1; // ✓ Works fine with let
}
When to Use Each:
• Use const when you're only reading the value
• Use let when you need to modify the value inside the loop

Looping Over Array of Objects

More Complex Data Structures

The for...of loop works great with arrays of objects: 

const people = [
    { name: 'John', age: 30 },
    { name: 'Jane', age: 25 },
    { name: 'Jim', age: 40 }
];

for (const person of people) {
    console.log(`${person.name} is ${person.age} years old`);
}
Output:
John is 30 years old
Jane is 25 years old
Jim is 40 years old

How It Works

The variable person represents each object in the array:

Iteration 1:
person = { name: 'John', age: 30 }
Output: "John is 30 years old"
Iteration 2:
person = { name: 'Jane', age: 25 }
Output: "Jane is 25 years old"
Iteration 3:
person = { name: 'Jim', age: 40 }
Output: "Jim is 40 years old"
Accessing Object Properties:

Inside the loop, you can access any property of the current object using dot notation (e.g., person.name, person.age).

When to Use For...of Loops

Benefits of For...of Loops

  • Easy to read: Clear and concise syntax
  • No index management: Automatically iterates through values
  • Works with iterables: Arrays, strings, and more
  • Clean code: Makes your code more maintainable
  • Direct access: Get the value directly, not the index

Quick Reference

// Basic syntax for (variable of iterable) { // code block } // With array for (const item of array) { } // With string for (const char of string) { }

Key Takeaways

Remember:
✓ Use const when only reading values
✓ Use let when modifying values inside the loop
✓ Variable represents the current value (not the index)
✓ Perfect for looping over arrays and strings
✓ More readable than traditional for loops for simple iterations
Best Practice:

Use for...of loops when you need to access the values of an iterable. If you need the index as well, consider using a traditional for loop or forEach() with index parameter.
