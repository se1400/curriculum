Testing for Substrings

When working with strings in JavaScript, there are many cases where you might need to check whether a string contains a specific substring.

What is a Substring?

A substring is a smaller part of a larger string.

Common Use Cases

You might want to check if a user's input includes a specific word or character before performing some action:

  • Validate that an email contains "@"
  • Check if a message contains profanity
  • Verify that a password includes special characters
  • Search for keywords in text

The includes() Method

includes()

One way to check if a string contains a substring is by using the includes() method.

What It Does

The includes() method is used to check if a string contains a specific substring.

If Found

true

Returns true if the substring is found

If Not Found

false

Returns false if the substring is not found

Basic Syntax

Here's the basic syntax for includes():

string.includes(searchValue);

The searchValue Parameter

The searchValue is the substring you want to look for within the string.

Simple Examples

// Check if "hello" contains "ell"
"hello".includes("ell");  // true

// Check if "world" contains "xyz"
"world".includes("xyz");  // false

Finding a Substring

Here's an example where the substring is found:

let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");
console.log(result); // true
true

‚úì Substring Found

In this example, the word "awesome" is found within the string "JavaScript is awesome!"

So the includes() method returns true.

includes() is Case Sensitive

‚ö†Ô∏è Important Note

It's important to note that the includes() method is case-sensitive.

This means that the exact match of the characters is required, including their case.

Example: Case Mismatch

let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");
console.log(result); // false
false

‚ùå Why false?

Since "Awesome" (with an uppercase A) does not match "awesome" (with a lowercase a), the result is false.

Case Sensitivity Examples

Let's look at more examples to understand case sensitivity:

‚ùå Won't Match

"Hello World".includes("hello")
// false

"JavaScript".includes("JAVA")
// false

"Programming".includes("GRAM")
// false

‚úì Will Match

"Hello World".includes("Hello")
// true

"JavaScript".includes("Java")
// true

"Programming".includes("gram")
// true

üí° Remember

Every letter must match exactly, including uppercase and lowercase!

Specifying a Starting Position

You can also use includes() to check for a substring starting at a specific index by providing a second parameter.

Syntax with Starting Position

string.includes(searchValue, startPosition);

Example: Search from Position 7

let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);
console.log(result); // true
true

What Happened?

The search for the substring "JavaScript" starts from the 7th position in the string.

This ensures it skips any characters before this position.

Understanding Starting Position

Let's visualize how the starting position works:

let text = "Hello, JavaScript world!";
//          0123456789...
let result = text.includes("JavaScript", 7);
console.log(result); // true

Breaking It Down

  • Position 0-6: "Hello, " - SKIPPED
  • Position 7+: Search begins here
  • "JavaScript" is found starting at position 7
  • Result: true

Another Example: Not Found After Position

let text = "Hello, JavaScript world!";
let result = text.includes("Hello", 7);
console.log(result); // false

This returns false because "Hello" appears before position 7, so it's not found when searching from position 7 onward.

includes() vs indexOf()

The includes() method only returns a true or false result.

‚ö†Ô∏è Limitation

It does not provide information on:

  • Where the substring is located in the string
  • How many times it occurs

If you need that level of detail, other methods, such as the indexOf() method, might be more suitable.

Method Returns Use When
includes() Boolean (true/false) You just need to know IF substring exists
indexOf() Number (index or -1) You need to know WHERE substring is located

Practical Use Cases

Example 1: Email Validation

let email = "user@example.com";
if (email.includes("@") && email.includes(".")) {
  console.log("Valid email format");
} else {
  console.log("Invalid email format");
}

Example 2: Checking for Keywords

let message = "This is urgent! Please respond ASAP.";
if (message.includes("urgent") || message.includes("ASAP")) {
  console.log("Priority message detected");
}

Example 3: File Extension Check

let filename = "document.pdf";
if (filename.includes(".pdf")) {
  console.log("This is a PDF file");
} else if (filename.includes(".jpg") || filename.includes(".png")) {
  console.log("This is an image file");
}

Example 4: Password Validation

let password = "SecureP@ss123";
let hasSpecialChar = password.includes("@") || 
                     password.includes("!") || 
                     password.includes("#");
if (hasSpecialChar) {
  console.log("Password contains special character");
}

includes() Summary

Key Takeaways

What is includes()?

A string method that checks if a string contains a specific substring.

Syntax

string.includes(searchValue)
string.includes(searchValue, startPosition)

Parameters

  • searchValue: The substring to search for (required)
  • startPosition: Index to start searching from (optional)

Return Values

  • true - If substring is found
  • false - If substring is not found

Important Notes

  • includes() is case-sensitive
  • Returns only a boolean (true/false)
  • Does not tell you WHERE the substring is
  • Does not tell you HOW MANY times it appears
  • Use indexOf() if you need position information

Common Use Cases

  • Checking if input contains specific words
  • Validating email/password formats
  • Searching for keywords in text
  • File type/extension detection
  • Simple substring existence checks
