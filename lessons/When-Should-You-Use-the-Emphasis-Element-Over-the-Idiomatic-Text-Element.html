The i Element (Idiomatic Text)

The idiomatic text element (i) was originally used for presentational purposes to display text in italics.

Now, it is frequently used for highlighting:

  • Alternative voice or mood
  • Idiomatic terms from another language
  • Technical terms
  • Thoughts

The i element does not indicate if the text is important or not—it only shows that it's somehow different from the surrounding text.

Idiomatic Text Example
<p>
  There is a certain 
  <i lang="fr">je ne sais quoi</i> 
  in the air.
</p>

/* Shows text is different,
   but NOT important */

Using the lang Attribute

The lang attribute inside the opening <i> tag is used to specify the language of the content.

This is especially useful when using the i element for idiomatic phrases from another language, as it provides additional context for browsers and assistive technologies.

Language Specification
<!-- French phrase -->
<i lang="fr">je ne sais quoi</i>

<!-- Spanish phrase -->
<i lang="es">hasta la vista</i>

<!-- German phrase -->
<i lang="de">zeitgeist</i>

/* Helps screen readers pronounce
   correctly */

The em Element (Emphasis)

If you need to emphasize the importance of the text, you can use a similar semantic element called the emphasis element (em).

This is usually recommended if you need to provide more context. You should use this element for parts of the text that require a special emphasis compared to surrounding text.

It's usually limited to only a few words, because it can alter the meaning of the sentence.

Emphasis for Importance
<p>
  Never give up on <em>your</em> dreams.
</p>

/* "your" is emphasized because
   it's important to the meaning */
How Emphasis Changes Meaning
Different emphasis, different meaning:

<em>I</em> didn't say that.
→ Someone else said it

I <em>didn't</em> say that.
→ I'm denying it

I didn't <em>say</em> that.
→ I implied it another way

Visual vs Semantic Meaning

In the browser, you would see the word "your" italicized to tell readers that this is an important word in the sentence.

Even if it looks the same when the text was inside the idiomatic text element, the semantic emphasis element conveys its meaning and importance behind the scenes.

This semantic meaning is crucial for assistive technologies like screen readers.

Both Look the Same
/* Visual appearance: */

<i>italicized text</i>
<em>italicized text</em>

Both display as italics ↑
But Different Meaning
/* Semantic meaning: */

<i>text</i>
→ Text is different
→ No importance indicated

<em>text</em>
→ Text is emphasized
→ Importance conveyed
→ Screen readers add vocal stress

When to Use CSS Instead

These elements should not be used for presentational purposes only.

If you need to display the text in italics, but the text doesn't have a special purpose, style, or meaning in the paragraph, you should use CSS instead.

Always ask: "Does this text have semantic meaning, or do I just want it styled differently?"

Wrong: Using HTML for Style Only
<!-- Just want italics for style -->
<p>
  This is <i>fancy</i> text.
</p>

✗ "fancy" has no special meaning,
  just styled differently
Right: Using CSS for Style
<!-- Use CSS for pure styling -->
<p>
  This is <span class="fancy">fancy</span> text.
</p>

/* CSS */
.fancy {
  font-style: italic;
}

✓ No misleading semantics

Summary: i vs em

Use i to indicate that text differs from surrounding content without conveying specific importance (foreign phrases, technical terms, thoughts).

Use em to emphasize important text that requires special emphasis compared to surrounding text.

Use CSS when you just want to display text in italics for presentational purposes only, without any semantic meaning.

Quick Reference
/* i = Different, not important */
<i lang="fr">bon appétit</i>
<i>Homo sapiens</i>
<i>I thought to myself...</i>

/* em = Emphasized, important */
Never <em>ever</em> give up!
This is <em>critical</em> information.

/* CSS = Style only */
<span class="italic">styled text</span>
