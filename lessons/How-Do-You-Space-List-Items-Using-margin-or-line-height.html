Spacing List Items

Margins and line-height are essential for spacing list items to enhance readability and visual appeal.

Two Main Approaches

  • Margins: Control space between list items
  • Line-height: Control space within list items

Default List Spacing

By default, HTML does not apply much spacing between list items.

Basic HTML

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Default Output

  • Item 1
  • Item 2
  • Item 3

Notice the minimal spacing between items.

Method 1: Using Margins

Margins create space between list items by applying margin properties to the li elements.

This method controls the spacing outside each list item, effectively increasing or decreasing the gap between them.

Key Property

margin-bottom - Adds space below each list item

Margin Example

CSS Code

ul li {
  margin-bottom: 40px;
}

HTML

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Result

  • Item 1
  • Item 2
  • Item 3

40px of margin is applied to the bottom of each list item inside the unordered list.

Method 2: Using Line-Height

The line-height property adjusts the vertical spacing between lines of text within a single list item.

Important Distinctions

  • Primarily affects spacing between lines of text within each item
  • Can indirectly influence overall spacing if items contain only a single line
  • Does not directly adjust spacing between separate list items

Line-Height: Single Line Items

When list items have only single lines of text, line-height can affect the overall spacing between items.

CSS Code

ul li {
  line-height: 2;
}

line-height: 2 sets the line height to be twice the font size.

Result

  • Item 1
  • Item 2
  • Item 3

Creates more vertical space within each list item.

Line-Height: Multi-Line Items

If list items have multiple lines of text, the line-height will affect the spacing between those lines.

Key Point

line-height does not directly adjust the spacing between separate list items themselves.

It only controls the spacing within each individual item's text content.

Choosing the Right Method

Use Margin

  • To control spacing between individual list items
  • For consistent gaps between items
  • When you need precise control over item separation

Use Line-Height

  • To adjust spacing within multi-line list items
  • For better readability of text content
  • When items contain paragraphs or multiple lines

Best Practice

To control the spacing between individual list items, use margin or padding properties.
