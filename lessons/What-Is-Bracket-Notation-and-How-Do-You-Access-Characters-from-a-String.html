What is Bracket Notation?

In JavaScript, strings are treated as sequences of characters, and each character in a string can be accessed using bracket notation.

What It Does

Bracket notation allows you to retrieve a specific character from a string based on its position, which is called its index.

Basic Syntax

string[index]

Uses square brackets [] with the index of the character you want to access.

Understanding Index

What is an Index?

An index is the position of a character within a string, and it is zero-based.

Zero-Based Indexing

This means that:

  • The first character has an index of 0
  • The second character has an index of 1
  • The third character has an index of 2
  • And so on...

Using Bracket Notation

Let's see bracket notation in action:

let greeting = "hello";
console.log(greeting[1]); // "e"

Breaking It Down

In this example, we can access the character at index 1, which is "e".

String: "hello"

Index: [1]

Result: "e"

Output: "e"

Accessing the Last Character

To get the last character of a string, you can use the length of the string minus one.

Why Minus One?

The length property tells you how many characters the string contains.

Since indexing starts at 0, the last character is always at position length - 1.

Example

let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"

How It Works

The length of "hello" is 5

The last character ("o") is at index 4, which is 5 - 1

Output: "o"

Getting Multiple Characters

If you want to get multiple characters, you can use bracket notation with concatenation:

let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);

How It Works

We are concatenating the first and second characters using bracket notation:

  • greeting[0] gives us "h"
  • greeting[1] gives us "e"
  • The + operator combines them into "he"
Output: "he"

Practical Uses of Bracket Notation

Bracket notation is useful when you need to access specific characters in a string.

Extracting Initials

let firstName = "Alice";
let lastName = "Smith";
let initials = firstName[0] + lastName[0];
console.log(initials); // "AS"

Checking First Letter

let word = "Hello";
if (word[0] === "H") {
  console.log("Starts with H!");
}

Common Use Cases

  • Extracting initials from names
  • Checking specific letters for validation
  • Accessing individual characters for analysis
  • Building new strings from specific positions

More Bracket Notation Examples

Example 1: Accessing Different Positions

let word = "JavaScript";
console.log(word[0]);  // "J" - first character
console.log(word[4]);  // "S" - fifth character
console.log(word[word.length - 1]); // "t" - last character

Example 2: Building a New String

let original = "JavaScript";
let rearranged = original[4] + original[0] + original[9] + original[1];
console.log(rearranged); // "SJta"

Example 3: Getting Middle Characters

let word = "hello";
let middle = word[1] + word[2] + word[3];
console.log(middle); // "ell"

Bracket Notation Summary

Key Takeaways

What is Bracket Notation?

A way to access individual characters in a string using square brackets [] and an index number.

Zero-Based Indexing

String indices start at 0, not 1:

  • First character: index 0
  • Second character: index 1
  • Third character: index 2

Common Patterns

First character: string[0]
Last character: string[string.length - 1]
Multiple characters: string[0] + string[1]

When to Use It

  • Extracting initials from names
  • Checking specific letters for validation
  • Accessing individual characters for processing
  • Building new strings from specific positions
