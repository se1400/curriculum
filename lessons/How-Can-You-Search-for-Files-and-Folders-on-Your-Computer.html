Searching for Files on Windows

You can search for files and folders on Windows from two main locations: the Taskbar or File Explorer.

Both methods provide quick access to your files across your PC and OneDrive.

Two Search Locations

  • Taskbar: Quick search from bottom of screen
  • File Explorer: Search within Explorer window

Searching from the Taskbar

To search from the Taskbar, write the name of the file or folder that you are looking for. You can also search for keywords.

You'll see results from across your PC and OneDrive.

If you only want to see documents, you can click on the "Documents" tab. You also have tabs for apps and web results.

How to Search

  1. Click the search box on the Taskbar
  2. Type the file name, folder name, or keywords
  3. View results from PC and OneDrive
  4. Filter by tabs: Documents, Apps, Web

Searching from File Explorer

Another alternative is to use the Search File Explorer.

You will find it at the top right of the Explorer window, where you can see "Search Documents."

This method gives you more control over the scope of your search.

File Explorer Search Location

Look for the search box at the top right of the Explorer window labeled "Search Documents".

Search Scope Options

You have three options to define the scope of the search in File Explorer.

Each option provides different levels of breadth and speed for your search.

Home

Find files from your PC and from the cloud quickly

Folder

Find files stored inside a specific folder

This PC

Slow but in-depth search across entire computer

Searching on macOS with Spotlight

First, you can use Spotlight, a tool that helps you find files on your Mac.

To open Spotlight, go to your menu bar at the top right and find the magnifying glass icon.

If you don't see this icon on the menu bar, you can add it in the Control Center settings.

Opening Spotlight

  • Click magnifying glass icon in menu bar (top right)
  • Or use keyboard shortcut: Cmd + Spacebar

Adding Spotlight to Menu Bar

Go to Settings → Control Center → Scroll to Spotlight option

Using Spotlight Search

When you open Spotlight, you'll see the Spotlight Search where you can search for any file or folder by typing its name.

When you start typing, you'll see the results grouped by category, including suggestions, folders, presentations, photos, documents, and related searches.

You can also expand a specific category by clicking on "Show More."

Search Results Categories

  • Suggestions
  • Folders
  • Presentations
  • Photos
  • Documents
  • Related searches

Click "Show More" to expand a category

Finding Recently Opened Files

Another way to search for files that you have recently opened is to use Finder.

Open Finder by clicking on the Finder icon in the Dock. Then, go to "Recents" in the Finder sidebar.

There, you'll find all the files that you've recently viewed.

Access Recent Files

  1. Click Finder icon in the Dock
  2. Go to "Recents" in the sidebar
  3. View all recently opened files

Optimizing Your Workflow

With these search tools, you can optimize your workflow and find the files and folders you need very quickly.

Understanding both Windows and macOS search capabilities helps you work efficiently on any platform.

Windows Search Tools

  • Taskbar search for quick access
  • File Explorer with scope options
  • Filter by Documents, Apps, Web

macOS Search Tools

  • Spotlight for system-wide search
  • Categorized search results
  • Finder Recents for recent files
