Quoted Elements in HTML

In HTML, quoted elements are used to distinguish quoted text from the surrounding content. This gives the quoted text a format that is easy to identify.
Purpose: Quoted elements provide semantic meaning to content that comes from another source, making it clear to both readers and search engines that the text is a quotation.
There are two main types of quotation elements in HTML:

Block Quotation

For extended quotations from other sources

Element: <blockquote>

Inline Quotation

For short quotations within paragraphs

Element: <q>

Block Quotation Element

You should use the block quotation element (<blockquote>) for representing a section quoted from another source. It's mainly used for extended quotations.

Example: Simple Block Quote

<blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
  "Can you imagine what it would be like to be a successful developer?
  To have built software systems that help people?"
</blockquote>
Rendered Output:
"Can you imagine what it would be like to be a successful developer? To have built software systems that help people?"
This element has a cite attribute. The value of the cite attribute is the URL of the source.
Important: While the cite attribute doesn't change the presentation of the block quote, it's very helpful for giving screen readers and search engines more information about the quote. In the browser, you'll see that the text is slightly indented.

Multiple Paragraphs & Citation Element

You can write text directly within the block quotation element, or wrap it within paragraph elements. This is helpful when the text has multiple paragraphs, but you want to keep them within the same block quote.

Example: Block Quote with Multiple Paragraphs

<blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
  <p>Build your projects. Show them to your friends.</p>
  <p>Build your network. Help the people you meet along the way.</p>
  <p>It is not too late. Life is long.</p>
  <p>You will look back on this moment years from now.</p>
</blockquote>
Rendered Output:

Build your projects. Show them to your friends. Build projects for your friends.

Build your network. Help the people you meet along the way. What goes around comes around.

It is not too late. Life is long.

You will look back on this moment years from now and be glad you made a move.

All the paragraphs are contained within the same block quotation element, so they are part of the same quotation. The paragraphs are aligned relative to each other, but they are indented relative to their container.

Citation Element for Visual Attribution

The cite attribute doesn't show the source to the user - it only works behind the scenes. If you want to attribute the source visually, you can add a citation element (<cite>).
Important Distinction: The <cite> element is different from the cite attribute. The citation element is an HTML element used to mark up the title of a referenced creative work like a book, article, song, film, website, or research paper.

Inline Quotation Element

Sometimes you will only need to quote a few words within a larger paragraph. That's exactly what the inline quotation element (<q>) is for.
It's for short inline quotations from other sources. Most modern browsers will add quotation marks around the inline quote automatically when you use this element.

Example: Inline Quote

<p>
  As Quincy Larson said,
  <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    Momentum is everything.
  </q>
</p>
Rendered Output:

As Quincy Larson said, Momentum is everything.

You can also add a cite attribute to attribute the source. This works exactly the same as it did with the block quotation element - there won't be any visual changes, but it will give screen readers and search engines more information.

Block Quote vs Inline Quote

Aspect Block Quote (<blockquote>) Inline Quote (<q>)
Use Case Extended quotations from other sources Short quotations within paragraphs
Display Block-level element, indented Inline element, part of paragraph
Quotation Marks Must be added manually if desired Added automatically by browser
