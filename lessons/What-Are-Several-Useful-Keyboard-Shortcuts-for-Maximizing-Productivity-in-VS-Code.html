Basic OS Shortcuts

You're likely already familiar with some of the basic shortcuts, which are inherited from your operating system.

These common shortcuts work in VS Code just like they do in other applications on your computer.

Some keyboard shortcuts may differ by operating system, particularly between Windows and Mac.

Save File

Windows:
Ctrl + S

Copy Text

Windows:
Ctrl + C

Paste Text

Windows:
Ctrl + V

Format Code

This shortcut runs your configured formatter (such as Prettier for JavaScript projects) on the currently opened file.

Automatic formatting helps maintain consistent code style and saves time on manual formatting.

Run Formatter

Windows:
Shift + Alt + F

Search and Replace

These shortcuts allow you to search through all files in your workspace and perform find-and-replace operations.

Searching across multiple files is essential for finding references and making project-wide changes.

Search Workspace

Windows:
Ctrl + Shift + F
Mac:
Cmd + Shift + F

Find and Replace

Windows:
Ctrl + Shift + H
Mac:
Cmd + Shift + H

Delete Current Line

If you need to remove a line of code, this shortcut will delete the entire line where your cursor is positioned.

This is faster than manually selecting and deleting text, especially when removing multiple lines.

Delete Line

Windows:
Ctrl + Shift + K
Mac:
Cmd + Shift + K

Toggle Sidebar

Need some extra room for all your code? This shortcut will hide the sidebar, which contains the file list and extensions menu.

Toggling the sidebar gives you more screen space to focus on your code.

Hide/Show Sidebar

Windows:
Ctrl + B
Mac:
Cmd + B

Zoom In and Out

Maybe you just can't see your code? These shortcuts will increase or decrease the scaling of the editor.

Adjusting the zoom level helps with readability and can be useful for presentations or different screen sizes.

Zoom In

Windows:
Ctrl + +
Mac:
Cmd + +

Zoom Out

Windows:
Ctrl + -
Mac:
Cmd + -

Command Palette

If you forget any of these shortcuts, you always have the command palette.

The command palette opens a searchable menu where you can access any VS Code command or feature.

With this knowledge and a little practice, you are well on your way to becoming a VS Code power user.

Open Command Palette

Windows:
Ctrl + Shift + P
Mac:
Cmd + Shift + P
