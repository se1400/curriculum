Getting Started with VS Code

To install Visual Studio Code, you'll want to visit their website to get the appropriate installer for your operating system.

Important: Make sure you visit the VS Code website, not the Visual Studio website. These are two different products!

VS Code vs Visual Studio

Visual Studio Code (VS Code) is a lightweight code editor that's free and works on Windows, Mac, and Linux.

Visual Studio is a full IDE primarily for Windows, used for larger enterprise development.

Installing on Windows

For Windows, you can download an EXE installer from the VS Code website.

Once you've downloaded the installer, simply run the file and the setup wizard will guide you through the installation process.

Windows Installation Steps

  • Visit the VS Code website
  • Download the Windows EXE installer
  • Run the downloaded file
  • Follow the setup wizard prompts
  • Complete the installation

Installing on Mac

For Mac, you can download a zip archive which contains a .app file that you can run.

Alternatively, you can install VS Code using Homebrew, a popular package manager for macOS.

Mac Installation Methods

Method 1: Download the .zip archive, extract it, and run the .app file

Method 2: Use Homebrew

Homebrew Command
brew install --cask visual-studio-code

Installing on Linux

On Linux, you can download a .deb or .rpm file directly from the VS Code website.

Once installed, you can launch the application using the appropriate executable or CLI command (typically code).

Linux Installation

  • Download .deb file (for Debian/Ubuntu-based systems)
  • Download .rpm file (for Red Hat/Fedora-based systems)
  • Install using your package manager
  • Launch via executable or code command
Launching VS Code from Terminal
code

Ready to Code

Once you have VS Code installed on your system, you're ready to start coding!

You can launch VS Code from your applications menu, desktop shortcut, or by using the code command in your terminal.

VS Code is now configured and ready for development work across multiple programming languages and frameworks.
