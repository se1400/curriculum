Screen Magnifiers

What Are Screen Magnifiers?
Screen magnifiers are tools that help people with low vision and other visual impairments better access digital content and the web.
Role in Digital Accessibility
These tools play a crucial role in making digital content accessible to users with visual impairments.

How Screen Magnifiers Work

Functionality
Screen magnifiers work by enlarging texts, graphics, and other elements on a computer or mobile device screen.
Key Features
  • Allow users to enlarge the display by more than 200%
  • Enable navigation using pointing devices or keyboard
  • Offer customizable zoom percentages in settings
  • Provide additional customization features

Primary Uses of Screen Magnifiers

Reading Text
Small fonts in documents or applications can be challenging for people with low vision. By enlarging the text, users can read emails, articles, and other content without straining their eyes.
Web Browsing Assistance
Screen magnifiers help users locate and click on buttons, links, and other interactive elements that might be difficult to see. This improved visibility ensures users can browse websites, fill out forms, and engage in online activities without difficulty.

Making Products Magnifier-Friendly

Software developers need to make their digital products accessible to people with low vision. Key considerations include:
Using scalable fonts so users can resize the page without breaking the layout
Ensuring the user interface adapts to different screen sizes through responsive design
Using high-contrast color schemes and customizable colors
Implementing a non-sticky and tiny navbar so users can still see content when using magnifiers
Using regular HTML text instead of images of text
Providing feedback directly next to the element that triggers it

Built-in Operating System Magnifiers

All major operating systems include at least one magnifier built in by their manufacturers:
macOS and iOS
Zoom
macOS: Settings → Accessibility → Zoom → Toggle "Use keyboard shortcuts to zoom"
iOS: Settings → Accessibility → Zoom
Android
Magnification
Settings → Special Function → Accessibility → Magnification (or search "Magnification" on settings homepage)
Windows
Magnifier
Settings → Ease of Access → Magnifier
Linux
Zoom or Magnifier
Varies by distribution
