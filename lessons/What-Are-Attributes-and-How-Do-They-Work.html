What Are Attributes?

Attributes are values placed inside the opening tag of an HTML element that provide additional information or specify behavior.

Basic Syntax:

<element attribute="value"></element>
  • Attribute name followed by = and value in quotes
  • Values can be strings or numbers

Link Attributes: href & target

The href attribute specifies the URL destination. The target attribute specifies where to open the link.

<a href="https://www.freecodecamp.org" target="_blank"> Visit freeCodeCamp </a>
  • Without href, the link won't function (no destination URL)
  • target="_blank" opens the link in a new browser tab
  • The href attribute is required for functional links

Image Attributes: src & alt

The src attribute specifies the image source. The alt attribute provides alternative descriptive text for the image.

<img src="https://se1400.github.io/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch.">
  • The src attribute is required (specifies which image to display)
  • The alt attribute is not required but recommended for accessibility

Accessibility:

Accessibility ensures everyone, including those with disabilities, can use and understand websites, apps, and physical spaces.

Boolean Attributes: checked

Some attributes have unique syntax. The checked attribute doesn't require a value.

<input type="checkbox" checked />

How Boolean Attributes Work:

  • If present, the checkbox is checked by default
  • If not present, the checkbox is unchecked
  • No value is needed—this is a boolean attribute

The type attribute specifies the input type (in this case, a checkbox). Inputs collect data from users.

Common Boolean Attributes

Several boolean attributes specify element states:

  • disabled – element is disabled
  • readonly – element is read-only
  • required – element is required

Example: Disabled Text Input

<input type="text" disabled>

With the disabled attribute present, users cannot interact with the input. Remove the attribute, and the input becomes functional.

Key Takeaways

  • HTML has many attributes to customize element behavior and appearance
  • Understanding attributes is essential for creating interactive and accessible web content
  • Some attributes are required (like href and src)
  • Boolean attributes don't need values—their presence determines behavior
  • You'll learn more attributes and how to use them effectively in upcoming lessons
