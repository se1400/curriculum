Understanding HTML Attributes

Learn how attributes provide additional information and control element behavior in HTML

What Are Attributes?

Attributes are values placed inside the opening tag of HTML elements

Basic Syntax:

<element attribute="value"></element>
  • Attribute name followed by =
  • Value in quotes (string or number)
  • Provides additional information about the element

Link Attributes: href & target

href - Specifies the URL destination

target - Defines where to open the link

<a href="https://www.example.com" target="_blank"> Visit Example </a>
  • href is required for links to function
  • target="_blank" opens in new tab

Image Attributes: src & alt

src - Specifies the image source

alt - Provides alternative descriptive text

<img src="https://example.com/image.jpg" alt="Description of image">
  • src is required
  • alt is recommended for accessibility

Accessibility Matters

Accessibility ensures everyone, including people with disabilities, can use websites and applications

  • Alternative text helps screen readers describe images
  • Proper attributes make content accessible to all users
  • Consider accessibility when building web content

Boolean Attributes

Boolean attributes don't require values - their presence determines behavior

The checked Attribute:

<input type="checkbox" checked />
  • Present = checkbox is checked by default
  • Absent = checkbox is unchecked
  • No value needed

Common Boolean Attributes

  • checked - Checkbox/radio is selected
  • disabled - Element cannot be interacted with
  • readonly - Element value cannot be changed
  • required - Field must be filled out

Example:

<input type="text" disabled>

Key Takeaways

  • Attributes customize element behavior and appearance
  • Some attributes are required (like href, src)
  • Boolean attributes don't need values
  • Understanding attributes is essential for interactive and accessible web content
  • Many more attributes to explore in web development
