The <address> Element

The contact address element is used to represent contact information for a section on a web page.

The <address> element is versatile and can be used for:

  • Business pages
  • Author pages
  • Personal sites

Semantic HTML

When building contact sections, use the semantic <address> element over a generic element like <div>.

Why Use Semantic Elements?

Better: Provides meaning and context

<address> - Clear purpose for contact info

Generic: No inherent meaning

<div> - Just a container

Basic Address Structure

A typical contact address includes company name, physical address, phone, and email information.

Line Breaks in Addresses

Use the <br> (line break) element to show divisions between street name, city, and country in physical addresses.

Company Contact Example
<address>
  <h2>Company Name</h2>
  <p>
    1234 Elm Street<br>
    Springfield, IL 62701<br>
    United States
  </p>
  <p>Phone: <a href="tel:+15555555555">
    +1 (555) 555-5555
  </a></p>
  <p>Email: <a href="mailto:contact@company.com">
    contact@company.com
  </a></p>
</address>
Browser Output

Company Name

1234 Elm Street
Springfield, IL 62701
United States

Phone: +1 (555) 555-5555

Email: contact@company.com

Phone Number Links

Use an anchor element with the href value set for telephone numbers using the tel: protocol.

The tel:+ value inside the href attribute creates a clickable link to initiate a phone call on devices that support it.

Phone Link Syntax
<a href="tel:+15555555555">
  +1 (555) 555-5555
</a>
How It Works

Desktop: May open a calling app (Skype, FaceTime, etc.)

Mobile: Opens phone dialer with number pre-filled

Format Breakdown
/* tel: protocol structure */

tel:+[country code][area code][number]

/* Example: US phone number */
tel:+15555555555

/* The + indicates international format */

Email Links (mailto)

Use an anchor element with the href value set to a mailto: link.

mailto: links are used to allow users to open a new email within their preferred email client.

Downside of mailto Links

Users often perceive mailto: links as spam. Unfortunately, many spammers use this option to send emails to users.

Keep this in mind when deciding whether to use mailto: links on your website.

Email Link Syntax
<a href="mailto:contact@company.com">
  contact@company.com
</a>
What Happens

When clicked, opens the user's default email client (Gmail, Outlook, Apple Mail, etc.) with a new message addressed to the specified email.

⚠ Spam Concern: Spammers often target mailto links. Users may hesitate to click them due to spam associations.
