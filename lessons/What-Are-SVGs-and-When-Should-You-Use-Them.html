Raster Images

First, you need to understand how images work. Common image formats like PNG and JPG are classified as raster formats.

This essentially means that they are pixel-based, with the data tracking the color value in each pixel.

Raster Characteristics

  • Pixel-based image format
  • Stores color data for each pixel
  • Examples: PNG, JPG, GIF
  • Resolution-dependent

Raster Image Limitations

A large downside of raster-based images is that they do not upscale well.

If you've ever tried to make a PNG larger, you may have seen that it becomes pixelated or blurry.

When you enlarge a raster image, the pixels are stretched, resulting in quality loss.

Scaling Issues

Problem: Enlarging raster images causes pixelation and blur
Why: Fixed pixel data cannot adapt to larger sizes
Result: Loss of image quality and clarity

What Are SVGs?

An SVG is a different kind of image. SVG stands for Scalable Vector Graphic.

A vector graphic tracks data based on paths and equations to plot points, lines, and curves.

What this really means is that a vector graphic, like an SVG, can be scaled to any size without impacting the quality.

Vector Characteristics

  • Path-based, not pixel-based
  • Uses mathematical equations
  • Plots points, lines, and curves
  • Resolution-independent
  • Scales infinitely without quality loss

SVG Benefits

SVGs specifically have the added benefit of storing data in XML.

This means you can use them directly in your code as raw HTML with the <svg> element.

It also means you can programmatically change the color of the image.

Key Advantages

  • Stores data in XML format
  • Can be used directly in HTML
  • Programmatically editable
  • Colors can be changed with code
  • Small file sizes
  • Perfect for icons and logos

SVG Elements - Container

The <svg> element is the container for the whole drawing.

It sets up the space where all the shapes appear. Everything you want to draw with SVG, such as circles, lines, or paths, goes inside the svg element.

Basic SVG Structure
<svg
  width="100"
  height="100"
  viewBox="0 0 100 100"
  xmlns="http://www.w3.org/2000/svg"
>
  <!-- Shapes go here -->
</svg>

SVG Elements - Shapes

The <circle> element is used to make circles.

The <path> element is used to draw complex shapes and curved lines.

Each SVG element has attributes that control its appearance and position within the drawing area.

Smiley Face Example
<svg width="100" height="100" viewBox="0 0 100 100">
  <!-- Face -->
  <circle cx="50" cy="50" r="45"
    fill="yellow" />
  <!-- Eyes -->
  <circle cx="35" cy="40" r="5"
    fill="black" />
  <circle cx="65" cy="40" r="5"
    fill="black" />
  <!-- Smile -->
  <path d="M35 65 Q50 80 65 65"
    stroke="black" fill="transparent" />
</svg>

SVG Icon Examples

SVGs are perfect for creating icons that look crisp at any size.

To change the color for any of these examples, update the value for the fill attribute to any named color like red, green, blue, yellow, etc.

Examples Shown

Star (gold), Heart (crimson), Checkmark (green)

SVG Icon Code

Here are examples of the star, heart, and checkmark icons in code.

Notice how simple the code is and how easy it is to change colors by modifying the fill attribute.

Star Icon
<svg width="50" height="50"
  viewBox="0 0 24 24" fill="gold">
  <path d="M12 2L14.9 8.6L22 9.3..." />
</svg>
Heart Icon
<svg width="50" height="50"
  viewBox="0 0 24 24" fill="crimson">
  <path d="M12 21.35L10.55 20.03..." />
</svg>

When to Use SVGs

A great use case is for icons. If you want to create custom bullet points, or add icons to your links to represent social media platforms, using SVGs is the best approach.

One of the most popular icon libraries, Font Awesome, uses SVG images for their icons.

Ideal SVG Use Cases

  • Icons (social media, UI elements)
  • Logos and branding
  • Custom bullet points
  • Illustrations and graphics
  • Responsive design elements
  • Animations

SVGs for Responsive Design

SVGs are also great for webpage logos, because they scale perfectly.

They allow you to adapt your layout to any responsive design you need.

Next time you have an SVG locally, try opening it with a text editor and playing with the code.

Responsive Benefits

  • Perfect scaling at any screen size
  • No quality loss on retina displays
  • Adapts to responsive layouts
  • Small file sizes improve performance
  • Works great for mobile and desktop
