Background Image Properties

When working with background images in CSS, you have several properties to control how images are displayed.

background-size

Controls the dimensions of the image

background-repeat

Controls if/how the image repeats

background-position

Controls where the image is placed

background-attachment

Controls scrolling behavior

The background-image Property

First, let's look at how to set a background image:

body {
  background-image: url("image.jpg");
}

This CSS sets a background image for the body element.

Starting Point

All other background properties work together with background-image to control the image's appearance and behavior.

background-size Property

The background-size property allows you to set the size of the background image.

Common Values

  • contain - Scale image as large as possible without cropping or stretching
  • cover - Scale image to cover entire element while maintaining aspect ratio
  • Specific values - Use pixels or percentages

background-size: contain

CSS Code

body {
  background-image: url("image.jpg");
  background-size: contain;
  min-height: 100px;
}

contain scales the image as large as possible without cropping or stretching.

Visual Result

The entire image is visible, even if it doesn't fill the element.

Why Use min-height?

We set min-height: 100px so the background image is visible.

Purpose

The layout maintains a baseline height, ensuring that even with minimal content, the design appears balanced and visually appealing.

Without a minimum height, elements with no content might collapse and hide the background image entirely.

background-size: cover

CSS Code

body {
  background-image: url("image.jpg");
  background-size: cover;
  min-height: 100px;
}

cover scales the image to cover the entire element while maintaining its aspect ratio.

Visual Result

The element is completely filled, even if parts of the image are cropped.

Default Repeat Behavior

You probably noticed that background images continuously repeat by default.

Default Behavior

Background images repeat both horizontally and vertically to fill the entire element.

However, you can control this behavior using the background-repeat property.

background-repeat: no-repeat

CSS Code

body {
  background-image: url("image.jpg");
  background-size: contain;
  background-repeat: no-repeat;
  min-height: 100px;
}

Visual Result

The image appears only once and will no longer repeat on the screen.

background-repeat: repeat-x

CSS Code

body {
  background-image: url("image.jpg");
  background-size: contain;
  background-repeat: repeat-x;
  min-height: 100px;
}

Repeats the image horizontally only.

Visual Result

The image tiles across the width but not the height.

background-repeat: repeat-y

CSS Code

body {
  background-image: url("image.jpg");
  background-size: contain;
  background-repeat: repeat-y;
  min-height: 100px;
}

Repeats the image vertically only.

Visual Result

The image tiles down the height but not across the width.

background-position Property

The background-position property allows you to set where in the element the background image appears.

Value Options

  • Keywords: top, bottom, left, right, center
  • Pixel values: 20px 50px
  • Percentage values: 50% 100%

background-position Example

CSS Code

body {
  background-image: url("image.jpg");
  background-size: contain;
  background-repeat: no-repeat;
  background-position: center top;
  min-height: 100px;
}

Visual Result

This positions the image at the center horizontally and top vertically.

background-attachment Property

The background-attachment property determines whether the background image scrolls with the content or remains fixed when the page is scrolled.

scroll (default)

The background image scrolls with the content.

fixed

The background image stays in the same position on the screen.

background-attachment: fixed

body {
  background-image: url("image.jpg");
  background-position: center top;
  background-attachment: fixed;
}

Result

This sets the background image to remain fixed in place even when the page is scrolled.

Creates a parallax-like effect where content scrolls over the stationary background.

The background Shorthand

You can combine multiple properties into one line using the shorthand background property.

body {
  background: center top fixed url("image.jpg");
}

What This Combines

  • center top - background-position
  • fixed - background-attachment
  • url("image.jpg") - background-image

All in a single, concise declaration.

Mastering Background Properties

By mastering these properties, you'll have great control over how background images are displayed in your web designs.

Key Takeaways

  • background-size - Control image dimensions (contain, cover)
  • background-repeat - Control repetition (no-repeat, repeat-x, repeat-y)
  • background-position - Control placement (keywords, pixels, percentages)
  • background-attachment - Control scroll behavior (scroll, fixed)
  • background - Shorthand for combining properties

These properties allow for more visually appealing and functional layouts.
