Alt Text for Images

What Is Alternative Text?
Alternative text, often abbreviated as alt text, is a brief text description of an image. It provides essential information about the image for users who cannot see it, such as people who use screen readers and other assistive technologies.
Why It's Essential
It's essential for making websites accessible to people with visual disabilities.

Multiple Purposes of Alt Text

Beyond Accessibility
Alternative text serves several important purposes:
Accessibility: Provides image information for screen reader users
SEO: Used by search engines to understand images
Broken Image Fallback: Displayed when an image doesn't load properly due to missing files or connectivity issues

Example: Puppy on Beach

❌ Bad Alt Text
"A cute puppy."
This text is not specific enough to convey the important details of the image. What does the puppy look like? Where is the puppy? Are there any important objects around the puppy?
✓ Good Alt Text
"A black and white puppy with an orange collar lies on its belly in the sand, looking off to the side. A bright orange ball rests near its front paws."
HTML Example:
<img src="puppy.png" alt="A black and white puppy with an orange collar lies on its belly in the sand, looking off to the side. A bright orange ball rests near its front paws." />

Context Matters

No One Correct Way
It is important to note that there is no one correct way to write alt text for an image. What you include in your description will depend on the context the picture is used in.
Example: Context Changes Description
If the puppy picture was on a website about dog breeds, you might want to include more specifics about the physical description of the puppy and possibly ignore the fact that the puppy is at the beach playing with an orange ball.
Key Principle
The alt text you use for an image should reflect the primary purpose for including the image on the page, and the information you provide should give people who can't see the image the necessary details to understand that purpose.

Example: Tropical Resort

❌ Bad Alt Text
"Resort."
This is too short and it doesn't provide enough information about the image.
✓ Good Alt Text
"Tropical resort featuring a swimming pool surrounded by palm trees and bungalows."
HTML Example:
<img src="resort.png" alt="Tropical resort featuring a swimming pool surrounded by palm trees and bungalows." />

Best Practices for Writing Alt Text

Keep alt text short. It should be detailed enough to understand the image but not so long that it becomes confusing.
Do not try to describe every little detail. Focus on the most important aspects of the image.
Generally, you don't need to start with "image of" or "picture of." You can just start the description directly.
If there's similar text around the image, you don't need to write it again.

More Best Practices

It's usually recommended to end the alt text with a period for consistency.
Images as Links
If the image is a link to another page, instead of describing the image itself, the alt text should describe what will happen if users click on it.
❌ Describes Image
<a href="about.html"> <img src="arrow-right.png" alt="Right arrow." /> </a>
✓ Describes Action
<a href="about.html"> <img src="arrow-right.png" alt="Go to next page." /> </a>

Decorative Images

When to Use Empty Alt Text
Only images that convey important information should have alt text. If an image is only used for decorative purposes, it should have null (empty) alt text, so it can be ignored by screen readers and other assistive technologies.
Empty Alt Attribute Example:
<img src="decorative_image.jpg" alt="" />
Notice the alt attribute is present but empty (alt=""). This tells screen readers to skip the image.

Critical Requirements

Never Omit the Alt Attribute
Every image on your website should have an alt attribute, even if it's empty. If you omit the alt attribute completely, some screen readers will read the file name instead, which can be distracting for people who use assistive technologies, so this is not recommended.
Testing
Before your website is live, you should carefully test if screen readers can read the alt text correctly.
Creating Inclusive Experiences
Writing effective alt text is essential for creating accessible web content. As a web developer, by providing clear descriptions of your images, you can make sure that everyone can engage with your website in an inclusive online experience.
