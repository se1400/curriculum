Compound Assignment Operators

In JavaScript, all arithmetic operators have a compound assignment form.

What Are Compound Assignment Operators?

Compound assignment operators allow you to perform a mathematical operation and reassign the result back to the variable in one line of code.

Why Use Them?

  • Save time - Less typing required
  • Reduce clutter - Cleaner, more readable code
  • Combine steps - Operation + assignment in one line

Addition Assignment (+=)

+=

How It Works

The addition assignment operator takes the current value of the variable, adds the specified number to it, and then assigns the result back to the variable.

Example:

let total = 10;
total += 5;
console.log(total); // 15
15

What This Does

total += 5 is equivalent to total = total + 5

  1. Take current value: 10
  2. Add: 10 + 5 = 15
  3. Assign result back: total = 15

Subtraction Assignment (-=)

-=

How It Works

The subtraction assignment operator subtracts the specified value from the current value of the variable and assigns the new value back to the variable.

Example:

let score = 20;
score -= 7;
console.log(score); // 13
13

Without Compound Assignment

let score = 20;
score = score - 7;
console.log(score); // 13

Less efficient - requires writing score twice

What This Does

score -= 7 is equivalent to score = score - 7

Multiplication Assignment (*=)

*=

How It Works

The multiplication assignment operator multiplies the current value of the variable by the specified number and reassigns it back to the variable.

Example:

let points = 5;
points *= 3;
console.log(points); // 15
15

What This Does

points *= 3 is equivalent to points = points * 3

  1. Take current value: 5
  2. Multiply: 5 √ó 3 = 15
  3. Assign result back: points = 15

Real-World Example

let price = 50;
let quantity = 4;
price *= quantity;
console.log(price); // 200 (total cost)

Division Assignment (/=)

/=

How It Works

The division assignment operator lets you divide the current value of a variable by a number you specify, then assign the result back to the variable.

Example:

let balance = 100;
balance /= 4;
console.log(balance); // 25
25

What This Does

balance /= 4 is equivalent to balance = balance / 4

  1. Take current value: 100
  2. Divide: 100 √∑ 4 = 25
  3. Assign result back: balance = 25

Real-World Example

let totalBill = 120;
let people = 4;
totalBill /= people;
console.log(totalBill); // 30 (per person)

Remainder Assignment (%=)

%=

How It Works

The remainder assignment operator divides a variable by the specified number and assigns the remainder to the variable.

Example:

let number = 17;
number %= 5;
console.log(number); // 2
2

What This Does

number %= 5 is equivalent to number = number % 5

  1. Take current value: 17
  2. Divide and get remainder: 17 √∑ 5 = 3 remainder 2
  3. Assign remainder back: number = 2

Common Use Case

let hours = 27;
hours %= 24;  // Convert to 24-hour format
console.log(hours); // 3 (3 AM on next day)

Exponentiation Assignment (**=)

**=

How It Works

The exponentiation assignment operator raises a variable to the power of the specified number and reassigns the result to the variable.

Example:

let base = 2;
base **= 3;
console.log(base); // 8
8

What This Does

base **= 3 is equivalent to base = base ** 3

  1. Take current value: 2
  2. Raise to power: 2¬≥ = 2 √ó 2 √ó 2 = 8
  3. Assign result back: base = 8

Another Example

let number = 5;
number **= 2;
console.log(number); // 25 (5 squared)

Bitwise Operators (Advanced)

Apart from the arithmetic operators we've covered, there are also bitwise compound assignment operators:

Bitwise AND Assignment (&=)

Performs a bitwise AND operation with the specified number and reassigns the result to the variable.

let x = 5;      // Binary: 0101
x &= 3;         // Binary: 0011
console.log(x); // 1 (Binary: 0001)

Bitwise OR Assignment (|=)

Performs a bitwise OR operation with the specified number and reassigns the result to the variable.

let y = 5;      // Binary: 0101
y |= 3;         // Binary: 0011
console.log(y); // 7 (Binary: 0111)

üí° Note

Bitwise operators are advanced and used less frequently. Focus on mastering the arithmetic compound operators first!

Real-World Use Cases

Example 1: Shopping Cart Total

let total = 50;
let item1 = 20;
let item2 = 15;

total += item1;  // Add first item
total += item2;  // Add second item
console.log(total); // 85

Example 2: Game Score

let score = 100;
score += 50;   // Gain points
score -= 20;   // Lose points
score *= 2;    // Double score bonus
console.log(score); // 260

Example 3: Interest Calculator

let balance = 1000;
let interestRate = 0.05;  // 5%
balance *= (1 + interestRate);
console.log(balance); // 1050

Example 4: Split Bill Equally

let totalBill = 240;
let numberOfPeople = 6;
totalBill /= numberOfPeople;
console.log(totalBill); // 40 per person

Compound Assignment Summary

Key Takeaways

What Are They?

Compound assignment operators combine a mathematical operation with variable assignment in one line of code.

Why Use Them?

  • Shorter code: Less typing required
  • Cleaner code: Easier to read and understand
  • More efficient: One operation instead of two
