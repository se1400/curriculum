Math.min() and Math.max()

Math.min() Math.max()

What They Do

Math.min() and Math.max() both take a set of numbers and return the minimum and maximum value, respectively.

Example:

const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
1 9

Math.min()

Returns the smallest number from the set

Math.min(1, 5, 3, 9)  // 1

Math.min(7, 7, 7)     // 7

Math.max()

Returns the largest number from the set

Math.max(1, 5, 3, 9)  // 9
Math.max(10, -5, 0)   // 10
Math.max(7, 7, 7)     // 7

Math.ceil() - Round Up

Math.ceil()

What It Does

Math.ceil() rounds a number UP to the nearest whole integer.

"Ceil" is short for "ceiling" - think of rounding up to the ceiling!

Example:

console.log(Math.ceil(4.3)); // 5
5

‚úì What Happened?

Math.ceil() rounded 4.3 up to the nearest whole integer, which is 5.

More Examples

console.log(Math.ceil(4.1));   // 5
console.log(Math.ceil(4.9));   // 5
console.log(Math.ceil(4.001)); // 5
console.log(Math.ceil(4));     // 4 (already whole)
console.log(Math.ceil(-4.3));  // -4 (rounds toward zero)

Always rounds up to the next integer!

Math.floor() - Round Down

Math.floor()

What It Does

Math.floor() rounds a number DOWN to the nearest whole integer.

Think of rounding down to the floor!

Example:

console.log(Math.floor(4.7)); // 4
4

What Happened?

Math.floor() rounded 4.7 down to the nearest whole integer, which is 4.

More Examples

console.log(Math.floor(4.1));   // 4
console.log(Math.floor(4.9));   // 4
console.log(Math.floor(4.999)); // 4
console.log(Math.floor(4));     // 4 (already whole)
console.log(Math.floor(-4.3));  // -5 (rounds away from zero)

Always rounds down to the previous integer!

Math.round() - Round to Nearest

Math.round()

What It Does

Math.round() is the hybrid of Math.ceil() and Math.floor().

It rounds a number to its nearest integer, taking the decimal point into account.

Examples:

console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
2 5 5

The Rule

  • If decimal is less than .5 ‚Üí rounds down
  • If decimal is .5 or greater ‚Üí rounds up

More Examples

console.log(Math.round(4.4));   // 4 (< .5)
console.log(Math.round(4.5));   // 5 (= .5)
console.log(Math.round(4.6));   // 5 (> .5)
console.log(Math.round(-4.5));  // -4
console.log(Math.round(-4.6));  // -5

Random Number in a Range

Practical Application

A practical application of Math.floor() and Math.random() is to generate a random number between two whole numbers.

General Syntax:

const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min)) + min;
console.log(randomNum);

How It Works

  1. Math.random() generates 0 to 1
  2. Multiply by (max - min) to scale range
  3. Math.floor() removes decimals
  4. Add min to shift range up

Example: Random Number Between 1 and 20

const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);

Breaking It Down

  • Math.random() ‚Üí gives 0 to 0.999...
  • * 20 ‚Üí gives 0 to 19.999...
  • Math.floor() ‚Üí gives 0 to 19
  • + 1 ‚Üí gives 1 to 20

Math.trunc() - Remove Decimals

Math.trunc()

What It Does

Math.trunc() removes the decimal part of a number, returning only the integer portion, without rounding.

Examples:

console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
2 9

More Examples

console.log(Math.trunc(4.9));    // 4 (not rounded to 5)
console.log(Math.trunc(4.1));    // 4
console.log(Math.trunc(-4.9));   // -4 (not -5)
console.log(Math.trunc(-4.1));   // -4
console.log(Math.trunc(4));      // 4

Simply chops off the decimal, no rounding involved!

trunc vs floor

For positive numbers, they're the same. But for negative numbers:

Math.trunc(-4.9)  // -4 (removes decimal)
Math.floor(-4.9)  // -5 (rounds down)

Math.abs() - Absolute Value

Math.abs()

What It Does

Math.abs() returns the absolute value of a number, turning negatives into positives.

Examples:

console.log(Math.abs(-5)); // 5
console.log(Math.abs(5));  // 5
5 5

How It Works

  • Negative numbers become positive
  • Positive numbers stay positive
  • Zero stays zero

More Examples

console.log(Math.abs(-10));    // 10
console.log(Math.abs(10));     // 10
console.log(Math.abs(-3.5));   // 3.5
console.log(Math.abs(0));      // 0
console.log(Math.abs(-100));   // 100

Useful for finding distance or difference between numbers!

Real-World Use

const difference = Math.abs(score1 - score2);
// Always get positive difference

Math.pow() - Raise to Power
