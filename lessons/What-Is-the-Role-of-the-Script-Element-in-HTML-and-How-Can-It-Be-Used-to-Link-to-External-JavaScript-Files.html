The Script Element

The <script> element is used to embed executable code in HTML documents.

Most developers use this element to execute JavaScript code, which adds interactivity to web pages.

JavaScript enables dynamic features like interactive games, image sliders, and real-time form validation.

Basic Script Element
<body>
  <script>
    alert("Welcome to freeCodeCamp");
  </script>
</body>

JavaScript for Interactivity

JavaScript is the programming language that brings web pages to life with interactive features.

Common examples of JavaScript usage:

  • Interactive games
  • Image sliders and carousels
  • Dynamic forms that validate user input in real-time
  • Animations and visual effects
Inline JavaScript Example
<script>
  // Simple interactive code
  const button = document.querySelector('button');
  
  button.addEventListener('click', () => {
    alert('Button clicked!');
  });
</script>

Linking External JavaScript Files

While you can write JavaScript inside <script> tags, it's best practice to link to an external JavaScript file instead.

The src attribute specifies the location of the external JavaScript file. src stands for "source".

This approach keeps your code organized and maintainable.

External Script Syntax
<script src="path-to-javascript-file.js"></script>
Common File Paths
<!-- Same directory -->
<script src="script.js"></script>

<!-- Subdirectory -->
<script src="js/script.js"></script>

<!-- Parent directory -->
<script src="../script.js"></script>

Separation of Concerns

Separation of concerns is a design principle where you separate your programs into distinct sections, with each section addressing a separate concern.

In web development, this means keeping JavaScript code separate from HTML code.

This makes your code easier to maintain, debug, and reuse across different pages.

HTML File (index.html)
<!DOCTYPE html>
<html lang="en">
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Welcome</h1>
  <script src="script.js"></script>
</body>
</html>
JavaScript File (script.js)
// All JavaScript code here
const heading = document.querySelector('h1');

heading.addEventListener('click', () => {
  heading.textContent = 'Hello!';
});
