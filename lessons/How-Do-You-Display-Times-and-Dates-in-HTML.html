The <time> Element

The <time> element is used to represent a specific moment in time.

The datetime Attribute

The datetime attribute translates dates and times into a machine-readable format.

Why it matters:

  • Helps with search engine results
  • Helps browsers process date and time information more effectively
Basic Time Example
<p>
  The reservations are for 
  <time datetime="20:00">20:00</time>
</p>
Browser Output

The reservations are for 20:00

Note: 20:00 is twenty hundred hours, or 8 PM in the evening

Valid datetime Formats

The datetime attribute value must be one of the following:

  • Valid year
  • Valid month
  • Valid time
  • Local date
  • Global date
  • Valid duration string

ISO 8601 Format

The datetime value uses ISO 8601, an international standard to represent dates and times.

Date and Time Example
<p>
  The graduation will be on 
  <time datetime="2024-06-15T15:00">
    June 15
  </time>
</p>
Browser Output

The graduation will be on June 15

Understanding ISO 8601 Structure

The ISO 8601 format breaks down as follows:

2024-06-15T15:00

2024-06-15 = Year-Month-Day

T = Separator between date and time

15:00 = Hour:Minute (3 PM in the afternoon)

When to Use <time>

Use the <time> element whenever you need to represent:

  • Events
  • Publication dates
  • Appointments
Format Breakdown
/* ISO 8601 Date and Time Format */

YYYY-MM-DDTHH:MM:SS

/* Components */
YYYY = Year (4 digits)
MM   = Month (01-12)
DD   = Day (01-31)
T    = Date/Time separator
HH   = Hour (00-23)
MM   = Minute (00-59)
SS   = Second (00-59, optional)
Common Examples

2024-06-15 (Date only)

20:00 (Time only)

2024-06-15T15:00 (Date and time)
