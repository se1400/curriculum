What Is the Attribute Selector?

The attribute selector in CSS is a powerful tool that allows you to target HTML elements based on their attributes.

What You Can Do:

  • Target elements with specific attributes
  • Target elements with certain attribute values
  • Style elements dynamically
  • Go beyond class names for specificity
Attribute selectors enhance styling and improve accessibility by making interactive elements more distinguishable

Why Use Attribute Selectors?

Attribute selectors are particularly useful when class names alone don't provide enough specificity.

Common Use Cases:

  • Style all links with href attributes
  • Differentiate secure (HTTPS) vs insecure links
  • Style links with additional information (title attribute)
  • Target external vs internal links
  • Style file type links differently (.pdf, .doc, etc.)

Benefit:

Apply uniform styles without manually adding classes to every element

Basic Syntax: [attribute]

The simplest attribute selector targets elements that have a specific attribute, regardless of value.

Syntax:

element[attribute] {
  property: value;
}

Example: Target all links with href

a[href] {
  color: blue;
  text-decoration: underline;
}

What It Targets:

Any anchor element that has an href attribute, regardless of the href value

Targeting Specific Attributes: [title]

You can target elements with other attributes, like the title attribute.

Purpose of title Attribute:

The title attribute provides additional information about a link. You might want to style these links differently to indicate extra information is available.

CSS Example:

a[title] {
  font-weight: bold;
  text-decoration: none;
  color: #881518;
}

Visual Example: [title]

What's Happening:

  • Second link: No title ‚Üí default link styling

Combining Attributes: [attr1][attr2]

You can combine multiple attribute selectors for more precision.

CSS Example:

a[href][title] {
  color: green;
  font-weight: bold;
}

What It Targets:

Only links that have both an href attribute and a title attribute

This level of control is what makes attribute selectors so powerful in CSS

Visual Example: [href][title]

What's Happening:

  • First link: Has both ‚Üí green color, bold
  • Second link: Has only href ‚Üí no special styling
  • Third link: Has only title ‚Üí no special styling

Space-Separated Values: [attr~=value]

The [attr~=value] syntax matches a single value within a space-separated list.

HTML Example:

<a href="https://example.com"
   class="btn primary large">
  Visit Example Site
</a>

CSS Example:

a[class~="primary"] {
  background-color: #881518;
  color: white;
}

What It Targets:

Targets elements where the class attribute contains the word "primary" (even if there are other classes)

Visual Example: [class~="primary"]

Styling links where class contains "primary":

Link with "primary" class Link without "primary" class

What's Happening:

  • First link: Class list includes "primary" ‚Üí styled as button
  • Second link: Class list has "secondary" instead ‚Üí no special styling

Suffix Matching: [attr$=value]

The [attr$=value] syntax targets elements where the attribute value ends with a specific string.

CSS Example:

a[href$=".com"] {
  color: #881518;
  font-weight: bold;
}

a[href$=".com"]::after {
  content: ' (.com)';
}

Common Uses:

  • Style different file types (.pdf, .doc, .zip)
  • Differentiate domain extensions (.com, .org, .edu)
  • Add indicators for file downloads

Visual Example: [href$=".com"]

  • Second link: Ends with ".org" ‚Üí no special styling
  • Third link: Ends with ".edu" ‚Üí no special styling

Attribute Selector Syntax Summary

Syntax Description Example
[attr] Has the attribute a[href]
[attr=value] Exact value match a[target="_blank"]
[attr~=value] Contains value in space-separated list a[class~="primary"]
[attr^=value] Starts with value a[href^="https"]
[attr$=value] Ends with value a[href$=".pdf"]
[attr*=value] Contains value anywhere a[href*="example"]

Attribute Selectors Summary

Key Takeaways:

  • Attribute selectors target elements based on attributes
  • [attr] - Targets elements with that attribute
  • [attr=value] - Exact value match
  • [attr~=value] - Value in space-separated list
  • [attr^=value] - Starts with value (prefix)
  • [attr$=value] - Ends with value (suffix)
  • [attr*=value] - Contains value anywhere
  • Combine selectors for precision: [href][title]
  • Enhances accessibility and styling flexibility
Attribute selectors provide powerful, dynamic styling without requiring additional classes in your HTML
