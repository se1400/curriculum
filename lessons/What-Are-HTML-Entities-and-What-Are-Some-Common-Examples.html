What Are HTML Entities?

An HTML entity (or character reference) is a set of characters used to represent a reserved character in HTML.

When you want to display HTML code as text on the screen (like showing <img />), you need to use HTML entities.

Without entities, the HTML parser would interpret < and > as actual HTML tags rather than text to display.

Problem - Wrong Display
<p>This is an <img /> element</p>

/* Result: This is an  element */
/* The <img /> is parsed as HTML */
Solution - Correct Display
<p>This is an &lt;img /&gt; element</p>

/* Result: This is an <img /> element */
/* Now displayed as text */

Named Character References

Named character references start with an ampersand (&) and end with a semicolon (;).

By using a named reference, the HTML parser will not confuse it with an actual HTML element.

These are the most readable and commonly used type of HTML entities.

Common Named Entities
&lt;    →  <  (less than)
&gt;    →  >  (greater than)
&amp;   →  &  (ampersand)
&quot;  →  "  (quotation mark)
Example Usage
<p>&lt;p&gt;learning is fun&lt;/p&gt;</p>

/* Displays on screen: */
<p>learning is fun</p>

Decimal Numeric References

Decimal numeric references start with an ampersand (&) and hash symbol (#), followed by one or more decimal digits, and end with a semicolon (;).

Each character has a unique decimal code that represents it.

Decimal Examples
&#60;   →  <  (less than)
&#62;   →  >  (greater than)
&#169;  →  ©  (copyright)
&#174;  →  ®  (registered trademark)
Usage Example
<p>Copyright &#169; 2024</p>

/* Displays on screen: */
Copyright © 2024

Hexadecimal Numeric References

Hexadecimal numeric references start with an ampersand (&), hash symbol (#), and the letter x.

This is followed by one or more ASCII hexadecimal digits and ends with a semicolon (;).

These use base-16 numbering system (0-9 and A-F).

Hexadecimal Examples
&#x3C;    →  <  (less than)
&#x3E;    →  >  (greater than)
&#x20AC;  →  €  (euro symbol)
&#x03A9;  →  Ω  (Greek Omega)
Usage Example
<p>Price: &#x20AC;50</p>

/* Displays on screen: */
Price: €50

Three Types Summary

HTML entities come in three types, all representing the same characters in different formats:

  • Named - Most readable and commonly used
  • Decimal Numeric - Uses decimal numbers
  • Hexadecimal Numeric - Uses hexadecimal numbers

All three formats work the same way - choose the one that's most convenient for your situation.

Same Character, Three Ways
/* All display the < symbol */

&lt;     ← Named
&#60;    ← Decimal
