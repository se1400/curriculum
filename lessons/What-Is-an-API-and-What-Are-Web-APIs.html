What Is an API?

API = Application Programming Interface

APIs establish a set of rules and protocols that allow software applications to communicate with each other and exchange data efficiently.

Think of APIs as Building Blocks

You can think of APIs as constructs that allow developers to create more complex functionality based on simpler building blocks that are already implemented.

API Communication Flow

Application A
Requests data
↔️
API
Rules & Protocols
↔️
Application B
Sends response
Key Point:

APIs are essential for web development because they enable different software systems to work together, share data, and provide enhanced functionality without having to build everything from scratch.

What Are Web APIs?

APIs for Web Applications

Web APIs are specifically designed for web applications. They are used in client-side JavaScript development but are not part of JavaScript itself.

Two Main Categories

Web APIs are often divided into two main categories:

Browser APIs

Built into the browser

• Come pre-installed
• Ready to use immediately
• No external code needed

Third-Party APIs

External services

• Not built into browser
• Need to retrieve their code
• Come with documentation

Important Note:

Web APIs are not part of JavaScript itself - they are additional tools provided by browsers or external services that you can access using JavaScript.

Browser APIs

Built-In Browser Functionality

Browser APIs expose data from the browser and provide access to various functionalities. As a web developer, you can access and manipulate this data using JavaScript.

What Browser APIs Provide

  • Manipulate website structure (HTML, CSS)
  • Handle events (clicks, keyboard, mouse)
  • Work with storage (save data locally)
  • Communicate with network (fetch data)
  • Access device features (location, camera)
  • Control media (audio, video)

Common Browser APIs

DOM API

Manipulate HTML elements, styles, and attributes

Storage API

Store data locally on the user's device

Fetch API

Make network requests to retrieve data

Geolocation API

Get the user's current location

Browser API Example: DOM API

The DOM API

The DOM (Document Object Model) API is one of the most commonly used Browser APIs. You can use it to manipulate HTML elements, their styles, and attributes.

DOM API in Action

// Get an element by its ID
const heading = document.getElementById('main-heading');

// Change its text content
heading.textContent = 'Welcome to My Website!';

// Change its style
heading.style.color = 'blue';
heading.style.fontSize = '36px';

// Add a CSS class
heading.classList.add('highlight');

What This Does

Step 1: Find the heading element on the page
Step 2: Change the text inside it
Step 3: Modify its appearance (color, size)
Step 4: Add a CSS class for styling
Core Concept:

The DOM API is a core concept in web development. You will learn much more about it in coming lessons. It allows you to dynamically update and change your website's content and appearance!

Third-Party APIs

External API Services

Third-party APIs are not built into the browser by default. To use them, you need to retrieve their code in some way. They typically come with detailed documentation explaining how to use their services.

Types of Third-Party APIs

Maps APIs
Google Maps, Mapbox
Weather APIs
OpenWeatherMap, WeatherAPI
Social Media APIs
Twitter, Facebook, Instagram
Payment APIs
Stripe, PayPal, Square
Data APIs
Stock prices, Sports scores
Translation APIs
Google Translate, DeepL

How to Use Third-Party APIs

  1. Find the API that fits your needs
  2. Read the documentation to understand how it works
  3. Get an API key (if required)
  4. Include the API code in your project
  5. Make API calls to use the service

Third-Party API Example

Google Maps API

An example of a third-party API is the Google Maps API, which you can use to display interactive maps on your website.

Using a Third-Party API

// 1. Include the API script in your HTML
// <script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY"></script>

// 2. Use the API in your JavaScript
function initMap() {
    // Create a new map centered on a location
    const map = new google.maps.Map(document.getElementById('map'), {
        center: { lat: 40.7128, lng: -74.0060 }, // New York City
        zoom: 12
    });
    
    // Add a marker
    const marker = new google.maps.Marker({
        position: { lat: 40.7128, lng: -74.0060 },
        map: map,
        title: 'New York City'
    });
}

Typical Weather API Request

// Fetch weather data from a weather API
fetch('https://api.weatherapi.com/v1/current.json?key=YOUR_KEY&q=London')
    .then(response => response.json())
    .then(data => {
        console.log(`Temperature: ${data.current.temp_c}°C`);
        console.log(`Condition: ${data.current.condition.text}`);
    });
Remember:

Third-party APIs typically require an API key for authentication. Always read the API's documentation to understand how to properly use their service!
