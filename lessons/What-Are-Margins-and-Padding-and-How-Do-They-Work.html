Margins and Padding

Margin and padding are essential CSS properties for creating well-structured, readable, and visually appealing web pages.

Key Differences

  • Margin: Controls space outside an element, separating it from other elements
  • Padding: Controls space inside an element, between content and border

The Box Model

Understanding how margin and padding work together in the CSS box model:

Margin Property

Margin creates space outside an element, helping to separate it from other elements and define layout structure.

Example: margin-top

Applying spacing to the top of paragraph elements using margin-top.

CSS
p {
  margin-top: 20px;
  border: 2px solid black;
}

Margin Shorthand

You can specify one, two, three, or four values at once using the margin shorthand property.

Values Syntax Result
1 Value margin: 10px; All sides: 10px
2 Values margin: 10px 20px; Top/Bottom: 10px, Left/Right: 20px
3 Values margin: 10px 20px 30px; Top: 10px, Left/Right: 20px, Bottom: 30px
4 Values margin: 10px 20px 30px 40px; Top: 10px, Right: 20px, Bottom: 30px, Left: 40px

One Value Shorthand

When using a singular value, that exact value will be applied to all four sides.

CSS
p {
  margin: 10px;
  border: 2px solid black;
}

Two Values Shorthand

First value applies to top and bottom, second value applies to left and right.

CSS
p {
  margin: 10px 20px;
  border: 2px solid black;
}
Element
10px (top)
20px (right)
10px (bottom)
20px (left)

Vertical: 10px | Horizontal: 20px

Three Values Shorthand

First value: top, second value: left and right, third value: bottom.

CSS
p {
  margin: 10px 20px 30px;
  border: 2px solid black;
}
Element
10px (top)
20px (right)
30px (bottom)
20px (left)

Top: 10px | Sides: 20px | Bottom: 30px

Four Values Shorthand

Clockwise from top: top, right, bottom, left (think "TRouBLe").

CSS
p {
  margin: 10px 20px 30px 40px;
  border: 2px solid black;
}
Element
10px (top)
20px (right)
30px (bottom)
40px (left)

Independent control of each side

Padding Property

The padding property applies space inside the element, between the content and its border.

Four Padding Properties

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
CSS
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}

Padding Shorthand

Like the margin property, you can use shorthand for padding with one, two, three, or four values.

Syntax Result
padding: 10px; All sides: 10px
padding: 10px 20px; Top/Bottom: 10px, Left/Right: 20px
padding: 10px 20px 30px; Top: 10px, Left/Right: 20px, Bottom: 30px
padding: 10px 20px 30px 40px; Top: 10px, Right: 20px, Bottom: 30px, Left: 40px

Key Takeaways

  • Margin creates space outside the element (between elements)
  • Padding creates space inside the element (between content and border)
  • Both have four directional properties: top, right, bottom, left
  • Both support shorthand notation with 1, 2, 3, or 4 values
  • Four-value order: top, right, bottom, left (clockwise from top)
