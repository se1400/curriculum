Adding and Removing DOM Nodes

The Need for Dynamic DOM Manipulation

There will be times where you need to add or remove nodes from the DOM.

There are a couple of Web APIs you can use to accomplish this.

appendChild()

Adds a node to the end of the list of children of a specified parent node.

parentNode.appendChild(newNode)

removeChild()

Removes a node from the DOM by removing a child from a specified parent node.

parentNode.removeChild(childNode)

What We'll Cover

In this lesson, we will cover the appendChild() and removeChild() methods.

The appendChild() Method

Purpose

The appendChild() method is used to add a node to the end of the list of children of a specified parent node.

parentNode.appendChild(newNode);

Parameters

parentNode: The element that will receive the new child

newNode: The element to be added as a child

Key Points

‚úì Returns the appended child

‚úì Commonly used with createElement()

appendChild() Example - Setup

Example: Adding Desserts to a List

Let's see how to use appendChild() with a real example.

Starting HTML

<ul id="desserts"> <li>Cake</li> <li>Pie</li> </ul>

We have an unordered list with an ID of desserts containing two list items: Cake and Pie.

Goal

We want to add a third dessert, Cookies, to the end of this list using JavaScript.

appendChild() Step-by-Step

1 Access the Parent Element

const dessertsList = document.getElementById("desserts");

Store the reference to the <ul> element in a variable called dessertsList.

2 Create a New List Item

const dessertsList = document.getElementById("desserts"); const listItem = document.createElement("li");

Create a new <li> element using createElement() and store it in listItem.

3 Append the New Node

const dessertsList = document.getElementById("desserts"); const listItem = document.createElement("li"); dessertsList.appendChild(listItem);

Use appendChild() to add the new list item to the end of the list.

Adding Text Content

The Problem

If you run the code from the previous slide, you'll see that a new list item element is added to the end of the list.

But the new list item is empty!

Before Adding Text

<ul id="desserts"> <li>Cake</li> <li>Pie</li> <li></li> <!-- Empty! --> </ul>

After Adding Text

<ul id="desserts"> <li>Cake</li> <li>Pie</li> <li>Cookies</li> </ul>

Solution: Use textContent

To add text content to the new list item, use the textContent property.

const dessertsList = document.getElementById("desserts"); const listItem = document.createElement("li"); listItem.textContent = "Cookies"; dessertsList.appendChild(listItem);

Now the list will show Cake, Pie, and Cookies!

The removeChild() Method

Purpose

To remove a node from the DOM, you can use the removeChild() method.

parentNode.removeChild(childNode);

Parameters

parentNode: The element containing the child to remove

childNode: The element to be removed

‚úì Removes the child from the DOM

‚úì Child must be a direct descendant

removeChild() Example - Setup

Example: Removing Last Paragraph

Let's remove the last paragraph element from a section element.

Starting HTML

<section id="example-section"> <h2>Example sub heading</h2> <p>first paragraph</p> <p>second paragraph</p> </section>

Goal

Remove the second paragraph (last <p> element) from the section.

removeChild() Step-by-Step

1 Access the Parent Element

const sectionEl = document.getElementById("example-section");

Access the section element using getElementById().

2 Select the Child to Remove

const sectionEl = document.getElementById("example-section"); const lastParagraph = document.querySelector("#example-section p:last-of-type");

Use querySelector() with the :last-of-type pseudo-class to select the last paragraph.

3 Remove the Child

const sectionEl = document.getElementById("example-section"); const lastParagraph = document.querySelector("#example-section p:last-of-type"); sectionEl.removeChild(lastParagraph);

Now we have both the parent and child nodes, so we can use removeChild().

removeChild() Result

Before removeChild()

<section id="example-section"> <h2>Example sub heading</h2> <p>first paragraph</p> <p>second paragraph</p> </section>

After removeChild()

<section id="example-section"> <h2>Example sub heading</h2> <p>first paragraph</p> </section>

The :last-of-type Pseudo-class

We used :last-of-type to select the last paragraph element inside the section.

This CSS pseudo-class is very useful for selecting the last element of a specific type within a parent.

When to Use These Methods

Dynamic Content Scenarios

When might you want to add or remove nodes from the DOM?

If you're working with dynamic content, you might need to add or remove nodes from the DOM.

Real-World Examples

üõí Shopping Carts

appendChild(): Add items when user clicks "Add to Cart"

removeChild(): Remove items when user clicks "Remove"

// Add product to cart cartList.appendChild(newProductElement); // Remove product from cart cartList.removeChild(productToRemove);

‚úì To-Do Lists

appendChild(): Add new tasks when user submits form

removeChild(): Remove tasks when marked as complete

// Add new task taskList.appendChild(newTaskElement); // Remove completed task taskList.removeChild(completedTask);

üì± Social Media Feeds

appendChild(): Add new posts as they load

removeChild(): Remove posts when user hides them

// Add new post to feed feed.appendChild(newPostElement); // Remove hidden post feed.removeChild(hiddenPost);

Key Takeaways

appendChild()

‚úì Adds node to end of children

‚úì Syntax: parent.appendChild(child)

‚úì Use with createElement()

‚úì Set textContent for text

‚úì Great for building lists dynamically

removeChild()

‚úì Removes child from parent

‚úì Syntax: parent.removeChild(child)

‚úì Needs both references

‚úì Use querySelector() to select

‚úì Perfect for removing list items

üí° Remember

Both methods are essential for creating dynamic, interactive web applications.

Common use cases include:

  • Shopping carts
  • To-do lists
  • Social media feeds
  • Comment sections
  • Dynamic forms

Best Practice

Always check if an element exists before trying to remove it to avoid errors:

if (childElement) { parentElement.removeChild(childElement); }
