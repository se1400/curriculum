CRUD Operations

What is CRUD?

CRUD is an acronym that stands for Create, Read, Update, and Delete.

These are the four operations of persistent storage.

C - Create

Add new data

R - Read

Retrieve existing data

U - Update

Modify existing data

D - Delete

Remove data

Persistent Storage

Persistent storage refers to saving data in a way that makes it available even after the power is turned off or the device is restarted.

Create Operation

Adding New Data

Create refers to the process of creating new data.

For example, in a web app, this could be when a user adds a new post to a blog.

Real-World Examples

‚Ä¢ Adding a new blog post

‚Ä¢ Registering a new user account

‚Ä¢ Creating a new product listing

‚Ä¢ Posting a new comment

‚Ä¢ Uploading a new photo

HTTP Method

POST is used to create a new resource.

JavaScript Example

let items = []; // Create operation function createItem(item) { items.push(item); } // Example usage createItem('Book'); console.log(items); // ['Book']

‚úì Key Point

Create operations add new data to your storage. The data didn't exist before!

Read Operation

Retrieving Data

Read is the operation where data is retrieved from a database.

For instance, when you visit a blog post or view your profile on a website, you're performing a read operation to fetch and display data stored in the database.

Real-World Examples

‚Ä¢ Viewing a blog post

‚Ä¢ Loading your profile page

‚Ä¢ Displaying a list of products

‚Ä¢ Reading comments on a post

‚Ä¢ Viewing search results

HTTP Method

GET is used to retrieve or read data.

JavaScript Example

let items = ['Book', 'Pen', 'Notebook']; // Read operation function readItems() { return items; } // Example usage const allItems = readItems(); console.log(allItems); // ['Book', 'Pen', 'Notebook']

‚úì Key Point

Read operations don't change the data - they just retrieve and display it!

Update Operation

Modifying Existing Data

Update involves modifying existing data in the database.

An example would be editing a blog post or updating your profile information.

Real-World Examples

‚Ä¢ Editing a blog post

‚Ä¢ Updating profile information

‚Ä¢ Changing a product price

‚Ä¢ Modifying a comment

‚Ä¢ Updating account settings

HTTP Methods

PUT is used to update a resource by replacing it entirely.

PATCH is used to partially update a resource.

JavaScript Example

let items = ['Book', 'Pen', 'Notebook']; // Update operation function updateItem(index, newItem) { items[index] = newItem; } // Example usage updateItem(0, 'Magazine'); console.log(items); // ['Magazine', 'Pen', 'Notebook']

PUT vs PATCH

PUT: Replace the entire resource

PATCH: Update only specific fields

Delete Operation

Removing Data

Delete is the operation that removes data from a database.

For instance, when you delete a blog post or account, you're performing a delete operation.

Real-World Examples

‚Ä¢ Deleting a blog post

‚Ä¢ Removing a user account

‚Ä¢ Deleting a product listing

‚Ä¢ Removing a comment

‚Ä¢ Deleting uploaded files

HTTP Method

DELETE is used to remove a resource.

JavaScript Example

let items = ['Book', 'Pen', 'Notebook']; // Delete operation function deleteItem(index) { items.splice(index, 1); } // Example usage deleteItem(0); console.log(items); // ['Pen', 'Notebook']

‚ö†Ô∏è Important

Delete operations are permanent! Always confirm before deleting important data.

CRUD and HTTP Methods

RESTful APIs

CRUD is used when working with databases, the UI, and RESTful APIs.

RESTful APIs are a set of conventions for building web services that allow the client to interact with a database or backend system by performing CRUD operations through standard HTTP methods.

HTTP - Hypertext Transfer Protocol

HTTP stands for Hypertext Transfer Protocol and it is the foundation for data communication on the web.

There are HTTP methods which define the actions that can be performed on resources over the web.

CRUD to HTTP Mapping

Read

Retrieve or read data

Update

PUT PATCH

PATCH: Partial update

Delete

Complete CRUD Example

All Operations Together

Here's an example of how CRUD operations might be represented in code using a simple array in JavaScript:

let items = []; // Create function createItem(item) { items.push(item); } // Read function readItems() { return items; } // Update function updateItem(index, newItem) { items[index] = newItem; } // Delete function deleteItem(index) { items.splice(index, 1); }

Example Usage

createItem('Book'); console.log(readItems()); // ['Book'] updateItem(0, 'Magazine'); console.log(readItems()); // ['Magazine'] deleteItem(0); console.log(readItems()); // []

Breaking Down the Example

In this example:

Create

We create an item by pushing it into an array.

items.push(item);

Read

We read the items by returning the array.

return items;

Update

We update an item by modifying the array element at a given index.

items[index] = newItem;

Delete

We delete an item by removing it from the array using splice().

items.splice(index, 1);

Conceptual Level

This is a basic representation of how CRUD operations work at a conceptual level.

In real applications, these operations would interact with databases, but the concepts remain the same!
