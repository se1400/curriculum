Arithmetic Operators in JavaScript

JavaScript provides tools to perform basic arithmetic operations on numbers, such as addition, subtraction, multiplication, and division.

What Are Arithmetic Operators?

All these tools are called arithmetic operators.

What JavaScript Includes

  • Basic operations: Addition, subtraction, multiplication, division
  • Complex operations: Remainder and exponentiation
  • Combinations: Mix multiple operators together

Let's look at these operators in detail, how to use them, and how to combine them!

Addition Operator (+)

+

The addition operator is a plus sign (+).

What It Does

The addition operator allows you to find the total of two or more numbers.

Important: In addition operations, the order of the numbers doesn't matter.

const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
15 15 30
10 + 5 = 15
5 + 10 = 15
5 + 10 + 15 = 30

Subtraction Operator (-)

-

The subtraction operator is a minus sign (-).

What It Does

It allows you to find the difference between two numbers.

Use the minus sign when you want to subtract a number from another number, usually a smaller one from a bigger one.

Basic Subtraction

const difference = 10 - 5;
console.log(difference); // 5
5

‚ö†Ô∏è Order Matters!

If a smaller number comes first, you'll get a negative result:

const difference = 5 - 10;
console.log(difference); // -5
-5

Subtraction with Variables

You can also assign the numbers to variables and do the subtraction with the variable names:

const num1 = 10;
const num2 = 5;
const result = num1 - num2;
console.log(result); // 5
5
10 - 5 = 5

Real-World Example

const totalCost = 100;
const discount = 20;
const finalPrice = totalCost - discount;
console.log(finalPrice); // 80

The final price after discount is $80

Remainder Operator (%)

%

The remainder operator, represented by a percentage sign (%), returns the remainder of a division.

What Is a Remainder?

The remainder in math is the leftover value after performing division.

const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;
console.log(remainder); // 1
1

Why Remainder is 1

10 √∑ 3 = 3 remainder 1

3 √ó 3 = 9
10 - 9 = 1 (leftover)

When you divide 10 by 3, you get 3 with a remainder of 1.

Remainder Operator Examples

More Examples:

console.log(17 % 5);  // 2  (17 = 5√ó3 + 2)
console.log(20 % 4);  // 0  (20 = 4√ó5 + 0, divides evenly)
console.log(7 % 2);   // 1  (7 = 2√ó3 + 1, odd number)
console.log(8 % 2);   // 0  (8 = 2√ó4 + 0, even number)
2 0 1 0

Common Use Cases

  • Check if even/odd: num % 2 === 0 (even) or num % 2 === 1 (odd)
  • Cycle through array: Use remainder to loop back to start
  • Check divisibility: If a % b === 0, then a is divisible by b

Practical Example: Even or Odd

const number = 7;
if (number % 2 === 0) {
  console.log("Even");
} else {
  console.log("Odd");
}
// Output: Odd

Mixing Operators

It's possible to mix operators in a single operation or expression:

const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
18

How JavaScript Calculates This

  1. 5 * 2 = 10
  2. 8 / 4 = 2
  3. 10 + 10 = 20
  4. 20 - 2 = 18

‚ö†Ô∏è Operator Precedence

When you mix different operators in a single expression, the JavaScript engine follows a system called operator precedence to determine the order of operations.

Operator precedence determines the order in which operations are executed in expressions.

You will learn more about operator precedence in future lessons.

Operator Precedence (Quick Overview)

JavaScript follows a specific order when evaluating expressions with multiple operators:

Priority Operator Description
1 (Highest) ** Exponentiation
2 * / % Multiplication, Division, Remainder
3 (Lowest) + - Addition, Subtraction

Example: Understanding Order

const result = 2 + 3 * 4;  // Multiplication first!
// Step 1: 3 * 4 = 12
// Step 2: 2 + 12 = 14
console.log(result); // 14
14

‚úì Use Parentheses for Clarity

const result = (2 + 3) * 4;  // Parentheses first!
// Step 1: 2 + 3 = 5
// Step 2: 5 * 4 = 20
console.log(result); // 20
