JSON.parse() and JSON.stringify()

Two Powerful JSON Methods

JavaScript provides two powerful methods for handling JSON data:

JSON.stringify()

Purpose:

Converts a JavaScript object into a JSON string

Object → String

JSON.parse()

Purpose:

Converts a JSON string into a JavaScript object

String → Object
Why These Methods Matter:

These methods are commonly used to convert between JSON strings and JavaScript objects, which is essential when:

  • Storing data in localStorage
  • Sending/receiving data from web servers
  • Transmitting data between systems

JSON.stringify() - Object to String

Converting Objects to JSON Strings

JSON.stringify() converts a JavaScript object into a JSON string. This is useful when you want to store or transmit data. 

const user = {
    name: "John",
    age: 30,
    isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
{"name":"John","age":30,"isAdmin":true}

Conversion Process

Input: JavaScript Object
{
  name: "John",
  age: 30,
  isAdmin: true
}
↓ JSON.stringify()
Output: JSON String
'{"name":"John","age":30,"isAdmin":true}'
Notice:

The entire object is now wrapped in quotes as a single string! The property names also get double quotes, and the spacing is removed.

JSON.stringify() - Replacer Parameter

Filtering Properties

The JSON.stringify() method accepts an optional replacer parameter, which can be an array that specifies which properties to include. 

const developerObj = {
    firstName: "Jessica",
    isAwesome: true,
    isMusician: true,
    country: "USA"
};

// Only include firstName and country
const result = JSON.stringify(developerObj, ["firstName", "country"]);
console.log(result);
{"firstName":"Jessica","country":"USA"}

How the Replacer Works

Original Object (4 properties)
firstName: "Jessica"
isAwesome: true
isMusician: true
country: "USA"
↓ stringify(obj, ["firstName", "country"])
Result (2 properties only)
'{"firstName":"Jessica","country":"USA"}'
Syntax:

JSON.stringify(object, [array of properties])

JSON.stringify() - Spacer Parameter

Formatting Output

Another optional parameter is the spacer, which controls the spacing for the stringified result, making it more readable. 

const developerObj = {
    firstName: "Jessica",
    isAwesome: true,
    isMusician: true,
    country: "USA"
};

// Add 2-space indentation
console.log(JSON.stringify(developerObj, null, 2));
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}

Without Spacer

JSON.stringify(obj)
{"firstName":"Jessica","isAwesome":true,"isMusician":true,"country":"USA"}

Hard to read - everything on one line

With Spacer: 2

JSON.stringify(obj, null, 2)
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}

Easy to read - nicely formatted!

Note:

Most of the time you won't use these optional parameters, but it's helpful to be aware of them for special cases like debugging or creating human-readable output.

JSON.parse() - String to Object

Converting JSON Strings to Objects

JSON.parse() converts a JSON string back into a JavaScript object. This is useful when you retrieve JSON data from a web server or from localStorage. 

const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);

console.log(userObject);
// Result: { name: 'John', age: 30, isAdmin: true }
{ name: 'John', age: 30, isAdmin: true }

Conversion Process

Input: JSON String
'{"name":"John","age":30,"isAdmin":true}'

This is a single string wrapped in quotes

↓ JSON.parse()
Output: JavaScript Object
{
  name: 'John',
  age: 30,
  isAdmin: true
}

Now a usable JavaScript object!

Why This Is Useful:

This allows you to work with the data in your program as a normal JavaScript object, making it easier to manipulate and use with dot notation or bracket notation.

When to Use Each Method

Common Use Cases

JSON.stringify()

When to use:
  • Storing data in localStorage
  • Sending data to a web server
  • Transmitting data between systems
  • Logging objects for debugging
// Storing in localStorage
const data = {user: "Alice"};
localStorage.setItem(
  'userData', 
  JSON.stringify(data)
);

JSON.parse()

When to use:
  • Retrieving data from localStorage
  • Receiving data from a web server
  • Processing JSON from APIs
  • Converting strings to usable objects
// Retrieving from localStorage
const stored = localStorage.getItem(
  'userData'
);
const data = JSON.parse(stored);
Remember:

• Use JSON.stringify() when you need to convert an object to a string
• Use JSON.parse() when you need to convert a string to an object
• They are opposite operations!

Summary & Quick Reference

The Two Methods

Method Converts Use For
JSON.stringify() Object → String Storing, sending data
JSON.parse() String → Object Retrieving, receiving data

Complete Example: Round Trip

// Step 1: Create object
const user = { name: "Alice", age: 25 };

// Step 2: Convert to string (for storage/transmission)
const jsonString = JSON.stringify(user);
// Result: '{"name":"Alice","age":25}'

// Step 3: Convert back to object (for use in program)
const parsed = JSON.parse(jsonString);
// Result: { name: "Alice", age: 25 }

// Step 4: Access properties
console.log(parsed.name); // "Alice"
console.log(parsed.age);  // 25

Optional Parameters (stringify only)

Replacer (2nd parameter):
JSON.stringify(obj, ["prop1", "prop2"])

Filter which properties to include

Spacer (3rd parameter):
JSON.stringify(obj, null, 2)

Add indentation for readability
