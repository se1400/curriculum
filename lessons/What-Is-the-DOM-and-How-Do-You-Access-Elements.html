What is the DOM?

Definition

DOM stands for Document Object Model

It's a programming interface that lets us interact with HTML documents.

What Can You Do with the DOM?

  • Add Elements: Insert new content into your webpage
  • Modify Elements: Change existing content, styles, and attributes
  • Delete Elements: Remove content from your webpage
  • Make Pages Interactive: Make elements listen to and respond to events

Why the DOM Matters

The DOM is the bridge between your HTML document and JavaScript. It's what allows you to create dynamic, interactive websites that respond to user actions.

DOM Tree Structure

In the DOM, an HTML document is represented as a tree of nodes. Each node represents an HTML element from the HTML document.

Example HTML Document

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>

Understanding the Hierarchy

  • Each element is a node in the tree
  • Elements can have parent and child nodes
  • The structure can be much more complex in real webpages

DOM Hierarchy

Root Node: Document

The HTML document is the root node in the DOM hierarchy.

It has one child node: the html element.

Root Element: HTML

The html element is the root element of the HTML document.

All other nodes descend from it.

Head Element

Contains metadata about the document

Provides essential information about the webpage

Examples: title, links to CSS, meta tags

Body Element

Contains the visible content of the webpage

Everything users see on the page

Examples: headings, paragraphs, images, buttons

Accessing Elements with JavaScript

The Power of the DOM

You can access DOM elements with JavaScript, work with them in your code, and even modify them dynamically.

Two Primary Methods

getElementById()

Get an element by its unique ID

querySelector()

Get an element using CSS selectors

Web APIs in Action

These methods are web APIs because they provide standardized ways to interact with the DOM using JavaScript.

getElementById() Method

How it Works

Returns an object that represents the HTML element with the specified id

Important: IDs must be unique in every HTML document, so this method returns only one Element object.

Syntax

const container = document.getElementById("container");

Complete Example

// HTML
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>

// JavaScript
const container = document.getElementById("container");
console.log(container); // Logs the div element

Key Point: Pass the ID within quotation marks as an argument.

querySelector() Method

More Flexible Than getElementById()

querySelector() gets the first element in the HTML document that matches the CSS selector passed as an argument.

Example: Selecting by Tag

// HTML
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>

// JavaScript
const section = document.querySelector("section");

This will select the first <section> element in the document.

querySelector() with CSS Selectors

Using Class Selectors

To select an element by its class name, prefix the class name with a dot (.)

Example: Selecting by Class

// HTML
<div class="highlight">
  <p>This content is highlighted.</p>
</div>

// JavaScript
const highlightedDiv = document.querySelector(".highlight");

CSS Selector Flexibility

You can use any valid CSS selector with querySelector():

  • .class-name for classes
  • #id-name for IDs
  • element for tag names
  • Complex selectors like div.class > p

More DOM Selection Methods

Methods for Multiple Elements

There are other methods available for selecting multiple elements at once.

getElementsByClassName()

Returns a collection of all elements with the specified class name

querySelectorAll()

Returns all elements that match a CSS selector
