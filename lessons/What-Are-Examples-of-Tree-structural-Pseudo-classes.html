Tree-structural Pseudo-classes

Tree-structural pseudo-classes allow you to target and style elements based on their position within the document tree.

What is the Document Tree?

The hierarchical structure of elements in an HTML document—how elements are nested and related to each other.

These pseudo-classes help you select elements based on their structural position, not their content or attributes

:root Pseudo-class

The :root pseudo-class represents the root element of the document, usually the <html> element.

Purpose:

  • Target the highest level in the document
  • Apply styles to the entire document
  • Commonly used for setting CSS variables
  • Has higher specificity than html selector

CSS Example:

:root {
  font-family: Arial, sans-serif;
  background-color: #f5f5f5;
}

:root with CSS Variables

The :root pseudo-class is commonly used for defining CSS variables (custom properties).

CSS Example:

:root {
  --main-font: 'Arial, sans-serif';
  --primary-color: blue;
  --secondary-color: green;
}

Benefits of CSS Variables:

  • Store values in one place
  • Reuse them throughout your stylesheet
  • Easy to maintain and update
  • Can be changed dynamically with JavaScript

:empty Pseudo-class

The :empty pseudo-class targets elements with no children (including text nodes), except whitespace.

What Counts as Empty?

  • No child elements
  • No text content
  • Whitespace only is considered empty

CSS Example:

li:empty {
  background-color: lightgray;
  min-height: 40px;
}

:nth-child(n) & :nth-last-child(n)

:nth-child(n) selects elements based on their position within a parent.

:nth-last-child(n) counts from the end.

Values for n:

  • Specific number: :nth-child(2)
  • odd: Selects odd-numbered children
  • even: Selects even-numbered children
  • Formula: :nth-child(3n) - every 3rd element

Common Use: Table Striping

tr:nth-child(even) { background: #f9f9f9; }
tr:nth-child(odd) { background: #fff; }

:first-child, :last-child, :only-child

These pseudo-classes target elements based on their position as children within a parent.

:first-child

Selects the first element in a parent

li:first-child {
  background: lightcoral;
}

:last-child

Selects the last element in a parent

li:last-child {
  background: lightblue;
}

:only-child Pseudo-class

:only-child selects an element if it's the only child of its parent.

When It Applies:

  • Element has no siblings
  • Parent has only one child element
  • Useful for special styling of standalone items

CSS Example:

.container div:only-child {
  background-color: lightgreen;
}

:first-of-type & :last-of-type

These select the first and last occurrence of a specific element type within its parent.

Key Difference from :first-child:

:first-of-type looks for the first element of that specific type, not just the first child overall.

CSS Example:

p:first-of-type { background: lightcoral; }
p:last-of-type { background: lightblue; }

:nth-of-type(n) Pseudo-class

:nth-of-type(n) selects a specific element based on its position among siblings of the same type.

Examples:

  • :nth-of-type(2) - Second element of that type
  • :nth-of-type(odd) - Odd-numbered elements
  • :nth-of-type(3n) - Every third element

CSS Example:

p:nth-of-type(2) {
  background-color: lightyellow;
}

:only-of-type Pseudo-class

:only-of-type selects an element if it's the only one of its type within its parent.

Key Points:

  • Element is the only one of that type
  • Parent may have other types of elements
  • Useful for emphasizing single items
  • Different from :only-child (which requires no siblings at all)

CSS Example:

p:only-of-type {
  background-color: lightgreen;
}

Tree-structural Pseudo-classes Summary

Key Takeaways:

  • Tree-structural pseudo-classes target elements based on document structure
  • :root - Root element (usually <html>), commonly for CSS variables
  • :empty - Elements with no children or content
  • :nth-child(n) - Select by position (odd, even, formulas)
  • :first-child/:last-child - First and last child of parent
  • :only-child - Only child element in parent
  • :nth-of-type(n) - Select by position of specific type
  • :first-of-type/:last-of-type - First/last of specific type
  • :only-of-type - Only element of that type in parent
These pseudo-classes provide powerful ways to select and style elements based on their structural position in the document tree
