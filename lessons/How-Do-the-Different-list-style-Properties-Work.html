CSS List-Style Properties

The list-style property controls the appearance of lists on a webpage.

It works with both ordered lists (ol) and unordered lists (ul), allowing you to customize how list items are displayed.

Shorthand Property

list-style is actually a shorthand for three individual properties that each control different aspects of list appearance.

Three List-Style Properties

The list-style shorthand encompasses three individual properties:

list-style-type

Controls the bullet or number style

list-style-position

Controls bullet/number placement

list-style-image

Uses custom images as bullets

Each property plays a different role in defining list appearance.

list-style-type

The list-style-type property defines the type of bullet point or number used in a list.

Unordered Lists

Choose from bullet styles: disc, circle, or square

Ordered Lists

Choose from numbering systems: decimal, Roman numerals, or alphabetical characters

list-style-type is the most commonly used of the three properties.

list-style-type Example

HTML Code

<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Result

  • Item 1
  • Item 2
  • Item 3

The bullet points of the unordered list are changed to squares.

list-style-position

The list-style-position property controls the position of the bullet or number in relation to the list item's content.

outside (default)

The bullet or number appears outside the content.

inside

The bullet or number appears inside the content, causing text to wrap and align with the bullet.

list-style-position Example

HTML Code

<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>

Visual Comparison

Inside:

  • Item 1
  • Item 2
  • Item 3

Outside:

  • Item 4
  • Item 5
  • Item 6

When to Use list-style-position

The list-style-position property is useful for controlling the alignment of list content.

Especially Helpful When:

  • You have multiple lines of text in a single list item
  • You need specific text alignment with bullets
  • You're working with complex list layouts

list-style-image

The list-style-image property allows you to use an image as the bullet point for your list items.

This can be useful for adding a unique visual style to your lists.

Best Practices

  • Choose images that are small and appropriate for your design
  • Avoid large or complex images that make lists harder to read
  • Ensure the image enhances, not distracts from, content

list-style-image Example

CSS Code

<style>
  ul {
    list-style-image: url('image.png');
    list-style-position: inside;
  }
</style>

HTML

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Result

The bullet points are replaced by a custom image, adding a personalized touch to the list.

Note

Make sure the image is small and appropriate for the design. Large or complex images can make lists harder to read.

The list-style Shorthand

You can combine all three properties into a single list-style shorthand property.

Key Points

  • The order of values doesn't matter
  • All three properties can be specified together
  • Values can be omitted (defaults will be used)

Shorthand Example

HTML Code

<ul style="list-style: square inside 
url('image.png');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

What This Does

  • square - Sets the list-style-type
  • inside - Sets the list-style-position
  • url('image.png') - Sets the list-style-image

Image Fallback

When using list-style-image, it's important to also specify a list-style-type.

Why?

If the image is unavailable or not rendered, the specified bullet style will be used as a fallback.

ul {
  list-style: square inside url('image.png');
}

In this example, if the image fails to load, square bullets will be displayed instead.
