Creating Gaps in Grid Layouts

Controlling Grid Spacing

In previous lessons, we talked a little bit about how to create space between grid items. But in this lesson, we will dive into more detail about how to use the row-gap, column-gap, and gap properties in a grid layout.

Three Properties for Grid Gaps:

  • column-gap – Space between columns
  • row-gap – Space between rows
  • gap – Shorthand for both

Why Gaps Matter: Gaps create visual breathing room, improve readability, and make your grid layouts more professional and polished.

Understanding Tracks

The Foundation of Gaps

But first we need to review what a track is in CSS grid.

A track is the space between two neighboring grid lines. These lines are automatically created when you use CSS Grid. In this context, tracks generally refer to the rows and columns that make up the grid layout.

Grid Lines and Tracks:

Grid Lines (dark borders) define the edges
Tracks (blue highlight) are the spaces between lines

Key Concept: Grid lines are created automatically, and the spaces between them (tracks) are what contain your grid items.

The column-gap Property

Creating Space Between Columns

To create gaps between columns in a CSS Grid, you can use the column-gap property.

column-gap Values

Acceptable values include:

  • pixels – e.g., 10px, 20px
  • em unit – e.g., 1em, 2em
  • percentages – e.g., 5%, 10%
  • normal keyword – default spacing

The "normal" value: If you use the normal value for column-gap, the result will be 0 for grid layouts and 1em for multi-column layouts.

column-gap in Action

CSS Implementation

For the CSS, we set the display to grid and the column-gap property to 10 pixels:

.grid-container { display: grid; grid-template-columns: 1fr 1fr 1fr 1fr; column-gap: 10px; }

The row-gap Property

Creating Space Between Rows

If we wanted to change the example to have two rows of boxes and create more space between the rows, we can use the row-gap property.

.grid-container { display: grid; grid-template-columns: 1fr 1fr; column-gap: 10px; row-gap: 30px; }

Changes: We're setting row-gap to 30 pixels and changing grid-template-columns to use just two 1fr units instead of four to create two rows of boxes.

Just like column-gap, acceptable values for row-gap can include percentages, em, and pixels.

The gap Shorthand Property

Simpler Syntax

If you want to use a shorthand for creating gaps between rows and columns, you can use the gap property.

gap: row-value optional-column-value;

How gap Works:

  • One value: Applied to both rows and columns
  • Two values: First = row, Second = column
/* One value - same for both */ gap: 20px; /* Equivalent to: */ row-gap: 20px; column-gap: 20px; /* Two values - different for each */ gap: 30px 10px; /* Equivalent to: */ row-gap: 30px; column-gap: 10px;

What Values Can You Use?

Supported Units

Acceptable values for the gap shorthand property include several CSS units and functions.

✓ Acceptable Values

  • Pixels – gap: 10px;
  • Em units – gap: 1em;
  • Percentages – gap: 5%;
  • calc() function – gap: calc(10px + 1em);

⚠️ Important: You cannot use fr units with the gap property. fr units are only for grid-template-columns and grid-template-rows.

/* ✓ Valid */ gap: 20px; gap: 1.5em 2em; gap: 5% 3%; gap: calc(10px + 5%); /* ✗ Invalid */ gap: 1fr; /* Won't work! */

Benefits of Gap Properties

Why Use These Properties?

The row-gap, column-gap, and gap properties provide flexible ways to control spacing between items in a CSS Grid layout.

  • Clean spacing – No need for margins on items
  • Consistent gaps – Applied uniformly across the grid
  • Easy to adjust – Change one value, update all gaps
  • Flexible units – Use pixels, em, percentages, or calc()
  • Shorthand convenience – gap property saves time
  • Visual clarity – Better readability and organization

Best Practice: By using these properties, you can easily create visually appealing grids with consistent and adjustable gaps between rows and columns.

Summary

Key Takeaways

Remember:

  • Tracks are spaces between grid lines
  • column-gap creates space between columns
  • row-gap creates space between rows
  • gap is the shorthand for both
  • One value: same for rows and columns
  • Two values: first = row, second = column
  • Acceptable values: pixels, em, %, calc()
  • Not allowed: fr units in gap properties
/* Three ways to create the same gaps */ /* Method 1: Individual properties */ row-gap: 20px; column-gap: 20px; /* Method 2: gap with one value */ gap: 20px; /* Method 3: gap with two values (if different) */ gap: 30px 10px; /* 30px rows, 10px columns */
