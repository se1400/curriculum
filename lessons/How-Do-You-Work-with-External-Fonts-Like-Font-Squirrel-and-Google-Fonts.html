What Are External Fonts?

An external font is a font file that is not included directly within your project files. They're usually hosted on a separate server.

What is a Server?

A computer that provides data or services to other computers over a network.

External fonts give you more flexibility—use custom fonts that may not be installed on the user's device

External Fonts: Two Implementation Methods

When you click "Get embed code," you'll see instructions for adding external fonts to your project.

Two Options for Web Development:

  • Link element: Add HTML links in the <head>
  • @import: Import fonts in your CSS file
Both methods load fonts from Google's servers when users visit your website

Method 1: Link Element

Copy and paste the HTML snippet into the <head> element of your HTML file.

HTML Example:

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto...">
<link rel="stylesheet" href="styles.css">

CSS Example:

.roboto-regular {
  font-family: "Roboto", sans-serif;
  font-weight: 400;
}

Note:

Each CSS rule includes fallback fonts (like sans-serif) in case custom fonts don't load properly.

Method 2: @import

Add the @import rule to your CSS file.

CSS Example:

@import url('https://fonts.googleapis.com/css2?family=Roboto...');

.roboto-regular {
  font-family: "Roboto", sans-serif;
  font-weight: 400;
}

Same CSS Rules:

The CSS rules for applying fonts are exactly the same regardless of whether you use link elements or @import.

Selective Inclusion

You can remove specific font styles from the snippets. The HTML and CSS snippets will update automatically.

Using Font Squirrel Fonts

You can add Font Squirrel fonts to your project in two ways:

Option 1: Local Files

Include the downloaded font files directly in your project directory

@font-face {
  font-family: 'CustomFont';
  src: url('fonts/customfont.woff2') format('woff2');
}

Option 2: External Hosting

Host fonts on a separate server and use them as external fonts (similar to Google Fonts)

Benefits & Considerations

Benefits of External Fonts:

  • Customize the overall look and design
  • Make your website truly unique
  • Access to thousands of font options
  • Professional typography without purchasing fonts

⚠️ Important Consideration:

Using multiple external fonts can increase the load time of the website, affecting its overall performance.

Always strive to find a balance between style and efficiency
