The Superscript Element

The superscript element (<sup>) displays text as a superscript - a symbol or letter printed above the normal line of text.

Basic Example: Exponents

<p>2<sup>2</sup> (2 squared) is 4.</p>
Output:
22 (2 squared) is 4.
The number 2 is wrapped in <sup> tags to represent the superscript. The second number 2 appears smaller and slightly higher than the first number 2.

Superscript Use Cases

Exponents: Mathematical expressions like 2², x³, or n⁴
Superior Lettering: Letters in superscript for abbreviations
Ordinal Numbers: 1st, 2nd, 3rd, 4th

Superior Lettering Example

Superior lettering refers to letters written in superscript, usually to indicate abbreviations.
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
Output:
Monseigneur is often written as Mgr.

When to Use Superscript

Important: The superscript element should only be used for typographical reasons. If you want to style a piece of text with a raised baseline for visual styling purposes, you should use CSS instead of the superscript element.
Use <sup> when the superscript conveys semantic meaning (like in mathematical notation or abbreviations), not just for visual styling.
For purely stylistic raised text, use CSS properties like:
vertical-align: super;
font-size: smaller;

The Subscript Element

To represent chemical equations in HTML, use the subscript element (<sub>). This element displays text with a lowered baseline using smaller text.

Chemical Formula Example

<p>CO<sub>2</sub></p>
Output:
CO2
The number two is wrapped inside <sub> tags to illustrate that the character should be displayed as a subscript below the baseline.

Subscript Use Cases

Chemical Formulas: H2O, CO2, C6H12O6
Footnotes: Reference markers in text1, like citations2
Variable Subscripts: Mathematical notation like x1, x2, or an
Like the superscript element, <sub> should be used for semantic meaning - when the subscript is part of the notation itself, not just for styling purposes.
