What is an Array?

Ordered Collection of Values

An array in JavaScript is an ordered collection of values, each identified by a numeric index.

Think of an array as a list or container that can hold multiple values in a specific order.

Visual Representation

index 0
"apple"
index 1
"banana"
index 2
"orange"

Each value has a position (index) in the array

Arrays are one of the most fundamental data structures in programming, allowing you to store and organize multiple related values efficiently.

Creating Arrays

Using Square Brackets

To create an array in JavaScript, use square brackets [] and separate values with commas.

let fruits = ["apple", "banana", "orange"];

Breaking It Down

  • let fruits - Declare a variable named fruits
  • [ ] - Square brackets create the array
  • "apple", "banana", "orange" - Three string values separated by commas
Syntax Pattern:
let arrayName = [value1, value2, value3, ...];

Arrays Can Hold Different Data Types

Versatile Storage

The values in a JavaScript array can be of different data types, including:

Numbers
[1, 2, 3, 4, 5]
Strings
["hello", "world"]
Booleans
[true, false, true]
Objects
[{name: "John"}, {name: "Jane"}]
Other Arrays
[[1, 2], [3, 4]]
Mixed Types
[1, "hello", true] 
// Array with mixed data types
let mixedArray = [
    42,                    // number
    "hello",               // string
    true,                  // boolean
    { name: "John" },       // object
    [1, 2, 3]              // array
];
This flexibility makes arrays incredibly powerful for storing diverse collections of data in your programs.

Zero-Indexed Arrays

A Key Characteristic

One of the key characteristics of arrays is that they are zero-indexed.

This means:

  • The first element has an index of 0
  • The second element has an index of 1
  • The third element has an index of 2
  • And so on...

Zero-Indexed Example

0 (first)
"apple"
1 (second)
"banana"
2 (third)
"orange"
Common Mistake: Beginners often forget that arrays start at 0, not 1! The first element is at index 0.

Accessing Array Elements

Using Index Notation

You can access individual elements in an array using their index with square bracket notation: arrayName[index]

let fruits = ["apple", "banana", "orange"];

console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"

The length Property

Counting Array Elements

Arrays in JavaScript have a special length property that returns the number of elements in the array.

You can access this property using the length keyword: arrayName.length

let fruits = ["apple", "banana", "orange"];

console.log(fruits.length); // 3

Length Property

0
"apple"
1
"banana"
2
"orange"

fruits.length = 3

The length property is very useful for loops and determining how many items are in an array!

Length vs Last Index

Important Distinction

Because arrays are zero-indexed, the length property is always one more than the highest index.

Property
Value
First Index
0
Last Index
length - 1
Total Elements
length
let fruits = ["apple", "banana", "orange"];

console.log(fruits.length);           // 3 (total elements)
console.log(fruits[0]);               // "apple" (first element)
console.log(fruits[fruits.length - 1]); // "orange" (last element)
To access the last element:
arrayName[arrayName.length - 1]
Trying to access arrayName[arrayName.length] will return undefined because that index doesn't exist!

Dynamic Arrays

Size Can Change

Another key characteristic of arrays in JavaScript is that they are dynamic, meaning that their size can change after they are created.

You can add or remove elements from an array at any time.

Array Methods (Preview)

JavaScript provides various methods to modify arrays:

  • push() - Add elements to the end
  • pop() - Remove element from the end
  • shift() - Remove element from the beginning
  • unshift() - Add elements to the beginning
  • splice() - Add or remove elements at any position
  • And many more...
These array methods will be taught in detail in upcoming lessons. For now, just know that arrays are flexible and can grow or shrink as needed! 
// Example of dynamic behavior (methods taught later)
let numbers = [1, 2, 3];
console.log(numbers.length); // 3

numbers.push(4); // Add element
console.log(numbers.length); // 4

numbers.pop(); // Remove element
console.log(numbers.length); // 3

Arrays Are Ordered Collections

Order Matters!

Arrays maintain the order of their elements. This is a fundamental characteristic that distinguishes arrays from other data structures.

Order is Preserved

Original Array:

0
"first"
1
"second"
2
"third"

The elements will always stay in this order unless you explicitly change it

Important: The statement "arrays are an unordered collection of values" is FALSE. Arrays ARE ordered!

Summary

Key Characteristics of JavaScript Arrays

  • Ordered collection: Arrays maintain the order of elements
  • Created with brackets: Use [] with comma-separated values
  • Multiple data types: Can contain numbers, strings, booleans, objects, and even other arrays
  • Zero-indexed: First element is at index 0, second at index 1, etc.
  • Access with brackets: Use arrayName[index] to access elements
  • length property: Returns the number of elements in the array
  • Dynamic size: Can grow or shrink after creation
  • Array methods: Many built-in methods for manipulation (covered in future lessons)
// Creating an array
let fruits = ["apple", "banana", "orange"];

// Accessing elements (zero-indexed)
console.log(fruits[0]);    // "apple"

// Getting array length
console.log(fruits.length); // 3

// Accessing last element
console.log(fruits[fruits.length - 1]); // "orange"
