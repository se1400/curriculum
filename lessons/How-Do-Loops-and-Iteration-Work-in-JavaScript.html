Introduction to Loops

What Are Loops?

Loops in programming are used to repeat a block of code multiple times.

Instead of writing the same code over and over, loops allow you to execute code repeatedly with different values or conditions.

Real-World Examples

Example 1: Printing a List

When designing a program that needs to print out a list of items, you could use a loop to print out each item in the list.

Items: ["Apples", "Bananas", "Oranges"]
‚Üí Loop prints each item one by one

Example 2: Game Character Movement

When designing a game and you want to move a character across the screen, you could use a loop to move the character a certain number of pixels each time the loop runs.

‚Üí Loop moves character 10 pixels each iteration
Key Point:

JavaScript has several types of loops. In this lesson, we'll focus on the for loop, which is one of the most commonly used loops.

For Loop Syntax

Basic Structure

for (initialization; condition; increment/decrement) { // code block to be executed }

The Three Parts of a For Loop

1. Initialization

Executed before the loop starts. Typically used to initialize a counter variable.

let i = 0

2. Condition

Evaluated before each iteration (each pass through the loop). If true, the code block executes. If false, the loop stops.

i < 5

3. Increment/Decrement

Executed after each iteration. Typically used to increment or decrement the counter variable.

i++
Counter Variable:

A counter variable (commonly i) keeps track of how many times the loop has run. Using i is a common convention in for loops.

A Simple For Loop Example

Basic Loop

for (let i = 0; i < 5; i++) {
    console.log(i);
}
Output:

3
4

Breaking It Down

  • let i = 0 - Start with i at 0
  • i < 5 - Continue as long as i is less than 5
  • i++ - Increase i by 1 after each iteration
  • console.log(i) - Print the current value of i

Loop Executes 5 Times

Iteration 1: i = 0 ‚Üí Prints 0
Iteration 2: i = 1 ‚Üí Prints 1
Iteration 3: i = 2 ‚Üí Prints 2
Iteration 4: i = 3 ‚Üí Prints 3
Iteration 5: i = 4 ‚Üí Prints 4
Then i = 5, condition is false, loop stops

The Process Repeats:

We keep checking the condition, executing the code, and incrementing until the condition becomes false.

Key Insight:

When i becomes 5, the condition i < 5 is false, and the loop stops. The code moves on to whatever comes after the loop.

What Is an Infinite Loop?

An infinite loop is a loop that runs forever because the condition is always true.

‚ö†Ô∏è DANGER: Infinite Loop Example 
// DO NOT RUN THIS CODE!
for (let i = 0; i < 5; i--) {  // i-- decreases instead of increases!
    console.log(i);
}

// This will crash your program!

In this example, i starts at 0 and decreases each time (i--), so it will always be less than 5. The loop will never stop!

üí° How to Avoid Infinite Loops:
  • Make sure your condition can eventually become false
  • Ensure your increment/decrement moves toward the condition being false
  • Double-check the logic before running the loop
  • Test with small values first

Common Causes of Infinite Loops

  • Forgetting to increment/decrement the counter
  • Incrementing when you should decrement (or vice versa)
  • Using the wrong comparison operator
  • Condition that can never be false

Summary & Key Takeaways

What We Learned

  • Loops repeat a block of code multiple times
  • For loops have three parts: initialization, condition, increment/decrement
  • Initialization runs once before the loop starts
  • Condition is checked before each iteration
  • Increment/Decrement runs after each iteration
  • Loop stops when the condition becomes false
  • Avoid infinite loops by ensuring the condition can become false

For Loop Syntax Reminder
