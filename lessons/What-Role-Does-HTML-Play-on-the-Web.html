HTML's Role in Web Development

Hypertext Markup Language (HTML) is the foundational markup language for creating web pages.
HTML provides the content and structure you see on websites:
• Paragraphs and headings
• Links and images
• Videos and other media
Example Structure:
<h1>Main heading element</h1>
<p>I am a paragraph element.</p>

HTML Elements: Tags & Structure

Elements consist of:
• Opening tag (start tag)
• Content (text or other HTML elements)
• Closing tag (end tag)
Tag Syntax:
<p>Content goes here</p>
Tags use angle brackets: < and >
Closing tags include a forward slash: /
Best Practice: Write tag names in lowercase (though HTML is case-insensitive)

Void Elements

Void elements don't have closing tags and cannot contain content.
Image Element Example:
<img>
Alternative Format:
<img />
Both formats are valid. The trailing / is optional and has no effect according to HTML specs.
Note: You'll see both variations in professional development.

HTML Attributes

Attributes are special values that modify HTML element behavior.

The src Attribute

Specifies the location/source of an image.
<img src="https://example.com/image.jpg">

The alt Attribute

Provides descriptive text for images (accessibility and fallback).
<img src="https://example.com/cats.jpg" alt="Two cats sleeping">
Best Practice: Always include the alt attribute with images.

Is HTML Enough?

For simple projects: HTML alone can display text and images.
For modern professional websites: You need all three technologies:

The Three Pillars of Web Development

HTML – Content and structure (the foundation)
CSS – Styling and visual design (the aesthetics)
JavaScript – Interactivity and functionality (the systems)

The Building Analogy

Think of a website like a complete building:
HTML = Structural Framework
The blocks, concrete, and steel that form the walls and foundation
Provides structural integrity
CSS = Interior/Exterior Design
The paint, decorations, and layout
Creates visual appeal
JavaScript = Electrical & Plumbing
The systems that make everything functional
Ensures interactivity and dynamic features
