Anatomy of a CSS Rule

CSS is responsible for the styles of a web page. All of these styles are made up of various CSS rules.
Two Main Parts
A CSS rule is made up of two main parts: a selector and a declaration block.

Basic Syntax

CSS Rule Structure:
selector { property: value; }
Selector
Identifies which HTML elements to target
Declaration Block (curly braces)
Contains the styles to apply
Declaration
Property and value pair inside the block

Selectors

What Is a Selector?
A selector is a pattern used in CSS to identify and target specific HTML elements for styling.
Type Selectors
Target elements by their tag name (e.g., p, h1, div)
Class Selectors
Target elements with a specific class attribute (start with .)
ID Selectors
Target elements with a specific id attribute (start with #)

Declaration Block

Curly Braces
The curly braces provided in the basic syntax are known as a declaration block. A declaration block applies a set of styles for a given selector, or selectors.
Inside the Block
Inside the declaration block, you will have a series of declarations. Each declaration consists of a property and a value.

Properties and Values

Property
The property is the CSS identifier that specifies which feature is being styled. An example of a property would be the background-color property.
Value
The value would be the specific setting applied to that property. For example, if the property is background-color, a value could be purple, which sets the background color to purple.
After each property name, you need to place a colon, and after each value, you should have a semicolon.

Complete Example

CSS Rule for Paragraphs:
p { color: blue; }
Selector: p
A type selector that targets all paragraph elements on the page
Declaration Block: { }
Contains one declaration
Declaration: color: blue;
Property is "color" and value is "blue" - changes text color for all paragraph elements to blue

Selector Lists

If you want to apply the same set of styles to multiple selectors, you can create a selector list with each selector separated by a comma.
Multiple Selectors Example:
#title, .subheading { color: navy; }
In this selector list, there is an ID selector (#title) targeting the HTML element with the id value of title, and a class selector (.subheading) targeting all HTML elements with the class value of subheading.

ID and Class Selectors

ID Selector (#)
#title { color: navy; }
All ID selectors must start with a hash (#) symbol. Targets the element with id="title".
Class Selector (.)
.subheading { color: navy; }
All class selectors must start with a dot (.) symbol. Targets all elements with class="subheading".
