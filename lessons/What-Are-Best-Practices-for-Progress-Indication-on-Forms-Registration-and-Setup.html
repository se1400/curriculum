Progress Indication

A way to show users how far they are in a process.

Common Uses

  • Forms
  • Registration processes
  • Setup processes

Goal

Help users understand where they are and how much more they need to do.

Transparency is Key

Use progress indication bars to show users what is left to do when filling forms.

Problem Without Progress Indication

Users filling out lengthy forms don't know how many more steps they need to complete.

Benefit

Users can decide if they have enough time to complete the form now, or if they need to come back later.

Best Practice: Keep It Simple

Don't overwhelm users with too much information.

Result of Complexity

Users get frustrated and may leave the site.

Solution

Use clear, minimal progress indicators that are easy to understand at a glance.

Best Practice: Allow Going Back

Make it possible to go back to previous steps.

Why It's Important

Users may want to:

  • Check their previous answers
  • Make changes to earlier entries

Navigation Example

Contact Information

Best Practice: Easy to Find

Make the progress indication section easy to find.

Problem

If users can't find it, they won't know how far they are in the process.

Solution

Place progress indicators at the top of the form where they're immediately visible.

Best Practice: Clear Context

Include clear section titles, percentages, or steps.

Problem

A progress bar with no context leaves users confused about what it means.

Solution

Always label progress indicators with meaningful information.

Best Practices Summary

  • Keep it simple - Don't overwhelm users with information
  • Allow going back - Users may need to review or change answers
  • Easy to find - Place indicators prominently
  • Clear context - Use section titles, percentages, or step numbers
  • Transparency - Help users decide if they have time to complete
