CSS Length Units: Two Types

When designing pages with properties like widths, heights, padding, and margins, you must specify length units of measurement.

Two Types of Units

  • Absolute Units: Fixed length, not relative to anything else
  • Relative Units: Length relative to something else (screen size, parent element, etc.)
This lesson focuses on absolute units only.

Pixels (px): The Most Common Absolute Unit

What is a Pixel?

Pixels provide precise, fixed-size control over dimensions in CSS.

Standardization: 1px = 1/96th of an inch

Important Note

While 1px is standardized as 1/96th of an inch for CSS layout purposes, the actual physical size of a pixel may differ depending on the display device.

Setting Dimensions with Pixels

Fixed Dimensions

Dimensions set with pixels remain constant regardless of screen size.

100px × 100px box (actual size)

Whether on a small mobile screen or large desktop, this box stays 100px wide.

HTML

<div class="box"></div>

CSS

.box {
  width: 100px;
  height: 100px;
  background-color: red;
}

When Should You Use Absolute Units?

Use Cases

The decision depends on the situation, but generally use pixels when you need:

  • Precise control over element dimensions
  • Exact spacing control
  • Specific layout measurements

Common Applications

Pixels are often used for:

  • Margins
  • Padding
  • Borders

Setting Margins with Pixels

Margin: Space Outside Elements

Margins create space around elements. In this example, each box has 10px of space on all sides.

Notice the 10px margin around each box

HTML

<div class="box"></div>
<div class="box"></div>

CSS

.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}

Other Absolute Units

While pixels are most common for screen design, other absolute units exist, primarily used for print:

Unit Name Equals
in Inches 96px
cm Centimeters 25.2/64 of an inch
mm Millimeters 1/10th of a centimeter
q Quarter-millimeters 1/40th of a centimeter
pc Picas 1/6th of an inch
pt Points 1/72th of an inch
Remember: Most of these units are for print, not screens. Pixels are your go-to for web design.

Key Takeaways

About Absolute Units

  • Absolute units have fixed lengths
  • They are not relative to screen size or parent elements
  • Pixels (px) are the most common absolute unit for web design
  • 1px = 1/96th of an inch (CSS standard)

When to Use Pixels

  • When you need precise control over dimensions
  • For margins, padding, and borders
  • When exact measurements are required

Other Absolute Units

  • in, cm, mm, q, pc, pt exist but are mainly for print
  • Important to know they exist, but you'll rarely use them for web
