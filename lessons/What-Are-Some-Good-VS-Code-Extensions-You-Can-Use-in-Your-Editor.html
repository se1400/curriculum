Code Commenting

Better Comments is an extension that offers special highlighting for specific code comments.

It will call out "TODO" comments, as well as syntax to indicate questions or warnings.

This makes it easier to identify important notes and reminders in your codebase at a glance.

Better Comments

Highlights special comment types with different colors:

• TODO items

• Questions

• Warnings

• Important notes

Spell Checking

Code Spell Checker offers indication when you have spelled something incorrectly in your code.

Because it is designed for code files, it accounts for things like camel case.

This helps catch typos in variable names, comments, and strings without flagging common coding patterns.

Code Spell Checker

Designed specifically for code, this extension:

• Detects spelling errors

• Understands camelCase

• Recognizes common code patterns

• Works across multiple languages

Error Detection

Error Lens will help you catch any highlighted errors in your code.

Rather than having to rely on the underline that VS Code shows by default, this extension highlights the entire line and displays the error message.

Errors become much more visible and easier to fix quickly.

Error Lens

Makes errors impossible to miss by:

• Highlighting the entire error line

• Displaying the error message inline

• Making errors more visible

• Helping you catch issues faster

Code Visualization

Indent Rainbow adds color to your different levels of indentation.

In combination with VS Code's native bracket pair colorization, this can dramatically help identify what scope your code is in.

Color-coded indentation makes nested code structures easier to understand.

Indent Rainbow

Improves code readability by:

• Adding colors to indentation levels

• Making nested structures clear

• Working with bracket colorization

• Helping identify code scope

AI Assistants and Visual Tools

Consider an AI assistant, like GitHub Copilot or Tabnine, to offer you inline suggestions as you write your code.

An icon pack, such as VS Code Great Icons, can help make your file tree cleaner and easier to parse at a glance.

An extension like Colorize can help you understand the values in your CSS properties.

GitHub Copilot / Tabnine

AI-powered code suggestions as you type

VS Code Great Icons

Cleaner file tree with better icons for different file types

Colorize

Visualizes color values directly in your CSS code
