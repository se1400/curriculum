The innerHTML Property

What is innerHTML?

innerHTML is a property of Element objects that you can use to set their HTML markup.

element.innerHTML = "HTML string";

How It Works

You can set the HTML structure of an existing element with a string, creating all the necessary nodes.

Think of it as writing HTML within a string.

Basic Example

// Start with an empty div <div id="container"> <!-- Add new elements here --> </div> // Select the element const container = document.getElementById("container"); // Set innerHTML to create new elements container.innerHTML = "<p>Hello World!</p>";

Using innerHTML - Step by Step

Example: Adding a List of Pizza Ingredients

1 Start with Empty Container

<div id="container"></div>

const container = document.getElementById("container");

3 Set innerHTML with HTML String

// Create an unordered list with ingredients container.innerHTML = ` <ul> <li>Cheese</li> <li>Tomato</li> </ul> `;

You can write the string on multiple lines using template literals (backticks) to make it easier to understand.

innerHTML Result

After innerHTML

<div id="container"> <ul> <li>Cheese</li> <li>Tomato</li> </ul> </div>

innerHTML Security Risks

‚ö†Ô∏è Security Warning

innerHTML has security risks that you should be familiar with.

You shouldn't use it if you won't have full control over the string.

The Problem: User-Generated Content

If the string will be entered by the user, you shouldn't use innerHTML because the user might insert malicious content into your website.

Dangerous Example

// ‚ùå DANGEROUS - User input directly into innerHTML const userInput = document.getElementById("user-text").value; container.innerHTML = userInput; // Could contain malicious code!

What Could Go Wrong?

‚Ä¢ Cross-site scripting (XSS) attacks

‚Ä¢ Malicious scripts executed

‚Ä¢ Stolen user data

‚Ä¢ Compromised website

‚úì Safe Alternative

Use textContent instead to insert plain text.

// ‚úì SAFE - textContent for user input container.textContent = userInput;

The createElement() Method

Creating Elements Programmatically

Another way to create a new node is by using the createElement() method.

You create a new element by specifying its tag name.

document.createElement(tagName)

Basic Examples

// Create an image element document.createElement("img"); // Create a paragraph element document.createElement("p"); // Create a div element document.createElement("div");
// Assign to a variable const img = document.createElement("img"); // Now you can modify it img.src = "photo.jpg"; img.alt = "Description";

Return Value

The createElement() method returns:

‚Ä¢ A new HTMLElement object (if document is an HTMLDocument)

‚Ä¢ Or an Element object

Using createElement() with appendChild()

Complete Process

Once you have the new element ready, you need to add it to the DOM.

Step-by-Step Example

1 Create the Element

const newParagraph = document.createElement("p");

2 Set Its Content and Attributes

newParagraph.textContent = "This is a new paragraph!"; newParagraph.className = "highlight";

3 Add to DOM with appendChild()

const container = document.getElementById("container"); container.appendChild(newParagraph);

The appendChild() method adds the new element as a child of the container.

üí° Other Insertion Methods

You can also insert elements at specific locations using:

‚Ä¢ insertBefore() - Insert before another element

‚Ä¢ insertAdjacentElement() - Insert at a specific position

‚Ä¢ prepend() - Insert at the beginning

‚Ä¢ append() - Insert at the end

Choose the one that best fits your needs.

createElement() Complete Example

Building a List Dynamically

// Create the ul element const list = document.createElement("ul"); // Create first li element const item1 = document.createElement("li"); item1.textContent = "Cheese"; // Create second li element const item2 = document.createElement("li"); item2.textContent = "Tomato"; // Add list items to the ul list.appendChild(item1); list.appendChild(item2); // Add the complete list to the container const container = document.getElementById("container"); container.appendChild(list);
