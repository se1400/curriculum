Adding and Removing Array Elements

Dynamic Arrays in JavaScript

Arrays in JavaScript are dynamic, meaning you can easily add or remove elements from them.

There are four main methods for manipulating the beginning and end of arrays:

  • push() - Add to the end
  • pop() - Remove from the end
  • unshift() - Add to the beginning
  • shift() - Remove from the beginning

The push() Method

The push() method adds one or more elements to the end of an array and returns the new length of the array. 

const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");

console.log(newLength); // 3
console.log(fruits);    // ["apple", "banana", "orange"]
Output: 3
["apple", "banana", "orange"]

Understanding const with Arrays

Why Can We Modify a const Array?

You might notice we're using const to declare arrays, yet we can still modify them. How is this possible?

Key Concept:

Declaring an array with const creates a reference to the array. The array itself is mutable (can be modified), but you cannot reassign the variable to a new value.

What Works vs. What Doesn't

✓ This Works:
const fruits = ["apple", "banana"];
fruits.push("orange"); // Modifying the array is OK
console.log(fruits); // ["apple", "banana", "orange"]
✗ This Doesn't Work:
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"]; // Reassignment is NOT OK
Uncaught TypeError: Assignment to constant variable.

Removing from the End: pop()

The pop() Method

The pop() method removes the last element from an array and returns that element. It also modifies the original array. 

let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();

console.log(fruits);     // ["apple", "banana"]
console.log(lastFruit);  // "orange"
Array: ["apple", "banana"]
Removed: "orange"

How pop() Works

  • Removes the last element from the array
  • Returns the removed element
  • Modifies the original array (reduces length by 1)
  • Removes only one element at a time

Before pop()

["apple", "banana", "orange"]
Length: 3

After pop()

["apple", "banana"]
Length: 2

Adding to the Beginning: unshift()

The unshift() Method

The unshift() method adds one or more elements to the beginning of an array and returns the new length.

It works similarly to push(), but modifies the start of the array instead of the end. 

let numbers = [2, 3];
let newLength = numbers.unshift(1);

console.log(numbers);    // [1, 2, 3]
console.log(newLength);  // 3
Array: [1, 2, 3]
New Length: 3

How unshift() Works

  • Adds element(s) to the beginning of the array
  • Returns the new length of the array
  • Shifts all existing elements to higher indices
  • Can add multiple elements at once

Before unshift(1)

[2, 3]
Length: 2

After unshift(1)

[1, 2, 3]
Length: 3

Removing from the Beginning: shift()

The shift() Method

The shift() method removes the first element from an array and returns that element.

It's similar to pop(), but works at the beginning of the array instead of the end. 

let colors = ["red", "green", "blue"];
let firstColor = colors.shift();

console.log(colors);      // ["green", "blue"]
console.log(firstColor);  // "red"
Array: ["green", "blue"]
Removed: "red"

How shift() Works

  • Removes the first element from the array
  • Returns the removed element
  • Shifts all remaining elements to lower indices
  • Removes only one element at a time

Before shift()

["red", "green", "blue"]
Length: 3

After shift()

["green", "blue"]
Length: 2

Summary: All Four Methods

Method Comparison

Method Action Returns Multiple Elements?
push() Add to end New length ✓ Yes
pop() Remove from end Removed element ✗ No (one at a time)
unshift() Add to beginning New length ✓ Yes
shift() Remove from beginning Removed element ✗ No (one at a time)
Important Note:

While push() and unshift() can add multiple elements at once, pop() and shift() remove only one element at a time.

Key Takeaways

  • All four methods modify the original array
  • push() and pop() work on the end of arrays
  • unshift() and shift() work on the beginning
  • Adding methods return the new length
  • Removing methods return the removed element
  • const arrays can be modified but not reassigned
