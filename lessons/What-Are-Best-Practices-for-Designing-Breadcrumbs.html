Breadcrumbs

Web design includes many types of navigational aids.

Common Navigation Types

  • Top navigation bars
  • Sidebars
  • Footers
  • Breadcrumbs - For complex sites with deeper navigation levels

What Are Breadcrumbs?

A navigation aid that shows users where they are in the site's hierarchy.

Home / Electronics / Phones / Smartphone XYZ

Example from a mock electronics website

Understanding the Path

Breadcrumbs are displayed at the top of the page, showing the path users took to reach the current page.

Homepage → Electronics → Phones → Smartphone XYZ

Benefits of Breadcrumbs

Help users understand where they are in the site's hierarchy and how to navigate back to previous pages.

Especially Useful When:

  • User arrives from a search result
  • User comes from an external link
  • User needs to understand the context of the current page

Design Consideration: Separators

The separator is the character that separates different levels of the hierarchy.

Greater than sign (>)
Home > Electronics > Phones > Smartphone XYZ
Right angle quotation marks (»)
Home » Electronics » Phones » Smartphone XYZ
Forward slash (/)
Home / Electronics / Phones / Smartphone XYZ

HTML for Separators

<div class="breadcrumbs"> Home / Electronics / Phones / Smartphone XYZ </div> <div class="breadcrumbs"> Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ </div>

Design Consideration: Placement

Breadcrumbs are typically placed at the top of the page, either above or below the main navigation bar.

Important

Users shouldn't have to struggle to find breadcrumbs — make them visible and easy to locate.

Design Consideration: Size

Breadcrumbs should be large enough to be easily read, but not so large they take up too much space.

Remember

Breadcrumbs are a secondary navigational aid, not a primary navigation tool.

When Should You Use Breadcrumbs?

✓ Use Breadcrumbs For:

Sites with a deep hierarchy

  • E-commerce sites
  • News sites
  • Technical documentation sites

✗ Skip Breadcrumbs For:

Sites with a shallow hierarchy (only a few levels)

Breadcrumbs might not add much value in simple sites.

Best Practices Summary

  • Choose a separator - Use >, », or / consistently
  • Place at the top - Above or below main navigation, easy to find
  • Size appropriately - Readable but not oversized
  • Use for deep hierarchies - E-commerce, news, documentation sites
  • Secondary aid - Complement, don't replace primary navigation
