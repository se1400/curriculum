setAttribute() Basic Syntax

Method Overview

The setAttribute() method allows you to add or update attributes on HTML elements.

element.setAttribute(attribute, value);

Parameters

attribute: The name of the attribute (as a string)

value: The value to assign to the attribute (as a string)

Return Value

The method returns undefined.

It modifies the element directly without returning anything.

Simple Examples

// Add a class attribute element.setAttribute("class", "my-class"); // Add an id attribute element.setAttribute("id", "main-header"); // Add a src attribute to an image imgElement.setAttribute("src", "photo.jpg"); // Add a disabled attribute buttonElement.setAttribute("disabled", "true");

Example 1: Adding a New Attribute

Scenario: Adding a Class Attribute

We have a paragraph element without a class attribute, and we want to add one using JavaScript.

Starting HTML

<p id="para">I am a paragraph</p>

Current State

The paragraph has an id attribute but no class attribute.

Goal

Add a class attribute with the value "my-class" to the paragraph element.

Before:
<p id="para">I am a paragraph</p>
After:
<p id="para" class="my-class">I am a paragraph</p>

Example 1: Step-by-Step

1 Access the Element

const para = document.getElementById("para");

First, we need to access the paragraph element using getElementById().

2 Use setAttribute()

const para = document.getElementById("para"); para.setAttribute("class", "my-class");

Call setAttribute() with two arguments:

‚Ä¢ Attribute name: "class"

‚Ä¢ Attribute value: "my-class"

3 Verify with outerHTML

const para = document.getElementById("para"); para.setAttribute("class", "my-class"); console.log(para.outerHTML);

We use outerHTML to log the entire HTML element including its attributes.

Example 1: Result

Before setAttribute()

<p id="para"> I am a paragraph </p>

Attributes: Only id

After setAttribute()

<p id="para" class="my-class"> I am a paragraph </p>

Attributes: Both id and class

Console Output

"<p id=\"para\" class=\"my-class\">I am a paragraph</p>"

The class attribute has been successfully added!

üí° About outerHTML

The outerHTML property returns a string containing the HTML element including its opening and closing tags and all attributes.

This is different from innerHTML, which only returns the content inside the element.

Example 2: Updating an Existing Attribute

Key Feature

If you have an existing attribute on an HTML element, you can update its value using the setAttribute() method.

Starting HTML

<div class="my-class"></div>

We have a div element with a class attribute set to "my-class".

Goal

Update the class attribute value from "my-class" to "example".

Before:
<div class="my-class"></div>
After:
<div class="example"></div>

Example 2: Implementation

The Code

// Select the div element const divElement = document.querySelector("div"); // Update the class attribute value divElement.setAttribute("class", "example"); // Log to see the result console.log(divElement.outerHTML);

Before

<div class="my-class"></div>

After

<div class="example"></div>

Console Output

"<div class=\"example\"></div>"

The existing class attribute value has been successfully updated!

üí° Important Note

When you use setAttribute() on an existing attribute, it replaces the old value with the new value.

The old value ("my-class") is completely replaced by the new value ("example").

Real-World Use Cases

When to Use setAttribute()

What are some real-world examples of when to use the setAttribute() method?

Key Takeaways

What setAttribute() Does

The setAttribute() method adds a new attribute or updates the value of an existing attribute on an HTML element.

Syntax

element.setAttribute(name, value)

‚Ä¢ name: Attribute name (string)

‚Ä¢ value: Attribute value (string)

Behavior

‚úì Creates new attribute if it doesn't exist

‚úì Updates value if attribute exists

‚úì Always replaces old value completely

‚úì Both parameters must be strings

Common Use Cases

üñºÔ∏è Image galleries: Update src and alt attributes

üìù Form validation: Add required, minlength, maxlength

üé® Dynamic styling: Change class attributes

‚ôø Accessibility: Add ARIA attributes like aria-label

üîó Links: Update href attributes dynamically

üîò Buttons: Add/remove disabled attribute

üí° Pro Tip

While setAttribute() works for all attributes, you can also access many common attributes directly:

// Using setAttribute() element.setAttribute("id", "my-id"); // Direct property access (also works) element.id = "my-id";

However, setAttribute() is more flexible and works with custom attributes!
