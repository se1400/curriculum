What Are CSS Gradients?

Gradients in CSS allow you to create smooth transitions between two or more specified colors.

Smooth Color Transition

Why Use Gradients?

  • Add depth to designs
  • Create texture and visual interest
  • No image files needed - pure CSS
  • Enhance visual appeal without extra HTTP requests

Two Main Types of Gradients

CSS supports two main types of gradients:

Linear Gradients
→

Blend colors along a straight line

Radial Gradients
◉

Radiate from a central point

Linear Gradients: The Basics

Linear gradients create a gradual blend between colors along a straight line. You can control the direction and the colors used.

CSS
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  width: 100%;
  height: 200px;
}
Red to Blue

This creates a gradient that transitions from red on the left to blue on the right.

Linear Gradient Directions

Multi-Color Linear Gradients

You can use multiple color stops in a linear gradient:

CSS
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  width: 100%;
  height: 200px;
}
Red → Yellow → Blue

This creates a gradient that transitions from red to yellow to blue, evenly spaced across the element.

Radial Gradients: The Basics

Radial gradients create circular or elliptical gradients that radiate from a central point.

CSS
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  width: 100%;
  height: 200px;
}
Center to Edges

This creates a circular gradient that starts with red in the center and transitions to blue at the edges.

Radial Gradient Shapes

You can specify the shape of radial gradients:

Circle
circle
radial-gradient(circle, ...)
Ellipse
ellipse

Positioning Radial Gradients

You can specify where the gradient originates from:

CSS
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  width: 100%;
  height: 200px;
}
Top Left Origin

This creates an elliptical gradient starting from the top left corner of the element.

Precise Color Stops

Both linear and radial gradients allow you to specify color stops at precise positions:

CSS
.precise-gradient {
  background: linear-gradient(
    to right,
    red 20%,
    yellow 40%,
    blue 80%
  );
}
Controlled Color Stops

This creates a gradient where:

  • Red takes up the first 20%
  • Then transitions to yellow at 40%
  • Finally to blue at 80%

Versatile Use Cases

Gradients can be used wherever you'd use an image in CSS:

Common Applications:

  • Backgrounds - Page or section backgrounds
  • Buttons - Eye-catching call-to-action elements
  • Headers - Hero sections and page headers
  • Text effects - Combined with background-clip for gradient text
  • Overlays - Over images for better text readability
  • Borders - Using border-image property

Accessibility Considerations

Key Takeaways

Linear Gradients:

  • Transition colors along a straight line
  • Control direction with keywords (to right) or angles (45deg)
  • Support multiple color stops

Radial Gradients:

  • Radiate from a central point
  • Can be circular or elliptical
  • Position can be customized

Best Practices:

  • Use precise color stops for control
  • Ensure sufficient contrast for accessibility
  • Gradients work anywhere images work in CSS
  • No image files needed - pure CSS solution
