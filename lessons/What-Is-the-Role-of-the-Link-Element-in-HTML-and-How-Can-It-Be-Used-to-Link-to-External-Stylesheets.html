The Link Element: Connecting to External Resources

The link element connects HTML documents to external resources like stylesheets and site icons.

Basic Syntax

<link rel="stylesheet" href="./styles.css" />

Understanding Link Element Attributes

The rel Attribute

Specifies the relationship between the linked resource and the HTML document.

  • For stylesheets, use rel="stylesheet"
  • Tells the browser how to handle the linked resource

The href Attribute

Defines the location or URL of the external resource.

  • ./ - looks in the current folder
  • Example: href="./styles.css"

Best Practice: Separation of Concerns

Keep HTML and CSS in separate files!

Use the link element to reference external CSS files instead of embedding styles in HTML.

Proper Placement

The link element belongs inside the <head> element.

<head> <meta charset="UTF-8" /> <meta name="viewport" content="width=device-width, initial-scale=1.0" /> <title>Examples of the link element</title> <link rel="stylesheet" href="./styles.css" /> </head>

Multiple Link Elements in Professional Codebases

Professional projects often use multiple link elements for different resources:

  • Multiple stylesheets
  • External fonts
  • Icons and favicons

Example: Linking Google Fonts

<link rel="preconnect" href="https://fonts.googleapis.com" /> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin /> <link href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap" rel="stylesheet" />

Understanding preconnect

The preconnect value for the rel attribute optimizes performance.

How it works:

Tells the browser to establish an early connection to the specified URL, reducing loading times for external resources.

Google Fonts

A collection of free, open-source custom fonts for any project. Google provides the necessary HTML and CSS code for each font.

Linking to Icons: Favicons

Favicon Syntax

<link rel="icon" href="favicon.ico" />

What is a Favicon?

A favicon (favorite icon) is a small icon displayed in the browser tab next to the site title.

  • Represents your website's brand
  • Appears in browser tabs, bookmarks, and history
  • Helps users quickly identify your site
