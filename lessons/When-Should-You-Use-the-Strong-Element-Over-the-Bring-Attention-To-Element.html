The b Element (Bring Attention To)

The "bring attention to" element (b) is commonly used to highlight keywords in summaries, or product names in reviews.

Usually, browsers display this text in boldface. This visual emphasis will draw readers' attention to specific content.

The b element draws attention without indicating that the text is more important or urgent.

Common Uses for b
/* Highlighting keywords */
<b>keywords</b> in summaries

/* Product names in reviews */
the <b>iPhone 15</b> performed well

/* Important terms */
learn about <b>accessibility</b>

→ Visual emphasis only
→ Not more important

Example: Product Review

Here's an example using the b element to highlight product names in a review.

The browser renders these parts of the text as bold text. This visual emphasis draws readers' attention to the product names without suggesting they are critically important.

Product Names Highlighted
<p>
  We tested several products, including 
  the <b>SuperSound 3000</b> for audio 
  quality, the <b>QuickCharge Pro</b> 
  for fast charging, and the 
  <b>EcoClean Vacuum</b> for cleaning. 
  The first two performed well, but the 
  <b>EcoClean Vacuum</b> did not meet 
  expectations.
</p>

The strong Element

If you need to emphasize the importance of the text, you should use the strong element instead of the b element.

strong is a semantic HTML element that emphasizes text that is crucial or urgent.

The strong element communicates a sense of urgency or critical importance to both users and assistive technologies.

Use strong for Importance
/* Critical information */
<strong>Warning:</strong> Do not mix chemicals

/* Urgent notices */
<strong>Deadline:</strong> Submit by Friday

/* Important instructions */
<strong>Required:</strong> Complete all fields

→ Conveys importance/urgency
→ Semantic meaning

Example: Important Warning

This is an example where the strong element is used to label a very important warning about potential allergic reactions that customers may have to a product.

The strong element communicates that sense of urgency, making it clear this is critical information that users must pay attention to.

Critical Warning
<p>
  <strong>Warning:</strong> This product 
  may cause allergic reactions.
</p>

/* "Warning" is crucial information
   Screen readers emphasize it
   Users understand it's urgent */
Other Important Uses
<p>
  <strong>Important:</strong> Save your work 
  frequently to avoid data loss.
</p>

<p>
  <strong>Note:</strong> This action cannot 
  be undone.
</p>

Visual Similarity, Semantic Difference

Visually, both are very similar because they are both rendered as bold by default.

But their meanings are quite different. While the "bring attention to" element only draws attention to the text without indicating a higher level of importance, the strong element does more than that. It conveys a sense of importance or urgency.

This is their main difference.

Visual Appearance
/* Both look the same: */

<b>bold text</b>
<strong>bold text</strong>

Both display as bold ↑
Semantic Meaning
/* Different meanings: */

<b>text</b>
→ Draws attention
→ Visual emphasis only
→ Not more important

<strong>text</strong>
→ Important or urgent
→ Semantic emphasis
→ Screen readers add importance

Choosing Between b and strong

To choose between them, consider the purpose of the text and its importance within the surrounding content.

Use b to draw attention to text for visual emphasis (keywords, product names) without conveying importance.

Use strong to indicate that text is specially important, crucial, or urgent.

Decision Guide
Ask yourself:

"Is this text more important than 
surrounding text?"

YES → Use <strong>
  Warnings, critical info,
  urgent notices

NO → Use <b>
  Product names, keywords,
  visual highlighting
Examples
/* Just highlighting (b) */
The <b>MacBook Pro</b> is fast.
Key <b>features</b> include...

/* Important/urgent (strong) */
<strong>Caution:</strong> Hot surface
<strong>Do not</strong> share passwords
