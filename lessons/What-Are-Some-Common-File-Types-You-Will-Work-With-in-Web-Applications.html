Understanding File Types

Files are classified based on their content and structure. The file type determines how a file is opened and processed by computer programs.

We identify file types based on their extensions. A file extension is a three or four letter suffix at the end of the file name.

File Extensions

File extensions appear at the end of file names and tell computers which program should open the file.

Examples:

document.html

styles.css

script.js

Core Web File Types

The three main file types that you will find in web applications are HTML, CSS, and JavaScript.

These form the foundation of every web application you'll build.

HTML

.html

Contains the structure and content of the web application

CSS

.css

Defines the styles and visual appearance

JavaScript

.js

Adds advanced functionality and interactivity

Image Formats: JPEG

JPEG stands for "Joint Photographic Experts Group."

It's known for its efficient compression and is great for photographs and images.

However, the compression is lossy, which means that some image quality is lost to reduce the file size.

JPEG

.jpg or .jpeg

Best for: Photographs and complex images

Compression: Lossy (reduces file size, loses some quality)

Pros: Small file sizes, widely supported

Image Formats: PNG

PNG stands for "Portable Network Graphics."

It preserves image quality, even after compression. This results in larger files.

PNG supports transparency and it's great for images with sharp edges, like logos and icons.

PNG

.png

Best for: Logos, icons, images with sharp edges

Compression: Lossless (preserves quality)

Features: Supports transparency

Trade-off: Larger file sizes

Image Formats: GIF and SVG

GIF (Graphics Interchange Format) supports both animation and transparency but has a limited color palette.

SVG (Scalable Vector Graphics) is a vector image format. These images can be scaled as much as needed without losing quality.

GIF

.gif

Best for: Simple animations

Features: Animation, transparency

Limitation: Limited color palette

SVG

.svg

Best for: Icons, logos, illustrations

Type: Vector format (infinitely scalable)

Advantage: No quality loss at any size

Audio Formats

MP3 is an audio format known for its efficient compression. It's a lossy format, so some audio data is lost during compression to make files smaller.

WAV is a lossless audio format that keeps the original quality of the audio.

MP3

.mp3

Compression: Lossy (smaller files)

Best for: Music, podcasts

Trade-off: Some audio quality lost

WAV

.wav

Compression: Lossless (preserves original quality)

Best for: High-quality audio

Trade-off: Larger file sizes

Video Formats

MP4 is one of the most common video formats. It offers good compression and supports multiple audio and video codecs, subtitles, and metadata.

MOV is the QuickTime multimedia file format developed by Apple, primarily associated with QuickTime Player.

WebM is a high-quality open-source video format.

MP4

.mp4

Most common video format with good compression

MOV

.mov

Apple's QuickTime format

WebM

.webm

High-quality open-source format for web

Font Formats

If you ever need to customize the fonts of your web application, you will work with font formats.

TTF (TrueType Font) is compatible with different operating systems.

WOFF (Web Open Font Format) is specifically designed for web development with better compression and metadata storage.

WOFF2 is the successor of WOFF with even more efficient compression and performance.

TTF - TrueType Font

.ttf

Cross-platform compatible

WOFF - Web Open Font Format

.woff

Modern format for web with better compression and metadata

WOFF2

.woff2

Even more efficient compression than WOFF

Archive Formats

You can create archive files if you need to group multiple folders and files.

The most widely used archive format is ZIP.

ZIP offers lossless compression, so no data is lost during the process. It's widely supported across operating systems and software applications.

ZIP

.zip

Purpose: Group multiple files and folders

Compression: Lossless (no data lost)

Support: Widely supported across all systems

Documentation: Markdown

In web applications, you will usually find a file called README that contains information about the application.

They are usually written in Markdown. With Markdown, you can create structured documents with headings, subheadings, links, images, lists, and more.

You will definitely read and write many README files throughout your career.

Markdown

.md or .markdown

Purpose: Documentation and README files

Content includes:

• How to use the application

• Installation instructions

• License information

• How to contribute

Features: Headings, links, images, lists, formatted text

Essential Knowledge

Learning about these common file types is essential for web development.

From the core building blocks of HTML, CSS, and JavaScript to image, video, and audio formats, every file type has an important role in creating interactive web applications.

File Type Categories

Core Web: HTML, CSS, JavaScript

Images: JPEG, PNG, GIF, SVG

Audio: MP3, WAV

Video: MP4, MOV, WebM

Archives: ZIP
