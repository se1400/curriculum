Common Flex Properties

Flex properties control how child elements are distributed and aligned within flex containers. Understanding these properties is essential for creating responsive, flexible layouts.

  • flex-wrap – Controls wrapping behavior of flex items
  • justify-content – Aligns items along the main axis
  • align-items – Aligns items along the cross axis

These properties work together to create flexible, responsive layouts that adapt to different screen sizes and content dimensions.

flex-wrap: nowrap

Default Wrapping Behavior

The flex-wrap property determines how flex items are wrapped within a container to fit the available space. The default value is nowrap.

With nowrap, flex items won't wrap onto a new line, even if their combined width exceeds the container's width. Instead, items will shrink to fit the available space.

flex-wrap: wrap & wrap-reverse

Wrapping Flex Items

Setting flex-wrap: wrap allows items to wrap onto new lines when they exceed the container's width, maintaining their specified dimensions.

The wrap-reverse value wraps items in reverse order, with new lines appearing above previous ones instead of below.

flex-flow Shorthand Property

Combining Direction and Wrap

The flex-flow property is a shorthand that combines flex-direction and flex-wrap into a single declaration, making your CSS more concise.

/* Shorthand syntax */ main { display: flex; flex-flow: column wrap-reverse; } /* Equivalent to: */ main { display: flex; flex-direction: column; flex-wrap: wrap-reverse; }

justify-content Property

Alignment Along the Main Axis

The justify-content property aligns child elements along the main axis of the flex container. By default, the main axis is horizontal (left to right).

When flex-direction is set to column, the main axis becomes vertical. The justify-content property always works along whichever axis is the main axis.

Main Axis: The primary axis along which flex items are laid out. Horizontal by default (row), or vertical when flex-direction is column.

Available Values:

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

justify-content: flex-start & flex-end

Start and End Alignment

flex-start aligns items to the start of the main axis (left side for horizontal, top for vertical).

flex-end aligns items to the end of the main axis (right side for horizontal, bottom for vertical).

justify-content: space-between & space-around

Distributing Space

space-between distributes items evenly with space only between items. The first and last items are flush with the container edges.

space-around adds equal space around each item. The space at the edges is half the space between items, creating a balanced look.

align-items Property

Alignment Along the Cross Axis

While justify-content controls the main axis, align-items controls alignment along the cross axis – the axis perpendicular to the main axis.

For horizontal flex containers (default), the cross axis is vertical. For vertical flex containers (column direction), the cross axis is horizontal.

Cross Axis: The axis perpendicular to the main axis. Vertical by default (for row direction), or horizontal when flex-direction is column.

Available Values:

  • flex-start
  • flex-end
  • center
  • stretch

align-items: center & flex-start

Cross Axis Alignment

center centers items along the cross axis (vertically for horizontal layouts, horizontally for vertical layouts).

flex-start aligns items to the start of the cross axis (top for horizontal layouts, left for vertical layouts).

flex-end aligns items to the end of the cross axis (bottom for horizontal layouts, right for vertical layouts).

stretch makes items fill the entire container along the cross axis. This works best with auto-sized elements or those without explicit width/height.

align-self Property

Individual Item Alignment

The align-self property allows you to override the container's align-items value for specific flex items, giving you fine-grained control over individual item alignment.

You can apply different alignments to different items within the same flex container.

.container { display: flex; align-items: flex-start; /* default for all items */ } .special-item { align-self: center; /* override for this item */ } /* Values: flex-start, flex-end, center, stretch */

Creating Responsive Layouts

Putting It All Together

With these CSS flex properties and your understanding of the flex model, you can create responsive layouts that provide a smooth and inclusive user experience across all devices.

Key Takeaways:

  • flex-wrap controls whether items wrap to new lines
  • justify-content distributes space along the main axis
  • align-items aligns items along the cross axis
  • align-self overrides alignment for individual items
  • flex-flow combines direction and wrap properties

These are the most commonly-used flex properties, but there are additional properties and values available for even more control over your responsive layouts.

/* Example: Common layout pattern */ .container { display: flex; justify-content: space-between; align-items: center; flex-wrap: wrap; gap: 20px; } /* Responsive and flexible! */
