Finding Substrings in Strings

When working with strings in JavaScript, there may be times when you need to locate the position of a specific substring within a larger string.

What is a Substring?

A substring is a sequence of characters that appears within a larger string.

Example

In the string "hello world":

  • "hello" is a substring
  • "world" is a substring
  • "lo wo" is also a substring

The indexOf() Method

indexOf()

To locate the position of a substring inside a string, you can use the indexOf() method.

What It Does

The indexOf() method allows you to search for a substring within a string.

‚úì If Found

indexOf() returns the index (or position) of the first occurrence of that substring.

‚úó If Not Found

indexOf() returns -1, which indicates that the search was unsuccessful.

indexOf() Arguments

The indexOf() method takes two arguments:

The substring you want to find within the larger string.

string.indexOf("search")

2Ô∏è‚É£ Second Argument (Optional)

If not provided, the search starts at index 0.

An argument is a value you give to a function or method when you call it, enabling that function or method to perform its task using the specific information you provide.

Finding a Substring

Here is an example of using indexOf() to find the position of the string "awesome":

let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome");
console.log(position); // 14
14

Why 14?

The word "awesome" starts at index 14 in the string "JavaScript is awesome!"

When the Substring Isn't Found

Now, let's see what happens when the substring isn't found:

let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
-1

‚ùå Returns -1

Since the word "fantastic" does not appear in the string, the method returns -1.

üí° Remember

-1 always means "not found"

Specifying a Starting Position

You can specify where to begin searching within the string by providing a second argument to indexOf().

Example: Finding the Second Occurrence

let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
27

What Happened?

The search for "JavaScript" begins after the 10th character.

The second occurrence of "JavaScript" is found at index 27.

This is useful when you want to find multiple occurrences of the same substring!

indexOf() is Case Sensitive

‚ö†Ô∏è Important Note

The indexOf() method is case sensitive.

Example: Case Mismatch

console.log("freeCodeCamp".indexOf("F")); // -1
-1

Why -1?

This returns -1 because the capital letter "F" is not found in the string "freeCodeCamp".

The string starts with a lowercase "f", not an uppercase "F".

‚ùå Won't Match

"hello".indexOf("H")  // -1
"World".indexOf("w")  // -1

‚úì Will Match

"hello".indexOf("h")  // 0
"World".indexOf("W")  // 0

Practical Uses of indexOf()

Using indexOf() can be very useful when you need to:

Common Use Cases

  • Check if a substring is present in a string
  • Determine its position for further operations
  • Find multiple occurrences of the same substring
  • Validate user input
  • Search through text data

Example 1: Checking if Substring Exists

let email = "user@example.com";
if (email.indexOf("@") !== -1) {
  console.log("Valid email format");
} else {
  console.log("Invalid email format");
}

Example 2: Finding File Extension

let filename = "document.pdf";
let dotPosition = filename.indexOf(".");
let extension = filename.substring(dotPosition + 1);
console.log(extension); // "pdf"

More indexOf() Examples

Example 1: Finding First Occurrence

let text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("the")); // 31 (finds lowercase "the")
31

Example 2: Searching from a Position

let phrase = "hello hello hello";
console.log(phrase.indexOf("hello"));     // 0 (first occurrence)
console.log(phrase.indexOf("hello", 1));  // 6 (second occurrence)
console.log(phrase.indexOf("hello", 7));  // 12 (third occurrence)
0 6 12

Example 3: Not Found

let message = "JavaScript is fun!";
console.log(message.indexOf("Python")); // -1 (not found)
-1

Understanding Return Values

Scenario Return Value Meaning
Substring found at start 0 Substring begins at index 0
Substring found in middle 14 (example) Substring begins at index 14
Substring not found -1 Substring doesn't exist in string
Empty string search 0 or start position Empty string matches at start

üí° Key Point

Always check if the return value is -1 to determine if the substring was found or not.

indexOf() Summary

Key Takeaways

What is indexOf()?

A string method that searches for a substring and returns its position (index).

Syntax

string.indexOf(searchString)
string.indexOf(searchString, startPosition)

Arguments

  • First: The substring to search for (required)
  • Second: Starting position for the search (optional)

Return Values

  • If found: Returns the index of the first occurrence
  • If not found: Returns -1

Important Notes

  • indexOf() is case sensitive
  • Returns the first occurrence only
  • Use the second argument to find subsequent occurrences
  • Always returns a number (index or -1)

Common Use Cases

  • Check if a substring exists in a string
  • Find the position of specific characters or words
  • Validate string formats (like emails)
  • Search and extract information from text
