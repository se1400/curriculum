Introduction to While Loops

What Is a While Loop?

A while loop will run a block of code as long as the condition is true.

Basic Syntax

while (condition) { // code block to be executed }

How It Works

  • The condition is checked before the block of code executes
  • If the condition is true, the code block runs
  • If the condition is false, the code block does not execute at all
  • The loop continues until the condition becomes false
When to Use While Loops:

While loops are useful when you do not know how many times you need to run the block of code. The number of iterations depends on when the condition becomes false.

While Loop Example

Basic Counter Example

let counter = 0;

while (counter < 5) {
    console.log(counter);
    counter++;
}
Output:
0
1
2
3
4

Step-by-Step Execution

counter = 0 → 0 < 5? TRUE → Print 0, counter becomes 1
counter = 1 → 1 < 5? TRUE → Print 1, counter becomes 2
counter = 2 → 2 < 5? TRUE → Print 2, counter becomes 3
counter = 3 → 3 < 5? TRUE → Print 3, counter becomes 4
counter = 4 → 4 < 5? TRUE → Print 4, counter becomes 5
counter = 5 → 5 < 5? FALSE → LOOP STOPS
Key Points:

• The variable counter is initialized to 0
• The condition checks if counter is less than 5
• Inside the loop, we log the value and increment the counter
• The loop stops when counter reaches 5

While Loop Flow Diagram

Execution Flow

Step 1: Check Condition
Is the condition true?
↓ If TRUE
Step 2: Execute Code Block
Run the code inside the loop
↓
Step 3: Repeat
Go back to Step 1
If FALSE: Exit Loop
Skip the code block entirely
⚠️ Important:

If the condition is initially false, the code block inside the while loop will never execute. The loop checks the condition before running the code.

The Do...while Loop

What Is a Do...while Loop?

The do...while loop is similar to the while loop, but with one key difference.

Basic Syntax

do { // code block to be executed } while (condition);

The Key Difference

The do...while loop will execute the code block at least once before checking the condition.

How It Works

  • The code block runs first
  • Then the condition is checked
  • If the condition is true, the loop continues
  • If the condition is false, the loop stops
Guaranteed Execution:

Because the condition is checked after executing the code, the do...while loop will always run at least once, even if the condition is initially false.

Do...while Loop Example

Basic Counter Example

let counter = 0;

do {
    console.log(counter);
    counter++;
} while (counter < 5);
Output:
0
1
2
3
4

Execution Flow

Step 1: Execute Code Block
Print counter, increment (counter = 0 → 1)
↓
Step 2: Check Condition
Is counter < 5? (1 < 5 = TRUE)
↓ If TRUE, repeat
Continues until counter = 5
Then 5 < 5 = FALSE → LOOP STOPS
Notice:

In this example, the output is the same as the while loop. However, the order of operations is different: execute first, then check condition.

While vs Do...while: The Key Difference

Same Example, Different Results

Let's see what happens when the condition is initially false:

While Loop

let counter = 10;

while (counter < 5) {
    console.log(counter);
    counter++;
}
Output: (nothing)
Result:
Condition is false from the start (10 < 5 = false), so the code block never executes.

Do...while Loop

let counter = 10;

do {
    console.log(counter);
    counter++;
} while (counter < 5);
Output: 10
Result:
Code block executes once before checking the condition. Prints 10, then checks (11 < 5 = false), stops.
This Is the Critical Difference!

• While loop: Checks condition FIRST → May execute 0 times
• Do...while loop: Executes FIRST → Always executes at least 1 time

When to Use Each Loop

Choosing the Right Loop

Use While Loop When:

  • You need to check the condition before executing
  • The code might need to execute 0 or more times
  • You want to avoid executing if condition is initially false
Example Use Case:
Processing items until a list is empty

Use Do...while Loop When:

  • You need to execute the code at least once
  • The condition check should happen after execution
  • Guaranteed first execution is required
Example Use Case:
Getting user input (menu must show at least once)
General Usage Note:

In most cases, you will probably use the while loop more often than the do...while loop. However, it's good to know both types of loops and understand when each is most appropriate.

Summary

  • While: Condition checked first, executes 0+ times
  • Do...while: Executes first, condition checked after, executes 1+ times
  • Both are useful for unknown iteration counts
  • Choose based on whether you need guaranteed first execution
