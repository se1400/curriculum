What Are Pseudo-elements?

Pseudo-elements are virtual or synthetic elements that don't directly match any actual HTML elements.

The Term "Pseudo":

"Pseudo" means "not real" — these are elements that exist in CSS but not in the HTML document.

What They Enable:

  • Style specific parts of an element
  • Insert content before or after elements
  • Add styling without extra HTML markup
  • Create visual effects and enhancements
Pseudo-elements allow you to style specific parts of an element's content or insert content without adding extra HTML

Pseudo-element Syntax

Pseudo-elements are attached to a selector using a double colon (::).

Basic Syntax:

selector::pseudo-element {
  property: value;
}

Key Points:

  • Use double colon (::) for pseudo-elements
  • Can use any selector type (class, ID, element)
  • Different from pseudo-classes (single colon :)
  • Attached to an "originating element"

Important Distinction:

Pseudo-elements (::) use double colon, while pseudo-classes (:) use a single colon.

The Originating Element

Pseudo-elements cannot exist independently — they must be attached to an element.

Originating Element:

The element to which a pseudo-element is attached is called its originating element.

Example:

button::before {
  content: '⭐';
}

In this example, button is the originating element, and ::before is the pseudo-element that depends on it.

Pseudo-elements depend on the originating element and cannot exist independently

::before Pseudo-element

The ::before pseudo-element lets you insert content before an element's content.

Common Uses:

  • Add icons or symbols before content
  • Insert decorative elements
  • Add visual indicators
  • Create custom bullets or markers

CSS Example:

.cta-button::before {
  content: '⭐';
  position: absolute;
  left: 10px;
}

The content Property:

The content property is required to define what should be inserted.

::before Visual Example

Button with a star added using ::before:

The CSS:

.before-demo::before {
  content: '⭐';
  position: absolute;
  left: 10px;
}

Notice how the star appears before "Learn More" without any HTML modification. You can also style the pseudo-element with color, font-size, and positioning.

::after Pseudo-element

The ::after pseudo-element lets you insert content after an element's content.

Common Uses:

  • Add icons or arrows after content
  • Insert decorative elements
  • Create visual effects
  • Add animated indicators

CSS Example:

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 10px;
  transition: transform 0.3s ease;
}

::after Visual Example

Button with an arrow added using ::after:

The CSS:

.after-demo::after {
  content: '➡️';
  position: absolute;
  right: 10px;
  transition: transform 0.3s ease;
}

The arrow appears after "Learn More" and can be animated or styled independently.

Adding Hover States

You can combine pseudo-classes with pseudo-elements for interactive effects.

CSS Example:

.after-demo::after {
  content: '➡️';
  transition: transform 0.3s ease;
}

.after-demo:hover::after {
  transform: translateX(2px);
}

Hover over the button to see the arrow move

How It Works:

The transform: translateX(2px) pushes the arrow 2px to the right on hover, and transition makes it smooth over 0.3 seconds.

::first-letter Pseudo-element

The ::first-letter pseudo-element targets the first letter of an element's content.

Common Uses:

  • Create drop caps in articles
  • Emphasize opening letters
  • Add typographic flair
  • Style magazine-style layouts

CSS Example:

p::first-letter {
  font-size: 3em;
  font-weight: bold;
  color: #881518;
  float: left;
  margin-right: 5px;
}

::marker Pseudo-element

The ::marker pseudo-element selects the marker, bullet, or numbering of list items.

Purpose:

  • Customize list bullets and numbers
  • Match your brand colors
  • Change marker size and color
  • Enhance list visual design

CSS Example:

li::marker {
  color: #881518;
  font-size: 1.5em;
}

More Pseudo-elements to Explore

There are many more pseudo-elements available in CSS:

::placeholder

Styles the placeholder text in input fields

::selection

Styles the portion of text selected by the user

::first-line

Targets the first line of text in a block element

::backdrop

Styles the backdrop behind fullscreen or dialog elements

Keep Exploring:

These are just a few examples. Pseudo-elements like ::spelling-error, ::grammar-error, and others offer even more styling possibilities.

Pseudo-elements Summary

Key Takeaways:

  • Pseudo-elements are virtual elements that don't exist in HTML
  • Use double colon (::) to distinguish from pseudo-classes (:)
  • ::before - Inserts content before an element
  • ::after - Inserts content after an element
  • ::first-letter - Styles the first letter (drop caps)
  • ::marker - Customizes list bullets and numbers
  • content property is required for ::before and ::after
  • Cannot exist independently - need an originating element
  • Can combine with pseudo-classes for hover effects
Pseudo-elements let you add visual enhancements and style specific parts of content without modifying HTML
