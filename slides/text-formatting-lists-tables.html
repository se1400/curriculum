Working with Special Characters

HTML uses character entities to represent special characters like © and ®.

Entity Syntax

  • Always start with &
  • Always end with ;
  • Can use name or numeric code
caf&eacute;  → café
caf&#233;   → café

Common Special Characters

©&copy;
®&reg;
™&trade;
&&amp;
<&lt;
>&gt;

Common Character Entities

CharCodeDescription
"&quot;Quotation mark
&&amp;Ampersand
<&lt;Less than
>&gt;Greater than
©&copy;Copyright
®&reg;Registered
CharCodeDescription
°&deg;Degree
½&frac12;One-half
¼&frac14;One-fourth
×&times;Multiply
÷&divide;Divide
é&eacute;Accented e

Modern Tip: With UTF-8 encoding (<meta charset="utf-8">), you rarely need entities. The main exceptions are &lt;, &gt;, and &amp;.

Boldface, Italic, and Special Formatting

Semantic Tags (Preferred)

  • <strong> — Strong emphasis (bold)
  • <em> — Emphasis (italic)
  • <sup> — Superscript (x²)
  • <sub> — Subscript (H₂O)
  • <pre> — Preformatted text

CSS Style Rules

  • font-weight: bold;
  • font-style: italic;
  • text-decoration: underline;
  • text-decoration: line-through;
<p style="font-weight:bold;
   font-style:italic;">
  Bold and italic!
</p>

Why Use <strong> and <em>?

Old Way (Avoid)

<b>Bold text</b>
<i>Italic text</i>

Only describes appearance

Modern Way (Use This)

<strong>Important</strong>
<em>Emphasized</em>

Describes meaning + appearance

Benefits of semantic tags:

  • Screen readers announce them differently for accessibility
  • Search engines understand content importance
  • You can style them however you want with CSS

Tweaking the Font with CSS

Key Font Properties

  • font-family — Typeface
  • font-size — Size
  • font-weight — Boldness
  • color — Text color
p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  color: #333;
}

Font Family Fallbacks

Always provide alternatives:

font-family: 'Helvetica Neue',
             Arial,
             sans-serif;

List specific fonts first, then generic family (serif, sans-serif, monospace) last.

Font Size Options

Pixels (px)

Fixed size: font-size: 16px;

Em/Rem

Relative: font-size: 1.2rem;

Keywords

small, medium, large

Viewport Units

Responsive: font-size: 2vw;

Modern Fluid Typography with clamp()

font-size: clamp(1rem, 2.5vw, 2rem);

Sets minimum (1rem), preferred (2.5vw), and maximum (2rem) sizes that scale smoothly!

Using Web Fonts

Load custom fonts from external sources like Google Fonts.

Step 1: Add to <head>

<link rel="preconnect"
  href="https://fonts.googleapis.com">
<link href="https://fonts.
  googleapis.com/css2?family=
  Cherry+Swash&display=swap"
  rel="stylesheet">

Step 2: Use in CSS

h1 {
  font-family: 'Cherry Swash',
               cursive;
}

Variable Fonts

A single font file containing all weight variations — fewer requests, smaller files!

Traditional Fonts

  • Separate files for each weight
  • Limited to 400, 700, etc.
  • Multiple HTTP requests

Variable Fonts

  • One file, all weights
  • Any value: 100-900
  • Single HTTP request
h1 { font-weight: 650; }
p  { font-weight: 320; }

Aligning Text on a Page

Use the text-align property to control horizontal alignment.

text-align: left;

Default alignment (most text)

text-align: center;

Center headings, captions

text-align: right;

Dates, numbers, signatures

text-align: justify;

Newspaper-style columns

<p style="text-align: center;">This paragraph is centered.</p>
<section style="text-align: right;">Everything in here is right-aligned.</section>

The Three Types of HTML Lists

<ol> Ordered List

Numbered items for sequences or rankings

<ol>
  <li>First</li>
  <li>Second</li>
</ol>
<ul> Unordered List

Bulleted items for non-sequential content

<ul>
  <li>Apple</li>
  <li>Orange</li>
</ul>
<dl> Definition List

Terms and their definitions

<dl>
  <dt>HTML</dt>
  <dd>Markup</dd>
</dl>

Styling Lists with CSS

list-style-type

Change bullet or number style:

ul { list-style-type: square; }
ul { list-style-type: circle; }
ol { list-style-type: upper-roman; }
ol { list-style-type: lower-alpha; }

Custom Bullets

Use images or emojis:

ul {
  list-style-type: none;
}
li::before {
  content: "→ ";
  color: #3498db;
}

Tip: Use list-style-position: inside; to move bullets inside the content flow.

Nesting Lists

Place one list inside another for hierarchical information.

<ul>
  <li>Fruits
    <ul>
      <li>Apples</li>
      <li>Oranges</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrots</li>
      <li>Broccoli</li>
    </ul>
  </li>
</ul>
  • Fruits
    • Apples
    • Oranges
  • Vegetables
    • Carrots
    • Broccoli

Nested lists automatically get different bullet styles at each level.

Creating HTML Tables

Tables display data in rows and columns — perfect for schedules, comparisons, and data.

Basic Table Structure

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Key Tags

  • <table> — Container
  • <tr> — Table row
  • <th> — Header cell (bold)
  • <td> — Data cell
  • <thead> — Header group
  • <tbody> — Body group

Spanning Rows and Columns

Merge cells across rows or columns for complex layouts.

colspan

Span across multiple columns:

<tr>
  <td colspan="2">
    Spans 2 columns
  </td>
</tr>

rowspan

Span across multiple rows:

<tr>
  <td rowspan="3">
    Spans 3 rows
  </td>
  <td>Row 1</td>
</tr>

Caution: Don't use tables for page layout! Use CSS Grid or Flexbox instead. Tables are for tabular data only.

Styling Tables with CSS

border-collapse

collapse merges borders; separate keeps gaps

padding

Add space inside cells

text-align

Align cell content

nth-child

Zebra striping for rows

table { border-collapse: collapse; width: 100%; }
th, td { border: 1px solid #ddd; padding: 12px; text-align: left; }
tr:nth-child(even) { background-color: #f9f9f9; }
tr:hover { background-color: #f1f1f1; }

CSS Columns

Create newspaper-style multi-column layouts without tables!

Column Properties

.article {
  column-count: 3;
  column-gap: 40px;
  column-rule: 1px solid #ccc;
}
  • column-count — Number of columns
  • column-width — Minimum width
  • column-gap — Space between
  • column-rule — Line between columns

Responsive Columns

.article {
  columns: 200px auto;
}

Browser creates as many 200px columns as will fit!
