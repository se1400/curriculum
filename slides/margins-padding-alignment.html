Using Margins

Margins add empty space around the outside of an element's rectangular area.

Margin Properties

  • margin-top — Sets the top margin
  • margin-right — Sets the right margin
  • margin-bottom — Sets the bottom margin
  • margin-left — Sets the left margin
  • margin — Shorthand for all four margins

Values

  • auto — Browser calculates the margin
  • Length units — px, em, rem, etc.
  • Percentages — Relative to container width
/* Individual properties */
margin-top: 20px;
margin-right: 10px;

/* Shorthand */
margin: 20px 10px;

Understanding Margin Shorthand: TRBL

Remember the order with the TRBL mnemonic (pronounced "trouble" or "tribble" for Star Trek fans!):

T Top
R Right
B Bottom
L Left
Think of it clockwise: start at the top and work your way around the rectangle
This TRBL order also applies to padding and border properties!

Margin Shorthand Patterns

The margin property accepts 1 to 4 values with different meanings:

One Value

margin: 15px;

Applies 15px to all four sides

Two Values

margin: 10px 20px;

10px top/bottom, 20px left/right

Three Values

margin: 10px 20px 15px;

10px top, 20px left/right, 15px bottom

Four Values

margin: 10px 20px 15px 5px;

Top, Right, Bottom, Left (TRBL)

Pro tip: Use 1 value or 4 values consistently for clarity in your code

Removing Default Body Margins

Browsers add default margins to the <body> element. To ensure precise layouts, reset it:

body {
  margin: 0px;
}

Notice how the divs now start flush with the browser edge!

Understanding Margin Collapse

Vertical margins collapse between adjacent elements — they don't add up!

Actual gap: 30px

Top Box

The larger margin wins! Only vertical margins collapse; horizontal margins don't.

Padding Elements

Padding adds space inside the rectangular area of an element.

Key Difference

  • Margin: Space outside the element
  • Padding: Space inside the element
  • Padding assumes the element's background color
  • Padding increases clickable area for buttons/links

Padding Properties

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
  • padding — Shorthand (uses TRBL order)
/* Two values */
padding: 12px 8px;
/* vertical  horizontal */
Remember: Padding follows the same TRBL shorthand rules as margins!

Keeping Everything Aligned

text-align (Horizontal)

  • left — Left-justified (default)
  • right — Right-justified
  • center — Centered
  • justify — Straight edges on both sides
.centered {
  text-align: center;
}

vertical-align (Inline Elements)

  • top — Aligns with top of line
  • middle — Aligns middle with parent's middle
  • bottom — Aligns with bottom of line
  • baseline — Aligns baselines
  • text-top / text-bottom
img {
  vertical-align: middle;
}
Important: vertical-align only works on inline or inline-block elements!

Centering Blocks of Content

To center block-level elements, use margin: 0 auto; with a defined width:

.centered-block {
  width: 50%;
  margin: 0 auto;
}
The element must have a width smaller than its container for this to work!

Understanding the float Property

The float property is used for wrapping text around images (not for page layouts).

How It Works

  • Elements float horizontally, not vertically
  • Values: left or right
  • Element moves as far as the container allows
  • Following content wraps around it
img {
  float: left;
  margin: 0 15px 15px 0;
}

Modern Usage

Use float for: Text wrapping around images (magazine-style layout)
Don't use float for: Page layouts — use Flexbox or Grid instead! (Lesson 12)
While float was historically used for entire page layouts, modern CSS (Flexbox and Grid) has replaced that use case

The clear Property

Use clear to control what appears after floated elements:

Clear Values

  • clear: left; — No floating elements on the left
  • clear: right; — No floating elements on the right
  • clear: both; — No floating elements on either side
  • clear: none; — Allow floating (default)

Example

img {
  float: left;
}

footer {
  clear: both;
  /* Appears BELOW floated image */
}

Summary: Key Takeaways

Margins & Padding

  • Margin = space outside element
  • Padding = space inside element
  • Both use TRBL shorthand order
  • Vertical margins collapse
  • Reset body margins for precise layouts
  • Use rem for responsive spacing

Alignment

  • text-align for horizontal alignment
  • vertical-align for inline elements only
  • Center blocks with margin: 0 auto;

Float

  • Use for text wrapping around images
  • Don't use for page layouts anymore
  • Use clear to control flow after floats

Modern Techniques

  • rem units for scalable, accessible spacing
  • em for relative sizing
  • Flexbox & Grid for layouts (Lesson 12)
